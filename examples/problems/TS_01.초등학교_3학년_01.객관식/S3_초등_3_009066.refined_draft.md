[문제 개요]
- 한 장짜리 초등 수학 문제 이미지이다.
- 상단에 문제 번호 `7.`과 지시문 `도형을 보고 알맞은 기호를 모두 선택하세요.`가 보인다.
- 가운데에는 도형 5개가 가로로 배열되어 있고, 각 도형 안에 `가, 나, 다, 라, 마`가 각각 들어 있다.
- 아래에는 조건 문장 `네 각이 모두 직각인 사각형은 (가, 나, 다, 라, 마)입니다.`가 보인다.
- 더 아래에 `(정답) 가, 다`가 인쇄되어 있고, 그 아래에 `(해설)` 영역이 따로 있다.

[원본에 보이는 요소]
- 보이는 텍스트
  - `7.`
  - `도형을 보고 알맞은 기호를 모두 선택하세요.`
  - `가`, `나`, `다`, `라`, `마`
  - `네 각이 모두 직각인 사각형은 ( 가, 나, 다, 라, 마 )입니다.`
  - `(정답) 가, 다`
  - `(해설)`
- 도형 구성
  - 상단 본문에 도형 5개가 좌우로 일정 간격으로 놓여 있다.
  - 각 도형은 외곽선만 있는 단색 도형이며, 내부 중앙에 글자가 1개씩 있다.
  - 배경은 연한 파란색 격자무늬이다.
- 해설 영역 구성
  - 하단에 같은 도형들이 다시 배열되어 있다.
  - 일부 꼭짓점 근처에 작은 직각 표시(ㄱ자 모양)가 추가되어 있다.
  - 해설 영역도 격자 배경이 유지된다.

[수식/관계 구조]
- 이 문항은 등호나 계산식이 있는 문제는 아니다.
- 구조는 “조건 문장 → 조건에 맞는 도형 선택”의 관계이다.
- 조건 문장:
  - `네 각이 모두 직각인 사각형`
- 선택 대상:
  - `가, 나, 다, 라, 마`가 각각 붙은 도형 5개
- 이 문제에서 중요한 것은 도형의 성질 판별이다.
- 단, 정답은 이미지에 이미 인쇄되어 보이므로, 보이는 사실로만 기록하고 새로 추론해서 채우지 않는다.

[등호별 변환 의미]
- 해당 없음.
- 등호를 포함한 식 변형이나 단계적 계산은 보이지 않는다.

[빈칸 정책]
- 이 문항에는 숫자 빈칸이나 계산 빈칸이 보이지 않는다.
- 보기 선택형 문제이므로, DSL 작성 시에도 보이는 텍스트와 도형 배치를 그대로 유지한다.
- 이미지에 이미 인쇄된 `(정답) 가, 다`는 관찰 사실로만 기록한다.
- 빈칸을 추론값으로 채우지 않는다.
- 해설 영역의 직각 표시도 불명확한 부분은 확정하지 않고 빈 상태로 두거나 TODO로 남긴다.

[반복 패턴]
- 상단과 하단에 같은 스타일의 연한 파란 격자 배경이 반복된다.
- 도형 5개가 거의 같은 크기와 간격으로 가로 정렬되어 있다.
- 상단 본문 도형 배열과 하단 해설 도형 배열이 같은 순서로 반복된다.
- 하단 해설에서는 각 도형의 모서리 근처에 작은 직각 표시가 반복적으로 나타난다.
- 글자 `가, 나, 다, 라, 마`는 각 도형의 중앙에 1개씩 들어간다.

[DSL 작성 힌트]
- 상단 문제 영역은 다음 요소를 포함하는 레이아웃이 적절하다:
  - 문제 번호 `7.`
  - 지시문 문장
  - 격자 배경 위에 놓인 도형 5개
  - 도형 내부 중앙 텍스트
  - 조건 문장
  - 정답 표기 텍스트
- 도형 배치는 가로 방향 균등 간격으로 두는 것이 좋다.
- 각 도형은 외곽선 도형으로 표현하고, 내부에 텍스트를 중앙 정렬로 넣는다.
- 배경 격자는 전체 도형 구역에 얇고 연한 선으로 반복되게 처리하면 된다.
- 하단 해설 구역은 별도 블록으로 분리하는 것이 좋다.
- 해설 구역에는 상단과 같은 도형 배열을 다시 배치하되, 일부 꼭짓점에 직각 표시를 덧붙인다.
- `(해설)`과 `(정답) 가, 다`는 텍스트 박스로 따로 처리하는 편이 안전하다.
- 정렬 힌트:
  - 문제 번호는 좌상단.
  - 도형 배열은 그 아래 중앙 영역.
  - 조건 문장은 도형 아래 중앙.
  - 정답 표기는 본문과 해설 사이의 왼쪽 쪽에 위치.
  - 해설은 하단 좌측 시작.

[semantic/solvable 힌트]
- 보이는 문장 `네 각이 모두 직각인 사각형`은 도형의 분류 조건을 설명한다.
- 겉보기로는 직사각형 형태의 도형과 기울어진 변을 가진 도형, 마름모 형태의 도형이 함께 제시되어 있다.
- 이 항목은 풀이를 전개하는 문제가 아니라, 조건에 맞는 도형을 선택하는 문제로 읽힌다.
- 다만 여기서는 정답을 새로 계산하거나 확인해서 채우지 않는다.
- 이미지에 인쇄된 정답 표기는 관찰 사실로만 남긴다.

[주의 및 TODO]
- 정답 추론 금지: 이미지에 보이는 정답 표기 외에는 새로 답을 계산하지 않는다.
- 빈칸은 추론값으로 채우지 말고 그대로 비워 둔다.
- 해설 영역의 직각 표시는 일부 잘려 보여서, 모든 꼭짓점에 표시가 있는지 단정하지 않는다.
- 첫 번째와 세 번째 도형은 직사각형처럼 보이지만, 해상도 때문에 정확한 변 길이와 각도는 단정하지 않는다.
- 다섯 번째 `마` 도형은 마름모처럼 보이지만, 세부 분류는 이미지에 보이는 수준까지만 기록한다.
- 격자색은 `연한 파란 격자` 정도로 표현하고, 너무 강한 색 이름은 쓰지 않는 것이 안전하다.
