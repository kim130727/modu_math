[문제 개요]
- 한 장짜리 초등 수학 판단 문항이다.
- 상단에 문제 번호 `44.`가 있고, 그 오른쪽에 문장형 질문이 길게 배치되어 있다.
- 중앙에는 물병 2개를 비교하는 그림이 좌우로 놓여 있으며, 각 물병에서 오른쪽 방향 화살표가 나가 컵 묶음으로 연결된다.
- 하단에는 `○`, `×` 선택 표시가 세로로 있고, 그 아래에 `(정답)×`가 인쇄되어 보인다.
- 더 아래에는 `(해설)`과 함께 컵 개수 비교 설명, 그리고 `7-4=3(개)`가 포함된 문장이 이어진다.

[원본에 보이는 요소]
- 문제 번호: `44.`
- 문제 문장:
  - `두 병에 물을 가득 채운 후 모양과 크기가 같은 컵에 각각 물을 옮겼습니다. 바르게 설명했으면 ○표, 그렇지 않으면 ×표를 선택하세요.`
- 왼쪽 물병 안의 표기: `가`
- 오른쪽 물병 안의 표기: `나`
- 중앙 문장:
  - `나 물병이 컵 3개만큼 물이 더 들어갑니다.`
- 선택 표시:
  - `○`
  - `×`
- 관찰 가능한 정답 표기:
  - `(정답)×`
- 해설 문장:
  - `(해설)가 물병: 컵 7개, 나 물병: 컵 4개`
  - `가 물병이 나 물병보다 컵 7-4=3(개)만큼 물이 더 들어갑니다.`
- 그림 요소:
  - 왼쪽: 큰 물병 1개
  - 오른쪽: 작은 물병 1개
  - 각 물병 오른쪽에 회색 화살표 1개씩
  - 화살표 끝에 파란 컵 여러 개
- 컵 개수는 그림과 해설에서 다음처럼 보인다:
  - `가` 물병 쪽: 컵 7개
  - `나` 물병 쪽: 컵 4개

[수식/관계 구조]
- 화면에 보이는 관계는 “물병에 들어 있는 물의 양을 같은 컵으로 옮겼을 때 몇 컵인지 비교”하는 구조이다.
- `가` 물병과 `나` 물병의 컵 개수를 비교하는 형태로 보인다.
- 해설에 적힌 수식은 `7-4=3(개)`이다.
- 이 식은 두 물병의 컵 개수 차이를 나타내는 관찰 가능한 표현이다.
- 등호 연쇄는 보이지 않고, 단일 등식 1개만 보인다.

[등호별 변환 의미]
- `7-4=3(개)`는 화면에 인쇄된 해설 표현이다.
- 왼쪽 `7`은 `(해설)가 물병: 컵 7개`와 대응해 보인다.
- `4`는 `(해설)나 물병: 컵 4개`와 대응해 보인다.
- `=3(개)`는 두 물병의 컵 수 차이를 나타내는 결과 표기로 보인다.
- 이 항목은 새로 계산한 결과가 아니라, 원본에 보이는 해설 문장을 해석해 정리한 것이다.
- 변형 단계가 여러 개 이어지는 형태는 아니므로, 단계별 등호 변환은 추가로 없다.

[빈칸 정책]
- 이 문항에는 답을 직접 쓰는 빈칸이 보이지 않는다.
- `○`, `×` 선택형 표시가 있으므로, 작성 시에는 선택 요소로 취급한다.
- 원본에 보이는 값 외의 추론값은 넣지 않는다.
- 특히 `○` 또는 `×` 중 어느 쪽이 정답인지, 원본에 `(정답)×`가 보이더라도 DSL용 본문에는 답을 임의로 채워 넣지 말고 관찰 사실만 분리해 기록한다.
- 빈칸이 따로 존재하는 것으로 보이지 않으므로, 빈칸을 새로 만들지 않는다.

[반복 패턴]
- 물병 2개가 좌우로 반복 배치된다.
- 각 물병마다:
  - 물병 외곽선
  - 내부의 원형 문자 `가`/`나`
  - 오른쪽 화살표
  - 컵 묶음
  이 같은 구조로 반복된다.
- 컵은 동일한 모양이 여러 개 반복되는 방식이다.
- 왼쪽 컵 묶음은 위아래 2줄로 나뉘어 있고, 오른쪽 컵 묶음도 위아래 2줄로 나뉘어 있다.
- 선택 표시 `○`, `×`도 세로로 반복 배치된 형식이다.
- 해설에서도 `가`, `나`를 다시 불러와 같은 비교 구조를 반복한다.

[DSL 작성 힌트]
- 상단 문제문, 중앙 비교 그림 2세트, 하단 선택 표시 및 해설 영역으로 레이아웃을 나누면 좋다.
- 두 물병은 서로 독립된 그룹으로 처리하는 것이 자연스럽다.
- 각 그룹에는 다음 요소를 포함시키면 된다:
  - 물병 윤곽
  - 병 안의 원형 라벨
  - 오른쪽 화살표
  - 컵 반복 배열
- 컵은 작은 파란색 동일 도형을 여러 개 배치한 형태로 표현한다.
- 좌측 컵 7개, 우측 컵 4개는 개수 차이가 드러나도록 배열한다.
- 컵 배열은 2줄로 나뉜 배치를 유지하는 것이 시각적으로 중요하다.
- 선택 표시 `○`, `×`는 세로 배치된 답안 선택 요소로 다룬다.
- 해설 텍스트는 본문과 분리된 하단 텍스트 블록으로 배치한다.
- 마지막 계산식 `7-4=3(개)`는 별도 텍스트 요소로 분리해도 좋다.
- 회색 화살표는 해설 강조용 시각 요소로 보이므로, 텍스트와 분리된 장식 요소로 처리할 수 있다.

[semantic/solvable 힌트]
- 문제의 의미는 “같은 컵으로 옮겼을 때 어느 물병이 더 많은가”를 판단하는 것이다.
- 그림과 해설 모두 `가` 물병이 7컵, `나` 물병이 4컵으로 보인다고 적고 있다.
- 문장 `나 물병이 컵 3개만큼 물이 더 들어갑니다.`는 그림/해설과 비교되는 진술로 배치되어 있다.
- 원본에는 `(정답)×`가 인쇄되어 보이지만, 답을 새로 계산해서 출력하지는 않는다.
- DSL에서는 문제의 참/거짓 판단 구조와 해설 구조를 분리해서 표시하는 것이 적절하다.

[주의 및 TODO]
- 정답 판단을 모델이 새로 계산해서 단정하지 말 것.
- `(정답)×`는 관찰 사실로만 기록하고, 정답 생성용 추론으로 확장하지 말 것.
- `가`, `나`의 의미는 문자 표기 자체만 기록하고, 추가 맥락은 부여하지 말 것.
- 컵 개수 7개, 4개는 해설에 직접 보이는 값이지만, 그림의 세부 간격/정렬은 해상도 한계로 완전 정밀 판독이 어렵다.
- 컵의 정확한 행간, 물병 윤곽의 곡선, 화살표 두께는 단정하지 말고 필요 시 TODO로 남겨라.
- 문제문에서 `바르게 설명했으면 ○표, 그렇지 않으면 ×표를 선택하세요.`는 그대로 유지하되, 선택 인터랙션이 실제로 있는지는 화면만으로는 추가 확인이 필요하다.
