[문제 개요]
- 한 페이지형 객관식 문제이다.
- 상단에 문제 번호 `93.`과 질문 문장이 있고, 중앙에 표가 주요 정보로 배치되어 있다.
- 표 아래에는 선택지 4개가 가로로 배치되어 있으며, 왼쪽 아래에 `(정답)②`, 그 아래에 `(해설)` 문장이 인쇄되어 있다.
- 전체 흐름은 `문제 문장 → 표 → 선택지 → 정답 표시 → 해설`이다.

[원본에 보이는 요소]
- 문제 번호 왼쪽에 작은 사각형 기호 `□`가 보인다.
- 문제 문장:
  - `93. 여수네 반 학생들이 학예회에서 하고 싶어 하는 종목을 조사하여 표로 나타내었습니다. 여학생이 가장 많이 하고 싶어 하는 종목은 무엇인가요?`
- 표 제목:
  - `여학생과 남학생이 학예회에서 하고 싶어 하는 종목`
- 표의 열 제목:
  - `종목`, `연극`, `합주`, `무용`, `합창`, `합계`
- 표의 행 제목:
  - `여학생 수(명)`
  - `남학생 수(명)`
- 표 안 숫자:
  - 여학생 수(명): `2, 7, 3, 1, 13`
  - 남학생 수(명): `4, 3, 2, 6, 15`
- 선택지:
  - `① 연극`
  - `② 합주`
  - `③ 무용`
  - `④ 합창`
- 하단 표기:
  - `(정답)②`
  - `(해설)2, 7, 3, 1 중 가장 큰 수는 7이므로 여학생이 가장 많이 하고 싶어 하는 종목은 합주입니다.`

[수식/관계 구조]
- 이 문항은 등호 연쇄 변형식이 아니라, 표에서 값을 비교하여 답을 찾는 구조이다.
- 여학생 수의 종목별 값은 `2, 7, 3, 1`로 나열되어 있고, 마지막 `13`은 합계 칸에 있다.
- 남학생 수의 종목별 값은 `4, 3, 2, 6`로 나열되어 있고, 마지막 `15`는 합계 칸에 있다.
- 수학적 의미로는 “여학생 수 중 최댓값이 어느 종목에 해당하는가”를 묻는 형태로 읽힌다.
- 다만 답을 계산해 새로 채우는 방식은 금지이며, 원본에 인쇄된 내용만 관찰 사실로 다룬다.

[등호별 변환 의미]
- 해당 없음.
- 화면에 `=`를 포함한 계산식이나 변환식이 보이지 않는다.
- 따라서 단계별 등호 변형 설명은 필요하지 않다.

[빈칸 정책]
- 빈칸이나 미지 값을 추론해서 채우지 않는다.
- 표의 숫자와 선택지, 정답 표시가 원본에 보이는 그대로 유지되어야 한다.
- 만약 DSL에서 빈 칸 슬롯이 필요하더라도, 관찰되지 않은 값은 절대 자동 보정하지 않는다.
- 이 문항에서는 질문의 정답을 계산해 문장으로 확정하지 말고, 원본에 적힌 정답 표기만 사실로 기록한다.

[반복 패턴]
- 표는 5열 구조로 반복된다.
  - 첫 열: 행 제목
  - 뒤의 4열: 종목명과 대응하는 수치
  - 마지막 열: 합계
- 두 행이 같은 종목 열을 공유하므로, 열 정렬과 행 대응이 중요하다.
- 선택지는 `①~④` 번호형 보기로 반복되며, 각 보기 앞의 번호가 파란색 동그라미 스타일로 보인다.
- 하단의 `(정답)`과 `(해설)`은 문제 본문과 분리된 보조 텍스트로 반복 배치된 요소처럼 보인다.

[DSL 작성 힌트]
- 표는 중앙 배치가 핵심이며, 제목은 표 바로 위 중앙에 따로 둔다.
- 표 구조는 다음처럼 읽는 것이 자연스럽다:
  - 1행: `종목 / 연극 / 합주 / 무용 / 합창 / 합계`
  - 2행: `여학생 수(명) / 2 / 7 / 3 / 1 / 13`
  - 3행: `남학생 수(명) / 4 / 3 / 2 / 6 / 15`
- 각 셀은 중앙 정렬에 가깝게 보이며, 왼쪽 첫 열은 행 제목 영역으로 약간 넓게 잡혀 있다.
- 표 테두리는 연한 갈색/베이지 계열의 얇은 선처럼 보인다.
- 문제 문장은 상단에서 두 줄로 줄바꿈되어 보이므로, 문장 길이와 줄바꿈 위치를 고려해 배치하는 것이 좋다.
- 선택지는 표 아래에 가로로 일정 간격을 두고 배치한다.
- 정답 표시는 `(정답)②`처럼 별도 텍스트 블록으로 분리하는 것이 적절하다.
- 해설 문장은 하단 왼쪽에 한 줄 이상으로 길게 배치되어 있으며, 숫자 비교 설명이 포함된다.

[semantic/solvable 힌트]
- 이 문제는 표 해석 문제이다.
- 핵심 의미는 여학생 수가 종목별로 어떻게 분포하는지 보고 가장 큰 값을 찾는 것이다.
- 원본 해설에는 `2, 7, 3, 1 중 가장 큰 수는 7`이라는 설명이 보인다.
- 다만 이 문장은 원본에 인쇄된 설명을 옮긴 관찰 사실로만 취급하고, 별도로 풀이를 확장하지 않는다.
- 문제 해결에 필요한 정보는 표와 선택지에 모두 나타나 있다.
- 답을 새로 계산하여 채워 넣지 말고, 원본에 보이는 정답 표기만 기록한다.
