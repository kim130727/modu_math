# [전체적인 구성]
- 화면 상단 왼쪽에 문제 번호와 문장이 있고, 가운데에는 저울 그림이 배치되어 있습니다.
- 저울 그림 위쪽에는 각각 물체 이름 라벨이 붙어 있습니다.
- 그림 아래에는 선택형 문장이 크게 제시되고, 그 아래에 `(정답)` 표시가 붙은 한 줄과 `(해설)` 문장이 이어집니다.
- 전체적으로 **문제 제시 → 시각 자료 → 빈칸이 있는 선택 문장 → 정답/해설** 순서의 구성입니다.

# [보이는 텍스트]
- `□ 66.`
- `저울을 사용하여 공책과 필통의 무게를 비교하려고 합니다. 알맞은 말을 선택하세요.`
- 라벨: `공책`, `필통`
- 선택 문장: `공책이 필통보다 더 ( 무겁습니다, 가볍습니다 ).`
- `(정답) 가볍습니다`
- `(해설) 공책을 올려놓은 접시가 위로 올라갔으므로 공책이 필통보다 더 가볍습니다.`

# [반복되는 시각 패턴]
- 물체 이름이 **작은 둥근 사각형 라벨** 안에 표시되어 있고, 각각 저울 위의 해당 물체 위치 위쪽에 놓여 있습니다.
- 선택 문장 안에서 `(` `)`로 감싼 **두 개의 선택어**가 나란히 제시됩니다.
- 정답과 해설은 본문과 분리되어 아래쪽에 소괄호 표기로 나타납니다.
- 저울 그림은 **좌우 대칭 구조**로, 양쪽 접시와 중앙 지지대가 뚜렷하게 구분됩니다.

# [수식 단계]

- 숫자 계산식이나 등호가 이어지는 변형식은 보이지 않습니다.

# [등호별 변환 의미]
- 해당 없음.  
- 등호를 포함한 수식 변형 단계가 제시되어 있지 않습니다.

# [세부 내용]
- 중심 시각 자료는 **양팔 저울**입니다.
- 저울의 왼쪽 접시에는 **공책**이 올라가 있고, 오른쪽 접시에는 **필통**이 올라가 있습니다.
- 왼쪽 접시 쪽이 더 위로 올라간 상태로 그려져 있어, 두 물체의 무게 차이를 비교하는 상황임을 보여 줍니다.
- 저울 중앙의 지지축과 양쪽 팔의 기울기로 균형이 맞지 않은 상태가 표현되어 있습니다.
- 선택 문장에서는 `공책이 필통보다 더` 다음에 두 가지 선택지 `무겁습니다, 가볍습니다`가 괄호 안에 함께 제시되어 있습니다.
- 아래의 `(정답)` 줄과 `(해설)` 줄은 문제를 푸는 답안 안내 영역으로 보입니다.

# [Python DSL 작성 힌트]
- 문제 텍스트 영역과 그림 영역을 분리해서 배치하면 좋습니다.
- 저울 그림은 다음 요소로 나누어 표현할 수 있습니다.
  - 중앙 받침대
  - 좌우 지지 팔
  - 왼쪽/오른쪽 접시
  - 왼쪽 접시 위의 공책
  - 오른쪽 접시 위의 필통
  - 각 물체 위 라벨 박스
- 선택 문장은 하나의 문장으로 두고, 괄호 안 선택어는 별도 텍스트 블록처럼 처리하면 좋습니다.
- 정답과 해설은 본문과 시각적으로 떨어진 하단 텍스트로 배치하는 것이 자연스럽습니다.
- 저울의 기울기 방향과 접시 높이 차이를 도형 속성으로 반영하면 그림 구조를 더 정확히 재현할 수 있습니다.

# [주의할 점]
- 정답은 이미지에 이미 표시되어 있지만, DSL 작성용 분석에서는 **그 값을 추론해서 문제 본문에 끼워 넣지 않는 방식**으로 다루는 것이 좋습니다.
- `무겁습니다/가볍습니다`는 선택지로 보이며, 본문 빈칸의 정답을 계산해 서술하지 않도록 주의해야 합니다.
- 저울의 상태를 설명할 때는 `왼쪽이 위`, `오른쪽이 아래`처럼 **보이는 방향만** 적는 것이 안전합니다.
- 해설 문장은 참고용 텍스트로 존재하지만, 이것을 바탕으로 다른 결론을 추가 생성하지 않도록 합니다.

# [불확실한 부분]
- 저울 그림의 정확한 기울기 각도는 이미지 해상도상 정밀하게 판독하기 어렵습니다.
- 공책과 필통의 세부 장식(표지 무늬, 글자, 색감의 미세한 차이)은 일부만 선명하게 보입니다.
- `(정답)` 표기의 위치는 본문 바로 아래로 보이지만, 줄 간격은 이미지에서 정확히 측정하기 어렵습니다.
