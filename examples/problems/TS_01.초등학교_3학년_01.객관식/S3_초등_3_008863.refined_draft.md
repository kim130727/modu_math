[문제 개요]
- 단일 객관식 문제 화면이다.
- 질문은 `L와 mL 사이의 관계를 잘못 나타낸 것은 어느 것일까요?`이다.
- 보기 5개가 원형 번호 표식과 함께 제시되어 있다.
- 하단에 `(정답)` 표시와 `(해설)` 표시가 별도로 있다.
- 원본 이미지에서는 `(정답) ⑤`가 보이며, `(해설)` 뒤에 등식 변형 설명이 한 줄로 이어진다.

[원본에 보이는 요소]
- 상단 왼쪽에 문제 번호 `9.`가 있다.
- 문제 문장: `L와 mL 사이의 관계를 잘못 나타낸 것은 어느 것일까요?`
- 보기:
  - `① 1900mL = 1L 900mL`
  - `② 4005mL = 4L 5mL`
  - `③ 1L 900mL = 1900mL`
  - `④ 6L 20mL = 6020mL`
  - `⑤ 5L 100mL = 5010mL`
- 하단:
  - `(정답) ⑤`
  - `(해설) ⑤ 5L 100mL = 5L + 100mL = 5000mL + 100mL = 5100mL`
- 파란색 원형 숫자 마커 `①~⑤`가 각 보기 앞에 붙어 있다.
- 보기 배치는 좌우 두 열 느낌이지만, 실제로는 왼쪽에 ①, ③, ⑤가 위에서 아래로 이어지고 오른쪽에 ②, ④가 배치되어 있다.

[수식/관계 구조]
- 보기들은 모두 `좌변 = 우변` 형태의 관계식으로 보인다.
- 단위는 `L`와 `mL`가 반복 사용된다.
- 해설은 한 식을 여러 표현으로 바꾸는 등호 연쇄 형태이다.
- 해설에 보이는 순서:
  1. `5L 100mL`
  2. `= 5L + 100mL`
  3. `= 5000mL + 100mL`
  4. `= 5100mL`
- 각 단계는 같은 양을 다른 표기법으로 나타낸 것으로 읽힌다.

[등호별 변환 의미]
- `5L 100mL = 5L + 100mL`
  - 보이는 의미: 리터와 밀리리터가 붙어 있는 표기를 더하기 형태로 나눈 것이다.
- `5L + 100mL = 5000mL + 100mL`
  - 보이는 의미: `5L`를 `5000mL`로 바꿔 쓴 것이다.
- `5000mL + 100mL = 5100mL`
  - 보이는 의미: 같은 단위끼리 더한 결과를 한 값으로 나타낸 것이다.
- `a/b ÷ n -> a/(b×n)` 형태의 변환은 보이지 않는다.
- 해설의 등호 사슬은 계산 과정 설명용 표기이며, 새 풀이를 전개하는 것이 아니라 화면에 있는 흐름을 읽어 정리한 것이다.

[빈칸 정책]
- 이 문제에는 빈칸 입력란이 보이지 않는다.
- 따라서 추론값을 채워 넣지 않는다.
- `(정답) ⑤`는 화면에 인쇄되어 보이는 사실만 기록한다.
- 해설의 마지막 값도 화면에 보이는 텍스트만 반영하고, 별도로 계산해서 새 답을 쓰지 않는다.

[반복 패턴]
- 각 보기는 `원형 번호 마커 + 수식 1개`의 반복 구조이다.
- `L`, `mL` 단위가 보기마다 반복된다.
- 보기들은 거의 같은 길이의 한 줄 식으로 제시되어 있어 시각적으로 일정한 패턴을 이룬다.
- 해설에서도 `=`를 여러 번 반복하는 연쇄 구조가 반복된다.
- `5L 100mL`처럼 수량+단위가 나란히 놓인 표기와, `5000mL + 100mL`처럼 덧셈식 표기가 함께 나온다.

[DSL 작성 힌트]
- 상단에 문제 번호와 질문 문장을 한 줄 또는 두 줄로 배치한다.
- 보기 5개는 번호 원형 마커를 포함한 텍스트 블록으로 구성한다.
- 보기 배치는 왼쪽 열에 ①, ③, ⑤ / 오른쪽 열에 ②, ④가 놓인 느낌을 살리면 좋다.
- 각 보기는 `=`
  를 중심으로 좌변과 우변이 읽히도록 정렬한다.
- `1L 900mL`처럼 리터와 밀리리터가 공백으로 이어진 표기는 그대로 유지한다.
- 해설은 `(해설)` 라벨 뒤에 연쇄 등식 한 줄을 두어야 한다.
- 해설 수식은 `5L 100mL = 5L + 100mL = 5000mL + 100mL = 5100mL`의 구조를 그대로 드러내면 된다.
- `(정답)`과 `(해설)`은 본문보다 작은 보조 레이블처럼 처리하면 화면 구조가 잘 맞는다.
- 표식 번호와 본문 수식 사이의 간격을 너무 좁히지 않도록 한다.

[semantic/solvable 힌트]
- 문제의 의미는 `L`와 `mL` 사이의 관계를 바르게/잘못 표현한 식을 고르는 형식이다.
- 보이는 범위에서는 `(정답) ⑤`가 이미 인쇄되어 있다.
- 해설은 ⑤번 식이 `5L 100mL`를 밀리리터로 바꾸는 과정임을 보여준다.
- 각 보기의 수학적 의미는 문제 해결을 위해 해석될 수 있지만, 여기서는 이미지에 보이는 표기만 정리한다.
- 보기 ①~⑤가 모두 등식 형태이므로, DSL에서는 각각 독립된 수식 줄로 처리하는 것이 적절하다.
