- 문제 ID: `S3_초등_3_008961`
- 화면 상단에 문제 번호 `□ 31.` 과 문제 문장이 보인다.

- 화면 중앙에는 `학생들의 혈액형` 표가 있고, 그 아래에 기호의 의미, 하단에 선택지 ①~⑤, 좌하단에 `(정답) ③`, 맨 아래에 해설 문장이 보인다.

[원본에 보이는 요소]
- 문제 번호: `□ 31.`
- 문제 문장: `정호네 학교 학생 1250명의 혈액형을 조사하여 그림그래프로 나타내었습니다. 학생 수가 많은 혈액형부터 차례대로 나타낸 것을 고르세요.`
- 표 제목: `학생들의 혈액형`
- 표의 열 제목: `혈액형`, `학생 수`
- 혈액형 행 표기: `A`, `B`, `AB`, `O`
- 기호 설명: 웃는 얼굴 아이콘 `100명`, 작은 얼굴 아이콘 `10명`
- 선택지:
  - `① A, B, O, AB`
  - `② AB, O, B, A`
  - `③ A, O, B, AB`
  - `④ O, A, B, AB`
  - `⑤ B, O, A, AB`
- 정답 표시: `(정답) ③`
- 해설 표시: `(해설)` 뒤에 문장형 설명이 이어진다.

[수식/관계 구조]
- 이 화면에는 등호를 포함한 계산식이나 변형식이 보이지 않는다.
- 보이는 관계 구조는 **그림 기호 개수 ↔ 학생 수 크기 비교**이다.
- 표의 각 행은 혈액형별로 같은 종류의 기호를 가로로 나열해 학생 수를 나타낸다.
- 아래 기호 설명은
  - 큰 웃는 얼굴 1개 = `100명`
  - 작은 얼굴 1개 = `10명`
  으로 대응된다.
- 따라서 각 혈액형의 학생 수는 **아이콘 개수의 크기 비교**로 읽는 구조이다.
- 단, 원본을 기준으로 볼 때 숫자값을 새로 계산해 채우는 방식은 하지 않는다.

[등호별 변환 의미]
- 해당 없음.
- 원본 화면에는 `=` 를 사용한 식 변형 연쇄가 보이지 않는다.
- 따라서 단계별 등호 변환 의미를 적을 수 없다.

[빈칸 정책]
- 화면에는 별도의 빈칸 입력란이 보이지 않는다.
- 선택지형 문제이므로, 보이는 보기 문구는 그대로 유지한다.
- 만약 DSL에서 빈칸 슬롯이 필요하더라도, 원본에 없는 값을 추론해 채우지 않는다.
- 정답이 이미지에 인쇄되어 보이더라도, 정답 번호 외의 계산 결과나 숨은 수치를 빈칸에 넣지 않는다.

[반복 패턴]
- 표의 각 혈액형 행마다 **같은 종류의 아이콘이 반복 배치**되어 있다.
- 큰 아이콘과 작은 아이콘이 섞여 있으며, 두 종류의 기호를 반복해 학생 수를 나타내는 패턴이다.
- `A`, `B`, `AB`, `O` 네 행이 같은 형식으로 반복 구성되어 있어, 행 간 비교 문제로 읽힌다.
- 선택지는 모두 혈액형 네 개를 쉼표로 나열한 순서형 보기이며, 순서만 달라진 형태가 반복된다.
- 해설 문장도 `비교 → 더 많음 → 순서 결정`의 서술 패턴으로 이어진다.

[DSL 작성 힌트]
- 레이아웃은 다음 덩어리로 나누어 배치하면 좋다.
  - 상단 문제 문장 영역
  - 중앙 표 영역
  - 표 아래 기호 설명 영역
  - 하단 선택지 영역
  - 최하단 해설 영역
- 중앙 표는 다음 구조로 잡는다.
  - 전체 테두리 직사각형 1개
  - 상단 제목 박스 1개
  - 열 제목 행 1개
  - 혈액형 행 4개
  - 왼쪽 좁은 열, 오른쪽 넓은 그림 열
- 표 제목 `학생들의 혈액형`은 표 위에 떠 있는 별도 박스처럼 보인다.
- 혈액형 이름은 왼쪽 열에 세로로 정렬되고, 오른쪽 칸에는 아이콘들이 가로로 나열된다.
- 각 행의 아이콘은 같은 간격으로 배열되어 있으며, 행마다 개수가 다르게 보인다.
- 기호 설명은 아이콘 2개와 텍스트 2개가 한 줄에 놓인 형태로 처리하면 자연스럽다.
- 선택지는 2열처럼 흩어져 배치되어 있으므로, 좌우 간격이 넓은 보기 영역으로 구성하는 것이 좋다.
- `(정답) ③`은 하단 보조 텍스트로 별도 배치하는 것이 적절하다.
- 해설은 문단형 텍스트로, 줄바꿈이 포함된 여러 줄 구성으로 처리한다.

[semantic/solvable 힌트]
- 이 문제의 핵심 의미는 **그림그래프에서 학생 수가 많은 혈액형을 순서대로 찾는 것**이다.
- 보이는 해설 문장도 아이콘 수를 비교해 `A형`, `O형`, `B형`, `AB형` 순으로 판단한다고 설명한다.
- 다만 이 문서는 정답을 새로 추론하는 용도가 아니라, **이미지에 보이는 내용을 정리하는 용도**로만 사용해야 한다.
- 따라서 실제 채점용 의미 해석은 가능하더라도, 여기서는 계산값을 새로 만들어 쓰지 않는다.
