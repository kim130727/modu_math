[전체적인 구성]
- 화면 상단에 문제 번호와 문장이 왼쪽부터 가로로 배치되어 있다.
- 가운데에는 **보온병 → 물이 담긴 투명한 컵/비커 형태 그림 → 우유갑 → 물이 담긴 투명한 컵/비커 형태 그림** 순서로 그림이 좌우로 배열되어 있고, 각 그림 사이에 회색 화살표가 있다.
- 각 물통 그림은 채워진 물의 양을 비교하는 시각 자료로 보이며, 왼쪽 물통과 오른쪽 물통의 물 높이가 서로 같아 보이도록 그려져 있다.
- 그림 아래에는 각각 **보온병**, **우유갑**이라는 글자가 붙어 있다.
- 오른쪽 아래에는 선택지를 나타내는 듯한 **(보온병, 우유갑)** 표기가 있다.
- 하단에는 **(정답) 보온병**과 **(해설) 그릇에 담긴 물의 양이 보온병이 더 많으므로 보온병의 들이가 더 많습니다.**라는 문장이 있다.

[보이는 텍스트]
- `□ 14. 보온병과 우유갑에 물을 가득 채운 후 물을 같은 그릇에 옮겨 담았습니다. 그림과 같이 물이 채워졌을 때 보온병과 우유갑 중 들이가 더 많은 것을 선택하세요.`
- `보온병`
- `우유갑`
- `(보온병, 우유갑)`
- `(정답) 보온병`
- `(해설) 그릇에 담긴 물의 양이 보온병이 더 많으므로 보온병의 들이가 더 많습니다.`

[반복되는 시각 패턴]
- 왼쪽의 보온병 그림과 오른쪽의 우유갑 그림이 각각 **같은 형태의 물통 그림(투명한 원통형 용기)** 으로 연결되어 있다.
- 두 번 모두 **용기 → 화살표 → 물이 담긴 원통형 용기**라는 구조가 반복된다.
- 물이 담긴 용기 내부는 연한 파란색 물, 위쪽은 빈 공간, 아래쪽은 점선/곡선으로 수면이 표현되어 있다.
- 화살표는 둘 다 오른쪽 방향이며, 비교 과정이 순차적으로 제시된다.
- 그림 아래의 레이블도 **용기 이름** 형태로 반복된다.

[세부 내용]
- 문제 문장은 “보온병과 우유갑에 물을 가득 채운 후”라는 조건을 먼저 제시한다.
- 이어서 “같은 그릇에 옮겨 담았습니다”라고 하여, 두 용기의 내용물을 같은 기준으로 비교하는 상황임을 그림으로 나타낸다.
- 중앙 그림은 두 용기의 물을 각각 같은 그릇에 부었을 때의 결과를 시각적으로 보여 준다.
- 왼쪽 결과 그림과 오른쪽 결과 그림의 물 높이가 다르게 보이는 점이 핵심 시각 정보로 보인다.
- 선택해야 하는 대상은 **보온병**과 **우유갑** 두 개이며, 정답 표기는 화면 하단에 따로 존재한다.  
- 다만 DSL 작성 시에는 정답 표기를 문제 본문 요소와 분리해서 다루는 것이 좋다.

[Python DSL 작성 힌트]
- 텍스트 영역을 상단에 한 줄 또는 두 줄로 배치하고, 문제 번호 `14.`를 포함한다.
- 가운데에는 4개 요소를 가로 정렬한다:
  1. 보온병 아이콘
  2. 오른쪽 방향 화살표
  3. 물이 담긴 원통형 용기
  4. 우유갑 아이콘
  5. 오른쪽 방향 화살표
  6. 물이 담긴 원통형 용기
- 각 아이콘 아래에 라벨을 별도로 둔다: `보온병`, `우유갑`.
- 오른쪽 아래에는 선택지 표기 `(보온병, 우유갑)`를 배치한다.
- 하단 해설/정답 문장은 본문과 구분되는 작은 글씨 영역으로 처리할 수 있다.
- 물통 그림은 투명 외곽선 원통, 상단 타원, 내부 수면, 물 채움색, 바닥 점선 등의 레이어로 구성하면 좋다.
- 두 결과 그림은 같은 크기와 같은 기준선 위에 놓되, 물 높이만 다르게 표현된 것처럼 보이게 해야 한다.

[주의할 점]
- 물의 높이 차이를 숫자로 해석하거나 계산하지 말아야 한다.
- 화면에 보이는 정답/해설 문구가 있으나, 이를 문제 풀이 과정으로 오해하지 않도록 분리해야 한다.
- 보온병과 우유갑의 실제 용량을 외부 지식으로 보충하지 말고, 화면의 그림과 문장만 근거로 설명해야 한다.
- 그림 속 원통형 용기는 실제 보온병/우유갑의 형태가 아니라, 비교 결과를 나타내기 위한 표시 도구처럼 보인다.

[불확실한 부분]
- 가운데 두 개의 원통형 용기가 정확히 ‘같은 그릇’인지, 혹은 비교용 용기인지 시각적으로는 비슷하지만 엄밀한 명칭은 단정하기 어렵다.
- 오른쪽 아래의 `(보온병, 우유갑)` 표기가 선택지인지, 비교 대상 안내인지 문맥상은 보이지만 형식상 완전히 동일한 문항 표시는 아니다.
- 화면 하단의 `(정답) 보온병`과 `(해설)`은 실제 시험지 본문 일부인지, 편집용 주석인지 확정하기 어렵다.
