[문제 개요]
- 초등 수학/생활 문제 형식의 한 페이지 지면이다.
- 주어진 상황은 바나나와 오이를 양손에 들어 보았을 때 무게가 비슷해 어느 것이 더 무거운지 바로 알 수 없는 경우이다.
- 바나나와 오이의 무게를 비교할 수 있는 방법을 고르는 객관식 문제이다.
- 하단에 `(정답)②`가 인쇄되어 보이며, 해설 문단도 함께 있다.

[원본에 보이는 요소]
- 좌상단에 `□ 33.` 형태의 문제 번호가 보인다.
- 본문 문장:
  - `바나나와 오이를 양손에 들어 보니 무게가 비슷하여 어느 것이 더 무거운지 알 수 없습니다. 바나나와 오이의 무게를 비교할 수 있는 방법을 바로`
  - 다음 줄: `게 말한 것을 선택해 보세요.`
- 중앙에 그림 2개가 나란히 있다.
  - 왼쪽: 바나나 그림
  - 오른쪽: 오이 그림
- 보기 2개가 세로로 배치되어 있다.
  - `① 바나나와 오이의 길이를 각각 재어 비교해. 길이가 더 긴 쪽이 더 무거워.`
  - `② 저울의 양쪽 접시에 바나나와 오이를 각각 올려놓고 무게를 비교해. 아래`
  - 다음 줄: `로 내려간 쪽이 더 무거워.`
- 하단 왼쪽에 `(정답)②`가 보인다.
- 그 아래에 `(해설)` 문단이 이어진다.
- 해설 텍스트:
  - `바나나와 오이는 길이가 같더라도 무게가 다를 수 있으므로 길이를 재어 무게를 비교할 수는 없습니다. 양손에 들어 보았을 때 무게가 비슷한 두 물건의 무게를 비교할 때에는 저울을 사용합니다.`

[수식/관계 구조]
- 등호, 분수, 계산식, 수식 변형은 보이지 않는다.
- 문제 구조는 “상황 설명 → 보기 2개 → 정답 표시 → 해설”의 선택형 관계 구조이다.
- 비교 관계로 읽히는 핵심 문장은 다음과 같다.
  - 방법 ①: 길이를 재어 비교
  - 방법 ②: 저울로 무게를 비교
- 보기에 포함된 판단 문장:
  - `길이가 더 긴 쪽이 더 무거워.`
  - `아래로 내려간 쪽이 더 무거워.`

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 대수적 변형이 없다.
- 비교 문장만 있으며, 단계별 수식 변환은 필요하지 않다.

[빈칸 정책]
- 빈칸을 추론값으로 채우지 않는다.
- 원본에서 빈칸이 실제로 보이면 빈칸 상태 그대로 유지한다.
- 이 이미지에서는 답안을 적는 빈칸 입력란은 보이지 않고, 문제 번호 앞의 `□`만 보인다.
- `□`는 답칸으로 단정하지 말고, 문제 번호 표식의 일부일 가능성도 열어 둔다.

[반복 패턴]
- 보기 번호 `①`, `②`가 파란색 원형 아이콘처럼 반복된다.
- `(정답)`과 `(해설)`이 괄호 라벨 형태로 반복된다.
- 본문과 보기에서 줄바꿈이 자연스럽게 들어가 있어, 텍스트 박스를 여러 줄로 나누어 배치하는 패턴이 보인다.
- 두 물체 그림이 좌우로 나란히 배치된 비교형 시각 패턴이 있다.

[DSL 작성 힌트]
- 상단: 문제 번호와 본문 문장을 좌측 정렬로 배치한다.
- 본문은 2줄로 끊어진 원문 줄바꿈을 반영한다.
- 중앙: 바나나와 오이 그림을 좌우로 배치한다.
  - 두 개의 별도 이미지/도형 요소로 처리
  - 서로 비슷한 높이의 중앙 영역에 놓기
- 보기 영역:
  - ①, ②를 각각 원형 번호 아이콘과 함께 세로로 배치
  - 각 보기는 줄바꿈이 포함된 2줄 텍스트로 표현
- 하단:
  - `(정답)②`는 작은 텍스트로 좌측 배치
  - `(해설)`은 별도 문단으로 길게 이어지게 배치
- 렌더링/레이아웃:
  - 전체적으로 흰 배경의 단일 페이지
  - 본문-그림-보기-정답/해설의 순서가 분명해야 함
  - 보기 간 간격을 넉넉히 두어 시각적으로 분리
  - 해설은 작은 글씨 크기로 여러 줄 가능하게 구성
- 그림의 세부 곡선/크기는 정확 복제보다 종류와 상대 위치를 우선한다.

[semantic/solvable 힌트]
- 문제는 “어떤 방법으로 무게를 비교해야 하는가”를 묻는다.
- 원본에 `(정답)②`가 보이므로, 정답 번호가 인쇄되어 있다는 사실은 관찰 가능하다.
- 다만 풀이를 새로 계산하거나 정답을 논리적으로 재도출할 필요는 없다.
- 해설은 길이와 무게가 항상 대응하지 않음을 설명하고, 무게 비교에는 저울을 사용한다고 말한다.
- 이는 선택지 문장 의미를 이해하는 데 도움을 주는 설명이다.
