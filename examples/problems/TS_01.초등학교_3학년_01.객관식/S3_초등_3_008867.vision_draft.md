# [전체적인 구성]
- 화면 상단 왼쪽에 문제 번호 `18.`이 있고, 그 오른쪽으로 긴 문장형 문제 설명이 한 줄 이상에 걸쳐 배치되어 있습니다.
- 중앙에는 **2열 표**가 있으며, 위쪽 행은 이름, 아래쪽 행은 물의 양을 나타냅니다.
- 표 오른쪽 아래 쪽에는 선택형 응답처럼 보이는 `（ 민재 , 수현 ）` 표기가 따로 있습니다.
- 화면 하단 왼쪽에는 `（정답）민재`가 있고, 그 아래에 `（해설）`과 함께 계산식 및 문장 해설이 여러 줄로 이어집니다.
- 전체적으로 “문제 제시 → 표 정보 → 정답 표시 → 해설” 순서로 구성된 학습용 지면입니다.

# [보이는 텍스트]
- `18.`
- `민재와 수현이가 2L의 물을 어림하여 각각의 물통에 담았습니다.`
- `담은 물의 양을 실제로 재었더니 다음과 같습니다.`
- `2L에 더 가깝게 어림한 친구는 누구인지 선택하세요.`
- 표 머리글:
  - `민재`
  - `수현`
- 표 아래 수치:
  - `1850 mL`
  - `2L 200mL`
- 오른쪽의 선택 표기:
  - `（ 민재 , 수현 ）`
- 하단:
  - `（정답）민재`
  - `（해설）2L = 2000mL`
  - `민재 : 2000mL - 1850mL = 150mL`
  - `수현 : 2L 200mL - 2L = 200mL`
  - `따라서 150<200이므로 2L에 더 가깝게 어림한 친구는 민재입니다.`

# [반복되는 시각 패턴]
- 표는 **2개의 열**로 구성되어 있고, 각 열의 상단에는 이름, 하단에는 해당 물의 양이 들어가는 구조입니다.
- 두 학생의 물의 양이 나란히 비교되도록 배치되어 있어, 좌우 비교 패턴이 명확합니다.
- 하단 해설에서는 `민재 : ...`, `수현 : ...`처럼 **동일한 형식의 계산식**이 반복됩니다.
- 마지막 문장에서는 비교 결과를 부등호 형태 `150<200`로 정리한 뒤 결론을 문장으로 제시합니다.
- 숫자 단위가 `mL`, `L`처럼 섞여 나타나며, 해설에서 단위를 맞춰 비교하는 흐름이 반복적으로 드러납니다.

# [수식 단계]
- `2L = 2000mL`
- `민재 : 2000mL - 1850mL = 150mL`
- `수현 : 2L 200mL - 2L = 200mL`
- `150<200`

# [등호별 변환 의미]
- `2L = 2000mL`
  - 보이는 의미: 리터 단위를 밀리리터 단위로 바꾼 기준이 제시됩니다.
  - 시각적으로는 같은 양을 다른 단위로 나타낸 등가 변환입니다.
- `민재 : 2000mL - 1850mL = 150mL`
  - 보이는 의미: `2L`와 민재의 실제 측정값 사이의 차이를 계산한 형태입니다.
  - 등호 왼쪽은 뺄셈식, 오른쪽은 그 결과값으로 구성되어 있습니다.
- `수현 : 2L 200mL - 2L = 200mL`
  - 보이는 의미: 수현의 물의 양이 `2L`보다 얼마나 더 많은지 계산한 형태입니다.
  - 표기상 `2L 200mL`는 한 덩어리 단위 표기이고, 이를 `2L`와 비교하여 차이를 나타냅니다.
- `150<200`
  - 보이는 의미: 두 차이를 비교한 부등식입니다.
  - 시각적으로는 어떤 값이 더 작은지 보여 주는 비교 단계입니다.

# [세부 내용]
- 문제 본문은 “2L의 물을 어림하여 각각의 물통에 담았다”는 상황을 제시하고, 실제로 잰 결과를 표로 보여 줍니다.
- 표의 왼쪽 열은 `민재`, 오른쪽 열은 `수현`으로 구분되어 있습니다.
- 각 학생 아래에는 물의 양이 한 칸씩 배치되어 있으며, 민재 쪽은 `1850 mL`, 수현 쪽은 `2L 200mL`로 적혀 있습니다.
- 표는 얇은 선으로 둘러싸인 직사각형 형태이며, 머리글 영역은 연한 청록색 배경처럼 보입니다.
- 선택지처럼 보이는 `（ 민재 , 수현 ）`는 표 오른쪽 아래에 따로 떨어져 있어, 답을 고르는 위치임을 암시합니다.
- 하단 해설은 단위 변환부터 차이 계산, 비교, 결론 순서로 세로 방향으로 정리되어 있습니다.
- 정답 표시 `（정답）민재`는 해설보다 위쪽에 따로 있어, 문제의 정답을 별도로 강조합니다.

# [Python DSL 작성 힌트]
- 문제 지문은 긴 문장형 텍스트로 처리하고, 표는 **2열 2행 구조**로 분리해서 구성하면 좋습니다.
- 표 상단 셀에는 이름, 하단 셀에는 측정값 텍스트를 그대로 배치하는 방식이 적절합니다.
- 우측의 `（ 민재 , 수현 ）`는 보기 또는 선택 영역으로 따로 처리하는 것이 자연스럽습니다.
- 하단의 해설 영역은 본문과 분리된 별도 텍스트 블록으로 작성하는 편이 좋습니다.
- 수식 블록이 있다면 `2L = 2000mL`, 각 학생별 계산식, 마지막 비교식 순서로 줄바꿈하여 배치할 수 있습니다.
- 단위 `L`, `mL`가 섞여 있으므로 텍스트 렌더링 시 간격과 대소문자 표기를 원문에 가깝게 유지하는 것이 좋습니다.
