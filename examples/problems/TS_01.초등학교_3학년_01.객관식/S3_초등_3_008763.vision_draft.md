# [전체적인 구성]
- 화면 상단 왼쪽에 문제 번호와 문제 문장이 한 줄로 배치되어 있습니다.
- 중앙 오른쪽에는 저울 그림이 크게 배치되어 있고, 저울의 왼쪽 접시와 오른쪽 접시 위에 각각 물건 그림이 올려져 있습니다.
- 저울 위쪽에는 각각의 물건 이름을 적은 작은 라벨 상자가 따로 붙어 있습니다.
- 화면 하단 왼쪽에는 `(정답)` 표시와 답이 함께 보이고, 그 아래에 `(해설)` 문장이 이어집니다.
- 전체적으로 **문장 + 그림 + 정답/해설**의 순서로 구성된 초등 문제 형식입니다.

# [보이는 텍스트]
- 상단 문제 문장:
  - `65. 저울을 사용하여 물건의 무게를 비교하려고 합니다. 더 가벼운 것을 선택하세요.`
- 저울 그림 위 라벨:
  - `감자`
  - `고구마`
- 하단:
  - `(정답)고구마`
  - `(해설)고구마를 올려놓은 접시가 위로 올라갔으므로 고구마가 더 가볍습니다.`

# [반복되는 시각 패턴]
- 직접적인 수식 반복은 보이지 않습니다.
- 대신 **물건 이름 라벨 상자 2개**가 좌우로 대응되게 배치되어 있습니다.
- 저울의 양쪽 접시가 대칭 구조로 표현되어 있고, 각각 하나씩 물건이 놓여 있습니다.
- 좌우 비교를 나타내는 전형적인 **균형 저울 도식**이 사용되었습니다.

# [수식 단계]
- 해당 없음
- 등호를 포함한 수식, 변형식, 계산식은 보이지 않습니다.

# [등호별 변환 의미]
- 해당 없음
- 수식의 등호 변화가 없으므로 변환 의미를 설명할 대상이 없습니다.

# [세부 내용]
- 문제는 저울을 이용해 두 물건의 무게를 비교하는 상황입니다.
- 왼쪽 접시에는 **감자** 그림이 있고, 오른쪽 접시에는 **고구마** 그림이 있습니다.
- 저울의 기울기는 **오른쪽 접시가 더 올라가 보이도록** 그려져 있습니다.
- 그림만 보면 비교 결과를 시각적으로 유도하는 구조이며, 문장 해설에서도 같은 내용을 말하고 있습니다.
- 라벨은 각 물건 위쪽에 작은 직사각형 상자로 표시되어 있어, 어떤 그림이 어떤 물건인지 연결하기 쉽습니다.
- 화면 왼쪽 아래의 `(정답)` 표시는 답이 이미 제시된 형태로 보입니다.

# [Python DSL 작성 힌트]
- 저울 전체를 하나의 중심 도형으로 배치하고, 좌우 접시와 받침대를 분리된 요소로 그리면 좋습니다.
- 왼쪽 접시 위에 `감자` 이미지, 오른쪽 접시 위에 `고구마` 이미지를 놓는 구성을 반영합니다.
- 각 물건 이름은 접시 위쪽에 작은 라벨 박스로 배치합니다.
- 저울의 기울어진 상태를 표현할 때는 왼쪽과 오른쪽 접시의 높이 차이를 주어야 합니다.
- 하단의 정답/해설은 텍스트 블록으로 따로 배치합니다.
- 문제 문장은 상단 왼쪽 정렬, 그림은 중앙 오른쪽 정렬로 나누어 구성하면 원본과 유사합니다.

# [주의할 점]
- 정답인 `고구마`를 분석 초안 단계에서 다른 답으로 바꾸어 추론하지 않도록 주의해야 합니다.
- 저울 그림은 단순 장식이 아니라 **무게 비교의 핵심 시각 정보**이므로 방향성과 높이 차이를 잘 살려야 합니다.
- 해설 문장은 결과를 설명하지만, 초안에서는 이를 문제 풀이로 확장해 덧붙이지 않도록 합니다.
- 물건 이미지의 실제 세부 형태는 간단화된 삽화로 보이며, 세밀한 묘사는 과하지 않게 처리하는 것이 좋습니다.

# [불확실한 부분]
- 저울 중앙의 눈금이나 바늘 세부는 해상도상 아주 선명하지 않아 정확한 모양을 단정하기 어렵습니다.
- 접시 연결 구조의 작은 부품들은 일부 생략하거나 단순화해도 될 가능성이 있습니다.
- 그림의 색감은 원본 인쇄/캡처 조건에 따라 약간 달라 보일 수 있습니다.
