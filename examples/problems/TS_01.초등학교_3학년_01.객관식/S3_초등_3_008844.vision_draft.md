# [세부 내용]
- 첫 번째 저울:
  - 왼쪽 접시에 `각도기`가 놓여 있습니다.
  - 오른쪽 접시에 `컴퍼스`가 놓여 있습니다.
  - 저울의 모양상 왼쪽이 더 내려가 보이며, 시각적으로 두 물건의 무게 차이를 암시합니다.
- 두 번째 저울:
  - 왼쪽 접시에 `지우개`가 놓여 있습니다.
  - 오른쪽 접시에 `컴퍼스`가 놓여 있습니다.
  - 저울의 모양상 오른쪽이 더 내려가 보이며, 역시 비교 결과를 시각적으로 나타냅니다.
- 문제 문장:
  - “저울로 각도기, 컴퍼스, 지우개의 무게를 비교하고 있습니다.”라는 설명 뒤에,
  - “가장 가벼운 것은 무엇인지 알맞은 말을 선택해 보세요.”라는 질문이 이어집니다.
- 아래 문장:
  - `(2) 컴퍼스보다 각도기가 가볍고, 지우개보다 컴퍼스가 가벼우므로 가장 가벼운 것은 각도기입니다.`
  - 이 문장은 비교 관계를 설명하는 완성 문장처럼 보입니다.
  - 다만 화면 하단의 `각도기`가 정답 입력 결과인지, 별도 표기인지 구분이 어렵습니다.

# [Python DSL 작성 힌트]
- 저울 그림 2개를 각각 독립된 객체로 배치하는 구성이 필요합니다.
- 각 저울에 대해:
  - 받침대,
  - 중앙 축,
  - 좌우 접시,
  - 접시에 놓인 물건 그림,
  - 물건 이름 라벨
  을 따로 표현하면 좋습니다.
- 상단 문제 문장은 2줄로 줄바꿈을 유지해서 배치하는 것이 자연스럽습니다.
- 선택지 문장은 괄호와 쉼표를 그대로 살려 텍스트 박스로 넣는 방식이 적절합니다.
- 마지막의 `각도기` 텍스트는 인쇄된 답안인지 사용 흔적인지 불확실하므로, 필요 시 별도 주석 처리하는 것이 안전합니다.
- 저울의 기울어진 방향은 이미지 기반으로 재현해야 하며, 비교 결과를 글로 단정하지 말고 시각적 방향만 먼저 묘사하는 것이 좋습니다.

# [주의할 점]
- 문제를 풀어 정답을 추론해 적지 마세요.
- “가장 가벼운 것”의 결론을 시각 분석 문서에서 단정적으로 다시 쓰지 마세요.
- 저울의 어느 쪽이 더 무거운지에 대한 해석은, 그림의 기울기를 근거로 한 시각 정보와 분리해서 설명하는 것이 안전합니다.
- 하단의 `각도기`는 화면에 보이지만, 그것이 문제 본문인지, 정답 표시인지, 캡처 후 남은 흔적인지 확정하지 마세요.
- 숫자 `(2)`는 문항 번호의 일부로 보이나, 보기/해설 번호일 가능성도 있으므로 맥락을 분리해서 기록하세요.

# [불확실한 부분]
- 하단의 `각도기` 텍스트가 인쇄된 정답인지, 화면에 우연히 포함된 별도 텍스트인지 불확실합니다.
- 두 저울의 정확한 기울기 정도는 이미지 해상도 때문에 세밀하게 단정하기 어렵습니다.
- `가`, `나` 표기가 각 저울의 이름인지, 그림 구분용 라벨인지 완전히 확정되지는 않습니다.
- 첫 번째 저울과 두 번째 저울에서 접시의 높낮이 차이를 정확한 수치로 읽을 수는 없습니다.
