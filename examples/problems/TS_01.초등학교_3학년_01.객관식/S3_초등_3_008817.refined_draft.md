[문제 개요]
- 한 페이지짜리 초등 수학 문제로 보입니다.
- 문제 번호 `50.`이 상단 왼쪽에 보입니다.
- 문장형 지문이 상단에서 왼쪽→오른쪽으로 이어집니다.
- 가운데에는 `가`, `나` 두 개의 같은 모양의 그릇 그림이 나란히 있습니다.
- 오른쪽 아래에는 `( 가, 나 ) 그릇` 형태의 표기가 보입니다.
- 하단에는 `(정답)`과 `(해설)`이 인쇄되어 있습니다.

[원본에 보이는 요소]
- 보이는 텍스트:
  - `50.`
  - `가 그릇과 나 그릇에 물을 가득 채운 후 모양과 크기가 같은 그릇에 옮겨 담았더니 그림과 같이 물이 채워졌습니다. 들이가 더 많은 그릇을 선택해 보세요.`
  - 원 안 글자: `가`, `나`
  - `( 가, 나 ) 그릇`
  - `(정답) 나 그릇`
  - `(해설)모양과 크기가 같은 그릇 담았을 때 물의 높이가 더 높은 나 그릇이 들이가 더 많은 그릇입니다.`
- 이미지상 보이는 배치:
  - 위쪽: 문장형 지문 1개
  - 중앙: 투명한 직육면체/정육면체 느낌의 그릇 2개
  - 왼쪽 그릇에 `가`, 오른쪽 그릇에 `나` 원형 라벨
  - 오른쪽 아래: 선택지/응답처럼 보이는 `( 가, 나 ) 그릇`
  - 하단 왼쪽: 정답/해설 텍스트 블록

[수식/관계 구조]
- 등호나 계산식은 보이지 않습니다.
- 수량 비교 구조로 해석될 수 있습니다.
- 보이는 관계:
  - 두 그릇의 모양과 크기는 같음
  - 물의 높이는 `나` 쪽이 더 높게 보임
  - 따라서 “들이가 더 많은 그릇”을 고르는 형식의 비교 문제로 보임
- 주의:
  - 이는 문제를 푸는 과정이 아니라, 화면에 보이는 비교 관계를 정리한 것입니다.

[등호별 변환 의미]
- 해당 없음.
- 등호(`=`)를 포함한 변형 단계가 보이지 않습니다.
- 계산식 전개도 보이지 않습니다.

[빈칸 정책]
- 빈칸에 해당하는 수치나 답을 추론해서 넣지 않습니다.
- 원본에 실제로 인쇄되어 보이는 경우에만 그 사실을 기록합니다.
- 이 문제는 빈칸형 수식 문제로 보이지 않으므로, 별도의 빈칸 슬롯은 보이지 않습니다.
- 만약 DSL에서 선택지/응답 슬롯을 만들더라도, 추론값을 자동으로 채우지 말고 원본 표기 그대로 유지해야 합니다.

[반복 패턴]
- 동일한 형태의 그릇 그림이 2개 반복됩니다.
- 두 그림 모두:
  - 같은 외곽선 구조
  - 같은 크기
  - 같은 투명 직육면체 형태
  - 물 채움 영역만 다름
- 각 그릇 위쪽 왼편에 원형 라벨이 붙어 있습니다.
  - 왼쪽: `가`
  - 오른쪽: `나`
- 물은 연한 파란색 계열의 반투명 면으로 표현되어 있습니다.
- 내부에는 점선/실선이 섞여 입체감을 나타냅니다.

[DSL 작성 힌트]
- 상단 문장은 한 줄 또는 여러 줄 텍스트 블록으로 배치하세요.
- 중앙의 두 그릇은 가로로 나란히 정렬하세요.
  - 왼쪽: `가`
  - 오른쪽: `나`
- 두 그릇은 동일한 외형으로 그리되, 물 높이만 다르게 설정하세요.
- 원형 라벨은 각 그릇의 좌상단 근처에 작게 배치하세요.
- 오른쪽 아래의 `( 가, 나 ) 그릇`은 별도 텍스트 요소로 분리하는 것이 좋습니다.
- 하단의 `(정답)`과 `(해설)`은 서로 다른 텍스트 블록으로 분리하세요.
- 렌더링 시 다음 요소를 구분하면 좋습니다:
  - 그릇 외곽선
  - 물 채움 영역
  - 내부 점선
  - 라벨 원
  - 하단 해설 텍스트

[semantic/solvable 힌트]
- 이 문제는 두 그릇의 물 높이를 보고 더 많은 양이 들어간 그릇을 고르는 비교 문제로 읽힙니다.
- 보이는 범위에서 `나`의 물 높이가 더 높게 표현되어 있습니다.
- 단, 여기서는 정답을 새로 계산하거나 추론하지 않고, 화면에 이미 인쇄된 표기만 관찰 사실로 다룹니다.
- `(정답) 나 그릇`이 화면에 실제로 보입니다.

- 오른쪽 아래의 `( 가, 나 ) 그릇`은 선택지인지 응답용 표기인지 화면만으로는 단정하지 말고, TODO로 처리할 수 있습니다.
