[문제 개요]
- 문제 번호: `□ 26.`
- 비교 대상: `간장병`, `기름병`
- 문제 문장:  
  `간장병과 기름병에 물을 가득 채운 후 모양과 크기가 같은 그릇에 옮겨 담았습니다.`
  `그릇에 담긴 물의 양이 간장병과 기름병 중 들이가 더 많은 것을 선택하세요.`
- 하단에 `(정답)`과 `(해설)` 영역이 함께 인쇄되어 있음.
- 보이는 정답 텍스트: `(정답)간장병`
- 보이는 해설 텍스트: `(해설)그릇에 담긴 물의 양이 간장병이 더 많으므로 간장병의 들이가 더 많습니다.`

[원본에 보이는 요소]
- 상단 왼쪽에 문제 번호와 문장 텍스트가 배치됨.
- 중앙에는 원통형 물통 그림 2개가 가로로 나란히 배치됨.
  - 왼쪽 그림 아래 라벨: `간장병`
  - 오른쪽 그림 아래 라벨: `기름병`
- 각 물통은 파란색 계열의 외곽선과 물 표현을 사용함.
- 각 물통 내부에 물 표면이 타원형으로 표현되어 있음.
- 물통 아래쪽에는 점선 형태의 곡선이 보임.
- 하단 왼쪽에 `(정답)`과 `(해설)`이 줄을 나누어 배치됨.

[수식/관계 구조]
- 등호를 포함한 수식이나 계산식은 보이지 않음.
- 구조는 “문장 제시 → 두 대상 그림 비교 → 정답/해설 제시”의 비교형 문제임.
- 비교 관계는 다음과 같이 읽힘:
  - 물을 가득 채운 뒤 같은 그릇에 옮겨 담았을 때
  - 그릇에 담긴 물의 양이 더 많은 쪽 = 들이가 더 많은 쪽
- 그림상 물 높이 차이를 통해 양의 많고 적음을 비교하는 형태임.
- 수치값, 단위, 계산식은 제시되지 않음.

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 식 변형 단계가 없다.
- 계산 과정 설명이 아니라, 그림 비교를 통한 선택 문제임.

[빈칸 정책]
- 문제 본문에는 빈칸/식의 빈 슬롯이 보이지 않음.
- 문제 번호 앞의 `□`는 빈칸으로 단정하지 말고, 기호 또는 체크박스성 표기로 유지할 것.
- 원본에 정답이 인쇄되어 보이더라도, DSL 작성 시 추론으로 빈칸을 채우지 말 것.
- 정답 텍스트가 보이는 경우에도, 이는 관찰 사실로만 기록하고 빈칸 정책과 분리할 것.

[반복 패턴]
- 물통 그림 2개가 같은 형태로 반복됨.
  - 둘 다 원통형 용기
  - 둘 다 위쪽이 타원형 입구처럼 표현됨
  - 둘 다 파란색 테두리와 물 색 사용
- 텍스트 라벨도 2개가 반복되어 각 그림 아래에 대응됨.
- 좌우 대칭에 가까운 배치이지만, 물 높이만 다르게 표현되어 비교 대상임을 강조함.
- 하단의 `(정답)`, `(해설)`도 같은 형식의 접두 표기가 반복됨.

[DSL 작성 힌트]
- 상단 문장은 좌측 정렬 텍스트 블록으로 배치하되, 원본처럼 2줄 이상으로 줄바꿈을 반영하면 좋음.
- 중앙 그림 영역은 좌우 2개의 동일 스타일 용기 도형으로 구성.
  - 왼쪽: 간장병
  - 오른쪽: 기름병
- 두 용기의 크기와 외형은 최대한 같게 두고, 물의 높이만 다르게 표현.
- 물 표면은 타원형 곡선으로 처리하고, 내부 하단의 점선 곡선도 함께 반영.
- 각 그림 아래 라벨은 중앙 정렬로 배치.
- 하단의 `(정답)`과 `(해설)`은 별도 텍스트 블록으로 두고, 본문과 분리해서 배치.
- 표, 분수 슬롯, 계산 박스는 없음.
- 전체 레이아웃은 “문장 상단 / 그림 중앙 / 정답·해설 하단”의 3단 구조로 잡으면 좋음.

[semantic/solvable 힌트]
- 문제의 핵심 의미는 “같은 그릇에 옮겨 담았을 때 물이 더 많이 담긴 쪽이 들이가 더 많다”는 비교 판단임.
- 시각적으로 왼쪽 간장병의 물 높이가 오른쪽보다 더 높게 그려져 있음.
- 따라서 원본의 인쇄 정답은 `간장병`으로 보임.
- 다만 DSL 작성 관점에서는 이 결론을 새로 계산해 넣기보다, 보이는 텍스트와 그림 관계를 기록하는 데 집중할 것.

[주의 및 TODO]
- `□` 기호의 정확한 의미는 불확실하므로 체크박스/문항 기호로만 취급할 것.
- 물통 내부 점선 곡선의 의미는 바닥 표현인지 장식인지 단정하지 말 것.
- 정답/해설 영역은 원본에 인쇄된 텍스트로 보이지만, 이를 편집용 정보인지 학생용 인쇄 요소인지 확정하지 말 것.
- 문제 문장의 줄바꿈은 이미지 해상도와 원본 판독 방식에 따라 약간 달라질 수 있으므로 유연하게 처리할 것.
- 색상값, 선 굵기, 정확한 위치는 단정하지 말고 상대 배치 중심으로 기록할 것.
