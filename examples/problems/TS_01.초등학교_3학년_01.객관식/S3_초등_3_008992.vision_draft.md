# [전체적인 구성]
- 문제 번호 **30번**이 왼쪽 상단에 있고, 그 오른쪽에 지시문 **“직각삼각형을 찾아 기호를 선택하세요.”**가 보입니다.
- 오른쪽에는 **연한 파란색 격자 배경** 위에 삼각형 3개가 가로로 배치되어 있습니다.
- 각 도형 안에는 각각 **“가”**, **“나”**, **“다”**라는 기호가 들어가 있습니다.
- 격자 아래에는 선택지처럼 **“( 가, 나, 다 )”**가 적혀 있습니다.
- 왼쪽 하단에는 **“(정답)가”**, 그 아래에 **“(해설) 한 각이 직각인 삼각형은 가입니다.”**라는 문장이 있습니다.

# [보이는 텍스트]
- `30.`
- `직각삼각형을 찾아 기호를 선택하세요.`
- `가`
- `나`
- `다`
- `( 가, 나, 다 )`
- `(정답)가`
- `(해설) 한 각이 직각인 삼각형은 가입니다.`

# [반복되는 시각 패턴]
- 삼각형 도형이 **3개** 반복되어 나타납니다.
- 각 삼각형은 모두 **선분 3개로 이루어진 외곽선** 형태입니다.
- 세 도형 모두 같은 격자 배경 위에 놓여 있어, **각도와 변의 기울기 비교**가 가능해 보입니다.
- 각 도형 내부에는 각각 한 글자씩 **기호 라벨**이 들어가 있으며, 도형과 기호가 1:1로 대응됩니다.

# [수식 단계]
- 해당 없음.  
- 문제는 수식 변형이나 등호 전개가 아니라, **도형의 성질을 보고 기호를 고르는 형식**입니다.

# [등호별 변환 의미]
- 해당 없음.  
- 등호를 사용한 단계적 변형식은 보이지 않습니다.

# [세부 내용]
- 오른쪽의 첫 번째 도형은 왼쪽에 거의 세로에 가까운 변, 위쪽에서 아래쪽으로 내려오는 사선 변, 아래쪽의 가로에 가까운 변으로 이루어진 삼각형처럼 보입니다.
- 두 번째 도형은 가는 사선 두 변이 위쪽에서 만나 아래로 열리는 형태이며, 내부에 **“나”**가 표시되어 있습니다.
- 세 번째 도형은 왼쪽과 오른쪽 변이 위쪽 꼭짓점에서 만나는 삼각형이며, 내부에 **“다”**가 표시되어 있습니다.
- 격자선은 도형의 각도와 길이를 비교하는 보조 배경으로 보입니다.
- 아래의 `( 가, 나, 다 )`는 세 도형 중 하나를 답으로 고르도록 제시한 선택 영역처럼 보입니다.
- 하단의 해설 문장은 정답이 **가**라고 제시하고 있으나, 이 문장은 시각자료에 함께 인쇄된 설명이므로 도형 자체의 배치와 구분해서 봐야 합니다.

# [Python DSL 작성 힌트]
- 전체 영역을 **문제 상단 텍스트 / 우측 도형 영역 / 하단 정답·해설 영역**으로 나누어 배치하면 좋습니다.
- 도형 영역은 연한 파란색 **격자 직사각형**을 먼저 그리고, 그 위에 삼각형 3개를 좌→우 순서로 배치합니다.
- 각 삼각형에는 내부 텍스트를 별도로 넣어 **가, 나, 다**를 중앙 근처에 배치합니다.
- 아래쪽에는 선택지 텍스트 **`( 가, 나, 다 )`**를 도형 영역 하단 중앙에 넣습니다.
- 왼쪽 하단의 정답과 해설은 본문과 분리된 텍스트 블록으로 처리하면 구조가 잘 드러납니다.
- 도형의 꼭짓점 위치는 격자 기준으로 상대 좌표를 사용해 재현하는 방식이 적절합니다.

# [주의할 점]
- **정답을 시각 분석 단계에서 추론해 적지 마세요.**
- “직각삼각형”이라는 문구와 해설이 보이더라도, 초안에서는 **보이는 사실**과 **추가로 읽히는 의미**를 구분해서 써야 합니다.
- 삼각형의 정확한 각도는 그림만으로 판단되지만, 분석 초안에서는 **각이 직각처럼 보인다** 정도의 관찰 표현에 그치는 것이 안전합니다.
- 격자선이 있으므로 실제 위치를 단정할 때는 **선의 겹침 여부**를 주의해서 봐야 합니다.
- 선택지 `( 가, 나, 다 )`는 답안란이 아니라 **기호 목록**으로 보입니다.

# [불확실한 부분]
- 세 삼각형의 꼭짓점과 변이 격자선에 정확히 어느 칸에 맞춰지는지는 이미지 해상도상 다소 불명확합니다.
- 각 도형의 내부 글자 위치가 정확한 중앙인지, 약간 왼쪽/오른쪽으로 치우쳤는지는 확대 없이 완전히 단정하기 어렵습니다.
- 왼쪽 하단의 `(정답)가`와 해설 문장은 화면 일부를 차지하므로, 실제 원본 편집에서는 위치가 조금 다를 가능성이 있습니다.
