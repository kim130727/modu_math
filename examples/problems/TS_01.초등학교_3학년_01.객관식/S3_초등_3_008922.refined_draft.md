[문제 개요]
- 문제 번호는 `88.`로 보인다.
- 지문은 월별 휴대전화 판매량을 그림그래프로 나타낸 자료를 보고, **휴대전화가 많이 팔린 달부터 순서대로** 나열한 것을 고르는 객관식 문제이다.
- 표 제목은 `월별 휴대전화 판매량`이다.
- 정답 표시는 화면에 `정답 ④`로 보인다. 다만 이는 **이미지에 보이는 사실**로만 기록하고, 풀이로 확정해 계산하지는 않는다.
- 하단 해설에는 `이 많은 달부터 순서대로 쓰면 11월, 10월, 12월, 9월입니다.`라는 문장이 보인다. 이것도 화면에 인쇄된 사실이다.

[원본에 보이는 요소]
- 상단 지문:
  - `어느 가게의 월별 휴대전화 판매량을 조사하여 그림그래프로 나타내었습니다. 휴대전화가 많이 팔린 달부터 순서대로 나열한 것을 고르세요.`
- 표 영역:
  - 제목: `월별 휴대전화 판매량`
  - 열 머리글: `월`, `판매량`
  - 행 라벨: `9월`, `10월`, `11월`, `12월`
- 범례:
  - 큰 휴대전화 아이콘 옆에 `10대`
  - 작은 휴대전화 아이콘 옆에 `1대`
- 보기:
  - `① 9월, 10월, 11월, 12월`
  - `② 10월, 12월, 11월, 9월`
  - `③ 11월, 12월, 10월, 9월`
  - `④ 11월, 10월, 12월, 9월`
- 하단 표기:
  - `(정답 ④)`
  - `(해설)`
- 해설 문장:
  - 큰 휴대전화 아이콘 뒤에 `이 많은 달부터 순서대로 쓰면 11월, 10월, 12월, 9월입니다.`가 이어져 보인다.

[수식/관계 구조]
- 이 문제는 산술식 전개나 등식 변형이 아니라, **그림그래프의 범례를 이용해 월별 판매량의 상대적 크기를 비교하는 구조**이다.
- 큰 휴대전화 그림은 `10대`, 작은 휴대전화 그림은 `1대`를 뜻하는 범례가 따로 제시되어 있다.
- 각 월의 판매량은 아이콘의 **종류와 개수**로 표현된다.
- 보기들은 모두 `4개의 월 이름을 쉼표로 나열`한 동일한 형식이다.
- 따라서 핵심 관계는:
  - 범례 해석
  - 각 월 행의 아이콘 조합 읽기
  - 월별 판매량의 크기 비교
  - 가장 많은 달부터 순서 선택
- 숫자 계산식, 분수, 등호 연쇄는 보이지 않는다.

[등호별 변환 의미]
- 해당 없음.
- 원본에는 `=` 기호가 들어간 식이나, 단계별 등식 변형이 보이지 않는다.
- 따라서 등호별 변환 설명은 필요하지 않다.

[빈칸 정책]
- 빈칸 문제는 아니다.
- 선택지와 해설 문장이 모두 인쇄되어 있으며, 별도의 빈칸 슬롯은 보이지 않는다.
- 원본에 정답이 `④`로 인쇄되어 보이지만, DSL 작성 시에도 **정답 추론값을 임의로 빈칸에 넣지 말고**, 보이는 텍스트만 그대로 기록하는 정책을 유지한다.
- 만약 DSL에서 답안 입력란이 필요하더라도, 이 이미지에서는 그런 입력란이 확인되지 않으므로 새로 만들지 않는다.

[반복 패턴]
- 각 달의 판매량은 휴대전화 아이콘이 여러 번 반복되는 형태로 표현된다.
- 반복되는 아이콘은 2종류이다.
  - 큰 휴대전화 아이콘: `10대`
  - 작은 휴대전화 아이콘: `1대`
- 표의 각 월 행은 같은 구조를 공유하지만, 아이콘의 배열과 개수는 월마다 다르다.
- 보기 ①~④는 모두 `월 이름 4개`를 쉼표로 연결한 문장 패턴이다.
- 하단 해설도 텍스트 사이에 아이콘 1개가 끼어드는 혼합 패턴으로 보인다.

[DSL 작성 힌트]
- 문제를 다음 덩어리로 나누어 기록하는 것이 좋다.
  - 지문 텍스트
  - 표 제목
  - 표의 열 이름
  - 각 월 행 라벨과 해당 행의 아이콘 배열
  - 범례 영역
  - 보기 4개
  - 정답 표시와 해설 영역
- 표는 `왼쪽 라벨 열 + 오른쪽 아이콘 열`의 2열 구조로 모델링하면 적합하다.
- 표 내부는 행 단위로 분리하고, 각 행에 대해
  - 큰 아이콘 몇 개
  - 작은 아이콘 몇 개
  - 아이콘의 좌우 나열 순서
  를 따로 기록하는 것이 좋다.
- 범례는 표 오른쪽에 독립 배치되어 있으며, 위아래 2줄 구성이다.
- 보기 영역은 표 아래쪽에 가로로 배치되어 있고, 각 선택지는 파란색 원형 숫자와 텍스트로 구성된다.
- 해설은 하단 왼쪽에 따로 있으며, 아이콘 1개가 문장 앞부분과 결합된 형태로 보인다.
- 렌더링/레이아웃 힌트:
  - 제목은 표 위 중앙 정렬
  - 표는 외곽선이 있는 직사각형
  - 행 구분선과 열 구분선이 있음
  - 범례는 표의 오른쪽 바깥쪽
  - 보기는 두 줄에 걸쳐 좌우로 분산 배치
  - 정답/해설은 본문보다 아래에 별도 블록으로 배치

[semantic/solvable 힌트]
- 이 문제의 의미는 그림그래프를 해석하여 **월별 판매량의 크기 순서를 찾는 것**이다.
- 범례를 먼저 읽어 큰 아이콘과 작은 아이콘의 값 차이를 이해한 뒤, 각 월의 판매량을 비교해야 한다.
- 보이는 해설 문장은 `11월, 10월, 12월, 9월` 순서를 제시한다.
- 다만 이는 원본에 인쇄된 해설 사실로만 취급하고, 이 문서에서 새로 계산해 결론을 내린 것으로 쓰지는 않는다.
- 정답 표시 `④`도 화면에 보이는 사실이다.
