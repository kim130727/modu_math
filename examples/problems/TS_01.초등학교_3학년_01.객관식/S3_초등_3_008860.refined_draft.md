[문제 개요]
- 문제 번호는 `7.`이다.
- 문제는 세숫대야와 바가지의 들이(용량)를 비교하여, 어느 것이 더 많은지 묻는 선택형 문항이다.
- 상단에는 긴 질문 문장이 왼쪽에서 오른쪽으로 배치되어 있다.
- 중앙에는 두 물건의 물양을 컵 개수로 비교한 그림이 위아래 2줄로 제시된다.
- 오른쪽 아래에는 선택지 형태의 괄호 문구가 보인다.
- 왼쪽 아래에는 `(정답)`, `(해설)` 표시가 포함된 텍스트가 인쇄되어 있다.
- 이 하단 텍스트는 문제 본문과 분리된 해설 영역처럼 보인다.

[원본에 보이는 요소]
- 보이는 문제 번호: `7.`
- 보이는 문제 문장:
  - `세숫대야와 바가지에 물을 가득 채운 후 물양과 같은 컵에 옮겨 담았더니 다음과 같았습니다. 세숫대야와 바가지 중에서 어느 것이 들이가 더 많을까요?`
- 중앙 그림:
  - 위쪽 줄: 주황색 세숫대야 그림 → 회색 오른쪽 화살표 → 파란 컵 여러 개
  - 아래쪽 줄: 분홍색 바가지 그림 → 회색 오른쪽 화살표 → 파란 컵 여러 개
- 오른쪽 아래 선택지:
  - `( 세숫대야 , 바가지 )`
- 왼쪽 아래 해설/정답 표기:
  - `(정답)세숫대야`
  - `(해설)세숫대야는 9컵, 바가지는 3컵이므로 세숫대야의 들이가 더 많습니다.`
- 원본 이미지 기준으로, 위쪽 컵은 9개, 아래쪽 컵은 3개로 보인다.
- 정답 문구는 화면에 실제로 인쇄되어 보인다.

[수식/관계 구조]
- 이 문제는 등호를 포함한 수식 전개가 아니라, 그림의 컵 개수 비교로 관계를 나타낸다.
- 관계 구조는 다음과 같이 읽을 수 있다.
  - 세숫대야 → 9컵
  - 바가지 → 3컵
  - 9컵과 3컵을 비교하여 세숫대야가 더 큼을 나타냄
- 숫자 비교의 결과는 해설 문장에만 직접 적혀 있고, 본문에서는 그림 비교가 핵심이다.

[등호별 변환 의미]
- 화면에 등호(`=`)를 이용한 변형 연쇄는 보이지 않는다.
- 따라서 단계별 등호 변환 설명은 해당 없음.
- 비교 의미는 “그 물건의 들이를 컵 개수로 옮겨 나타낸 것”으로 이해하면 된다.

[빈칸 정책]
- 빈칸을 추론값으로 채우지 않는다.
- 선택지 괄호 `( 세숫대야 , 바가지 )`는 원본에 보이는 그대로 유지한다.
- 정답이 원본에 인쇄되어 보이더라도, DSL 작성 시 빈칸이나 응답칸을 임의로 완성하지 않는다.
- 해설에 있는 수치 역시 본문 빈칸을 메우는 용도로 재구성하지 않는다.

[반복 패턴]
- 동일한 비교 구조가 두 번 반복된다.
  - 물건 그림
  - 오른쪽 화살표
  - 파란 컵 배열
- 위아래 두 줄 모두 같은 시각 패턴을 사용한다.
- 파란 컵은 동일한 모양의 반복 아이콘으로 표현된다.
- 위쪽은 더 많은 컵, 아래쪽은 더 적은 컵으로 배치되어 상대적 크기 차이를 직관적으로 드러낸다.
- 컵들은 가로로 연속 배치되며, 일부는 줄바꿈되어 두 줄처럼 보이도록 배치되어 있다.

[DSL 작성 힌트]
- 전체를 `상단 질문문장 / 중앙 비교 그림 2행 / 하단 선택지 및 해설`의 3영역으로 나누어 배치하는 것이 적절하다.
- 상단 질문문장은 한 줄이 길므로 자동 줄바꿈 또는 다줄 텍스트 박스로 처리한다.
- 중앙 비교 그림은 2행 구조로 구현한다.
  - 1행: 주황색 세숫대야 + 회색 오른쪽 화살표 + 파란 컵 9개
  - 2행: 분홍색 바가지 + 회색 오른쪽 화살표 + 파란 컵 3개
- 그림 요소 사이의 간격을 일정하게 두고, 화살표는 두 행에서 같은 스타일로 맞춘다.
- 컵은 반복 아이콘으로 만들되, 각 행의 개수가 정확히 드러나도록 간격과 줄바꿈을 조절한다.
- 오른쪽 아래 괄호 선택지는 별도의 텍스트 블록으로 배치한다.
- 왼쪽 아래의 `(정답)`, `(해설)`은 본문과 분리된 작은 텍스트 블록으로 처리한다.
- 정답이 보이는 경우에도 문제 본문 구성요소와는 독립적으로 취급한다.
- 주황색 세숫대야와 분홍색 바가지의 좌우 위치는 원본처럼 유지한다.

[semantic/solvable 힌트]
- 보이는 해설 문구에 따르면 세숫대야는 9컵, 바가지는 3컵이다.
- 따라서 비교 대상은 두 용기의 들이이며, 컵 개수로 용량을 표현한 문제이다.
- 문제의 핵심 의미는 “더 많은 컵으로 옮겨진 쪽이 더 큰 들이”라는 비교이다.
- 다만 DSL 작성에서는 정답을 추론해 넣지 말고, 관찰된 인쇄 정보만 반영한다.
- 원본에 정답 `세숫대야`가 인쇄되어 보이므로, 사실로는 기록 가능하다.
