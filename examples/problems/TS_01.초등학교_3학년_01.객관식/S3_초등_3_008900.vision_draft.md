# [전체적인 구성]
- 화면 상단에는 **문제 설명 문장**이 한 줄(실제로는 줄바꿈 포함)로 있고, 그 아래에 **과일 선호도 그림 자료**가 큼직한 박스 형태로 배치되어 있습니다.
- 그림 자료 아래에는 **‘좋아하는 과일별 학생 수’ 표**가 가로로 넓게 놓여 있습니다.
- 표 아래쪽에는 **문항 번호 66번**과 선택형 질문이 있으며, 오른쪽에 **선택지 (딸기, 귤, 포도, 사과)**가 괄호로 제시되어 있습니다.
- 맨 아래에는 **(정답) 사과**, **(해설)** 문장이 이어집니다.
- 전체적으로 위에서 아래로:
  1. 문제 소개
  2. 그림 그래프
  3. 표
  4. 선택 문제
  5. 정답/해설
  의 순서로 구성됩니다.

# [보이는 텍스트]
- 상단 문장:
  - `기태네 반 학생들이 좋아하는 과일을 조사하여 표로 나타내었습니다. 물음에 답하세요.`
- 그림 상자 제목:
  - `좋아하는 과일`
- 그림 안 과일 이름:
  - `딸기`
  - `귤`
  - `포도`
  - `사과`
- 범례:
  - 파란색 점 옆 `남학생`
  - 빨간색 점 옆 `여학생`
- 표 제목:
  - `좋아하는 과일별 학생 수`
- 표 머리글:
  - `과일`
  - `딸기`
  - `귤`
  - `포도`
  - `사과`
  - `합계`
- 표의 행 이름:
  - `남학생 수(명)`
  - `여학생 수(명)`
- 표 안에 보이는 숫자:
  - 남학생 수(명) × 딸기 칸에 `2`
  - 여학생 수(명) × 딸기 칸에 `3`
  - 여학생 수(명) × 사과 칸에 `4`
  - 여학생 수(명) × 합계 칸에 `11`
- 문항:
  - `66. 기태네 반에서 과일 간식을 한 가지만 준비하면 어떤 과일을 준비하면 좋을까요? 알맞은 것을 선택해보세요.`
- 선택지:
  - `( 딸기 , 귤 , 포도 , 사과 )`
- 하단 텍스트:
  - `(정답)사과`
  - `(해설)0에) 기태네 반 학생들이 가장 많이 좋아하는 과일은 사과이므로 과일 간식으로 사과를 준비하는 것이 좋습니다.`
  - 해설 문장은 화면에서 일부 어색하게 보이거나 줄바꿈이 포함되어 있습니다.

# [반복되는 시각 패턴]
- 그림 자료는 **2행 2열로 나뉜 큰 사각형** 안에 과일별 영역이 배치된 형태입니다.
- 각 과일 영역마다 **남학생(파란 점)**과 **여학생(빨간 점)**이 여러 개의 원형 점으로 표시되어 있습니다.
- 과일 이름은 각 칸의 **왼쪽 위 근처에 작은 둥근 말풍선/라벨 형태**로 들어가 있습니다.
- 표는 **열 6개, 행 3개 정도의 격자 구조**로 보이며, 머리글 행이 연한 연두색 배경으로 강조되어 있습니다.
- 그림의 점과 표의 숫자 사이에는 **시각적으로 대응되는 패턴**이 있어, 그림에서의 점 개수와 표의 수가 연결되는 자료로 보입니다.
- 범례는 점의 색과 성별을 연결하는 반복 패턴을 제공합니다.
  - 파란색 원 = 남학생
  - 빨간색 원 = 여학생

# [수식 단계]
- 해당 자료에는 등호를 이용한 수식 변형 단계가 **보이지 않습니다**.
- 계산식이나 연립식처럼 여러 단계의 변환이 제시된 부분도 없습니다.
- 따라서 이 섹션은 그림/표 자료에 대해서는 별도의 수식 단계가 없는 것으로 판단됩니다.

# [등호별 변환 의미]
- 해당 없음.
- 화면에 등호(`=`)를 포함한 수식 전개가 나타나지 않으므로, 단계별 변환 의미를 설명할 대상이 없습니다.

# [세부 내용]
- 큰 그림은 **테두리가 있는 직사각형** 안에 있고, 내부가 **십자 모양의 선**으로 4칸으로 나뉘어 있습니다.
- 4칸의 배치는 다음과 같이 보입니다.
  - 왼쪽 위: `딸기`
  - 오른쪽 위: `귤`
  - 왼쪽 아래: `포도`
  - 오른쪽 아래: `사과`
- 각 칸에는 점이 여러 개 있으며, 점은 **작은 원형 아이콘**입니다.
  - 파란 점은 남학생
  - 빨간 점은 여학생
- 그림 상자 제목 `좋아하는 과일`은 상자 **상단 중앙 바깥쪽에 붙은 작은 제목 상자**처럼 보입니다.
- 표는 그림 바로 아래에 있으며, 열 순서는 왼쪽부터 `과일 / 딸기 / 귤 / 포도 / 사과 / 합계`입니다.
- 첫 번째 행은 `남학생 수(명)`, 두 번째 행은 `여학생 수(명)`입니다.
- 현재 화면에서 숫자가 채워진 칸은 일부만 보이고, 나머지 칸은 비어 있는 것으로 보입니다.
- 표의 빈칸은 문제 해결을 위해 채우는 자리로 보이지만, 여기서는 **어떤 값이 들어가는지 추론하지 않습니다**.
- 문항 66번은 “한 가지만 준비하면 어떤 과일을 준비하면 좋을까요?”라는 선택 문제이며, **보기 4개 중 하나를 고르는 형식**입니다.

# [Python DSL 작성 힌트]
- 가장 먼저 전체 구성을 **상단 설명문 / 그림 박스 / 표 / 문항 텍스트 / 정답·해설**의 5개 블록으로 나누어 배치하면 좋습니다.
- 그림 박스는 다음 요소를 별도로 분리해 그리면 좋습니다.
  - 외곽 큰 둥근 사각형 테두리
  - 내부 십자 구분선
  - 4개의 과일 라벨
  - 남학생·여학생 점 원형 아이콘
  - 범례
- 표는 **격자표 형태**로 만들고,
  - 머리글 행 배경색
  - 좌측 행 제목 셀
  - 일부 숫자 셀의 텍스트
  - 나머지 빈칸
  을 구분해서 표현하는 방식이 적절합니다.
- 선택지는 한 줄 또는 괄호형 문자열로 처리하고, 문항 본문과 약간 떨어뜨려 배치하면 원본과 비슷합니다.
- 해설 영역은 별도 텍스트 블록으로 두되, 문제 본문과 시각적으로 구분되도록 아래쪽 여백을 두는 것이 좋습니다.
- 점 자료는 색상별로 반복 배치되므로, Python DSL에서는 **원형 점 반복 배치 함수**를 쓰면 구현이 쉬울 수 있습니다.
- 표와 그림의 대응을 위해, 과일별 칸을 동일한 기준 좌표로 정렬해 두면 나중에 숫자/점 대응 작업이 편합니다.

# [주의할 점]
- **답을 추론해서 채우지 마세요.**
  - 표의 빈칸, 문항의 정답, 과일별 총합 등을 이 초안 단계에서 계산해 넣지 않아야 합니다.
- 그림 속 점의 개수를 설명할 때도, **보이는 점만 사실대로 기록**하고 계산 결과를 단정하지 않는 것이 좋습니다.
- 해설 문장은 화면 하단에 보이지만, 일부는 줄바꿈과 함께 어색하게 표시되어 있으므로 **문장 전체를 깔끔하게 재작성할 때 원문 훼손에 주의**해야 합니다.
- 선택지는 괄호 안에 나열되어 있으나, 이 단계에서는 **어느 것이 맞는지 판정하지 않습니다.**
- 표의 빈칸이 여러 개 있어 보이므로, DSL에서 빈 셀을 그냥 비워둘지, 빈칸 표시를 넣을지 결정할 때 원본과 동일한 표현을 우선해야 합니다.

# [불확실한 부분]
- 해설 문장의 시작 부분 `0에)`는 원본 인쇄/렌더링 오류처럼 보이며, 정확한 의도는 화면만으로 단정하기 어렵습니다.
- 표의 일부 숫자와 빈칸은 확대해서 보면 더 정확할 수 있으나, 현재 해상도에서는 **모든 셀의 값을 확정적으로 읽기 어렵습니다**.
- 그림 안 점의 정확한 개수는 눈으로 확인 가능하지만, 일부 점은 겹치거나 작게 보여 **세밀한 카운트는 재검증이 필요할 수 있습니다**.
- 상단 문장과 하단 해설의 줄바꿈 위치는 화면 폭에 따라 달라질 수 있어, Python DSL에서 줄 배치를 정확히 맞추려면 추가 조정이 필요합니다.
