# [전체적인 구성]
- 상단 왼쪽에 문제 번호와 지시문이 있고, 중앙에는 두 명의 어린이 캐릭터가 대화하는 그림이 배치되어 있습니다.
- 왼쪽 말풍선과 오른쪽 말풍선이 각각 한 개씩 있으며, 두 말풍선 모두 ‘반직선’의 성질을 말하는 문장이 들어 있습니다.
- 그림 아래쪽에는 `(정답)` 표시와 함께 답안 텍스트가 있고, 그 아래에 `(해설)`이 이어집니다.
- 해설 부분은 두 줄의 비교 문장으로 구성되어 있으며, 각 줄 오른쪽에 세로 중괄호처럼 묶는 표시가 있고, 그 뒤에 화살표와 결과 문장이 이어집니다.
- 전체적으로 “말풍선 선택 → 정답 제시 → 해설”의 순서로 구성된 선택형 문제 화면입니다.

# [보이는 텍스트]
- `86. 알맞은 말을 선택하세요.`
- 왼쪽 말풍선:
  - `반직선 ㄱㄴ과 반직선 ㄴㄱ은`
  - `모두 (곧은 선, 곡은 선)이야.`
- 오른쪽 말풍선:
  - `반직선 ㄱㄴ과 반직선 ㄴㄱ은`
  - `끝없이 늘인 방향이 다르므로 같다고`
  - `말할 수 (있어, 없어).`
- 본문 아래:
  - `(정답) 곧은 선, 없어`
  - `(해설)`
  - `반직선 ㄱㄴ → 시작점: 점 ㄱ`
  - `반직선 ㄴㄱ → 시작점: 점 ㄴ`
  - `같지 않습니다.`

# [반복되는 시각 패턴]
- 말풍선이 좌우 대칭처럼 2개 배치되어 있습니다.
- 두 말풍선 모두 문장 끝에 괄호가 있는 선택형 표현 `(곧은 선, 곡은 선)`, `(있어, 없어)`를 포함합니다.
- 해설에서 두 항목이 각각 `반직선 ㄱㄴ`, `반직선 ㄴㄱ` 형식으로 나란히 제시되고, 각 항목 뒤에 `→` 기호가 붙어 설명으로 이어집니다.
- 두 줄의 해설 문장 오른쪽이 하나의 묶음 표시로 연결되어 최종 결론 문장 `같지 않습니다.`로 이어집니다.
- 동일한 기호 `ㄱ`, `ㄴ`이 여러 번 반복되어 반직선의 방향 비교를 강조하는 구성입니다.

# [수식 단계]
- 이 문제 화면에는 등호를 사용한 수식 변형 단계가 보이지 않습니다.
- 대신 기호와 문장형 설명이 비교 구조로 나타납니다.

# [등호별 변환 의미]
- 해당 사항 없음.
- 화면에 `=` 형태의 등식 변환 과정이 나타나지 않습니다.

# [세부 내용]
- 왼쪽 말풍선은 “반직선 ㄱㄴ과 반직선 ㄴㄱ은 모두 (곧은 선, 곡은 선)이야.”라는 형태로, 괄호 안의 두 선택지 중 하나를 고르는 구조입니다.
- 오른쪽 말풍선은 “반직선 ㄱㄴ과 반직선 ㄴㄱ은 끝없이 늘인 방향이 다르므로 같다고 말할 수 (있어, 없어).”라는 형태로, 마지막에 선택형 괄호가 있습니다.
- 해설에서는:
  - `반직선 ㄱㄴ → 시작점: 점 ㄱ`
  - `반직선 ㄴㄱ → 시작점: 점 ㄴ`
  - 이 두 줄을 오른쪽의 묶음 표시로 하나로 묶은 뒤
  - 화살표 `→`를 통해 `같지 않습니다.`로 연결합니다.
- 그림 속 인물은 설명용 삽화로, 실제 수학 도형은 텍스트와 기호 중심으로 제시됩니다.

# [Python DSL 작성 힌트]
- 상단 제목 영역, 중앙 말풍선 2개, 하단 정답/해설 영역으로 큰 레이아웃을 나누면 좋습니다.
- 말풍선은 좌우에 각각 둥근 말풍선 박스로 배치하고, 내부 텍스트는 줄바꿈을 유지해 작성합니다.
- 해설 영역에서는 두 줄을 세로로 정렬하고, 오른쪽에 중괄호/묶음 표시를 넣은 뒤 화살표와 결론 문장을 이어 배치합니다.
- `(정답)`과 `(해설)`은 굵게 또는 소제목처럼 구분되게 두는 것이 좋습니다.
- 선택지 괄호는 실제로는 빈칸이 아니라 병렬 선택 구조이므로, 빈칸으로 처리하지 말고 그대로 선택형 텍스트로 표현합니다.

# [주의할 점]
- 답을 추론해서 특정 선택지를 새로 채워 넣지 마세요.
- 보이는 텍스트가 정확히 무엇인지와, 그것이 의미하는 바를 구분해서 작성하세요.
- 말풍선 안의 선택지는 하나가 정답처럼 이미 아래에 제시되어 있어도, 시각 분석 초안에서는 “보이는 선택지” 자체만 기록해야 합니다.
- 해설의 묶음 표시가 정확히 어떤 도형인지 단정하지 말고, “세로 중괄호처럼 보이는 묶음 표시” 정도로 표현하는 것이 안전합니다.
- `ㄱ`, `ㄴ`의 위치 관계는 그림의 텍스트로만 확인하고, 실제 선분/도형은 이 화면에 보이지 않는다는 점을 반영하세요.

# [불확실한 부분]
- 말풍선의 테두리가 정확히 원형인지, 타원형인지, 약간 찌그러진 둥근 사각형인지 화면 해상도상 완전히 단정하기 어렵습니다.
- 해설 오른쪽의 묶음 표시는 중괄호인지, 세로 괄호인지, 단순 연결선인지 해상도상 다소 अस्पष्ट합니다.
- 왼쪽/오른쪽 말풍선의 문장 줄바꿈 위치는 시각적으로 보이는 대로 정리했지만, 실제 편집본에서는 약간 달라질 수 있습니다.
- 인물 그림의 세부 표정·의상 색상은 보이지만, 문제 풀이에 핵심은 아니므로 세밀한 묘사는 생략했습니다.
