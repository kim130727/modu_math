# [전체적인 구성]
- 화면은 초등 수학 문제 1개로 보이며, 좌상단에 문제 번호 **“12.”**가 있고 그 뒤에 지시문이 한 줄로 배치되어 있습니다.
- 문제 본문 오른쪽에는 **각이 큰 그림**이 하나 제시되어 있고, 아래쪽에는 선택지/표시용으로 보이는 원형 기호가 배치되어 있습니다.
- 하단에는 **정답 표시**와 **해설 문장**이 별도로 제시되어 있습니다.
- 전체적으로 **문장형 지시 + 도형 판별 + 하단 해설** 구조입니다.

# [보이는 텍스트]
- `12. 각이면 ○, 각이 아니면 ×를 선택하세요.`
- 왼쪽 아래 근처에 작은 파란색 원형 표시가 보임
- 하단에 `(정답) ×`
- 하단 해설:
  - `(해설) 한 점에서 그은 두 반직선으로 이루어진 도형을 각이라고 합니다.`

# [반복되는 시각 패턴]
- 선택을 나타내는 기호가 **원형 표시**로 반복됩니다.
- 본문 지시문에서 **○**와 **×**가 함께 제시되어 있고,
- 화면 아래쪽에도 파란색 원 안의 기호처럼 보이는 표시가 하나 더 있으며,
- 정답 영역에는 **×**가 다시 등장합니다.
- 오른쪽 도형은 **두 개의 선분/반직선처럼 보이는 선이 한 점에서 벌어지는 형태**로 그려져 있습니다.

# [수식 단계]

- 이 문제는 등호로 이어지는 수식 변형이 아니라, **도형을 보고 ○/×를 고르는 판별형 문제**로 보입니다.

# [등호별 변환 의미]
- 해당 없음.

# [세부 내용]
- 상단 문장:
  - 문제 번호 `12.`가 왼쪽에 작게 표시되어 있습니다.
  - 지시문은 `각이면 ○, 각이 아니면 ×를 선택하세요.`라고 쓰여 있습니다.
  - 여기서 ○와 ×는 **선택 기호**로 보이며, 문제의 답을 표시하라는 의미입니다.
- 오른쪽 그림:
  - 한 점을 중심으로 위쪽과 아래쪽 방향으로 **두 개의 검은 선**이 뻗어 있습니다.
  - 두 선은 왼쪽의 한 점 근처에서 시작해 오른쪽으로 갈수록 벌어지는 형태입니다.
  - 꼭짓점처럼 보이는 점은 선들이 만나는 왼쪽 부분에 위치합니다.
- 하단 정답/해설:
  - `(정답) ×`가 보이며, 실제 선택 결과를 제시한 것으로 보입니다.
  - 해설 문장은 “한 점에서 그은 두 반직선으로 이루어진 도형을 각이라고 합니다.”라고 되어 있어, 각의 정의를 설명합니다.
  - 다만 이 문장은 설명 텍스트이며, 문제 풀이 과정을 따로 보여주지는 않습니다.

# [Python DSL 작성 힌트]
- 문제 영역을 크게 **상단 지시문**, **오른쪽 도형**, **하단 정답/해설**로 나누어 배치하면 좋습니다.
- 지시문은 한 줄 텍스트로 처리하고, `○`, `×`는 별도 기호로 표시할 수 있습니다.
- 도형은 오른쪽에 **두 개의 선분/반직선**이 한 점에서 벌어지는 형태로 그리되, 정확한 길이와 기울기는 이미지 기준으로 맞추는 것이 좋습니다.
- 하단의 `(정답) ×`와 `(해설)` 텍스트는 별도 텍스트 블록으로 배치하면 됩니다.
- 원형 표시가 파란색으로 나타나므로, 단순 텍스트 기호와 실제 UI 표시를 분리해서 표현하면 좋습니다.

# [주의할 점]
- 이 문제는 **도형이 각인지 아닌지 판별하는 형식**이므로, 답을 추론해 빈칸에 채우지 마세요.
- 해설 문장이 보이더라도, 그것을 바탕으로 문제의 선택 결과를 임의로 다시 계산하거나 설명을 보강하지 마세요.
- 오른쪽 도형의 정확한 꼭짓점 위치나 선의 길이는 이미지 해상도에 따라 약간 다르게 보일 수 있습니다.
- 파란색 원형 표시는 실제 선택 상태를 나타내는 UI 요소일 수 있으므로, 단순한 문제 본문 기호와 구분해야 합니다.

# [불확실한 부분]
- 왼쪽 아래의 작은 파란색 원형 기호가 **선택 표시인지 장식 요소인지**는 이미지상 명확히 단정하기 어렵습니다.
- 오른쪽 도형의 선이 **반직선인지 선분인지**는 그림만으로는 엄밀히 구분하기 어렵고, 해설 문장에 따르면 반직선으로 해석되는 듯하지만 시각적으로는 짧은 선으로도 보일 수 있습니다.
- 정답 표시 `(정답) ×`는 화면에 직접 출력된 정답인지, 편집용 예시인지 문맥상 완전히 확정하기는 어렵습니다.
