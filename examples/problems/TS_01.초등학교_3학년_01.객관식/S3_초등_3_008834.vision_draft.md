# [전체적인 구성]
- 한 장의 초등 수학/과학형 비교 판단 문제 화면이다.
- 왼쪽 상단에 문제 번호와 지시문이 있고, 가운데 오른쪽에는 저울 그림이 배치되어 있다.
- 저울 그림 아래쪽 중앙에는 판단 문장이 크게 제시되어 있으며, 그 아래에 정답 표시와 해설 문장이 이어진다.
- 전체적으로 **텍스트 영역(상단/하단)**과 **그림 영역(중앙)**이 분리된 구성이다.

# [보이는 텍스트]
- 상단 문제 지시문:
  - `72. 저울을 사용하여 감자와 고구마의 무게를 비교했습니다.`
  - `바르게 비교하였으면 ○표, 그렇지 않으면 ×표를 선택하세요.`
- 저울 그림 위쪽 라벨:
  - `감자`
  - `고구마`
- 그림 아래 문장:
  - `고구마가 감자보다 더 무겁습니다. ( ○, × )`
- 하단 표시:
  - `(정답)○`
  - `(해설)고구마를 올려놓은 접시가 아래로 내려갔으므로 고구마가 감자보다 더 무겁습니다.`

# [반복되는 시각 패턴]
- **둥근 괄호 형태의 선택지 표기**가 문장 끝에 붙어 있다.
  - `( ○, × )`처럼 두 기호를 나란히 보여 주고 선택하도록 유도한다.
- **라벨 박스**가 그림 위에 작게 배치되어 있다.
  - `감자`, `고구마`가 각각 별도 영역처럼 보이는 연한 배경 박스 안에 있다.
- **정답/해설 표기**가 하단에서 반복된다.
  - `(정답)`, `(해설)`이 괄호를 포함한 형태로 시작한다.

# [수식 단계]
- 해당 없음.
- 등호, 계산식, 변형식은 보이지 않는다.

# [등호별 변환 의미]
- 해당 없음.
- 수식의 단계별 변환이 제시되지 않았다.

# [세부 내용]
- 저울 그림은 **양팔저울** 형태로 보인다.
  - 가운데 기둥과 좌우 받침대, 양쪽 접시가 있다.
  - 왼쪽 접시에는 `감자`가 놓여 있고, 오른쪽 접시에는 `고구마`가 놓여 있다.
- 저울의 **오른쪽 접시가 더 아래로 내려간 모습**으로 그려져 있다.
  - 이는 그림에서 직접 관찰되는 상태이다.
- 그림 위의 라벨은 각각 대상 물체를 가리키는 용도다.
  - 왼쪽 라벨 `감자`는 왼쪽 접시의 물체와 연결되어 보인다.
  - 오른쪽 라벨 `고구마`는 오른쪽 접시의 물체와 연결되어 보인다.
- 중앙의 판단 문장은 비교 결과를 말하는 진술문 형식이다.
  - 문장 뒤에 `( ○, × )`가 붙어 있어 참/거짓 판단형 문제임을 나타낸다.
- 하단의 `(정답)○`와 `(해설)`은 인쇄된 풀이 안내처럼 보인다.
  - 다만 이 값들은 문제 풀이 결과를 포함하므로, DSL 작성 시에는 별도 정답 입력용 요소인지 확인이 필요하다.

# [Python DSL 작성 힌트]
- 전체 레이아웃을 **상단 지시문 / 중앙 그림 / 하단 진술문 / 최하단 정답·해설**의 4영역으로 나누어 배치하면 좋다.
- 저울 그림은 다음 요소로 분해해서 그리기 좋다.
  - 받침대
  - 중앙 기둥
  - 왼쪽 접시와 오른쪽 접시
  - 각 접시 위의 물체 1개씩
  - 위쪽의 물체 이름 라벨 2개
- 문장 끝의 `( ○, × )`는
  - 둥근 괄호,
  - 선택 기호 2개,
  - 쉼표
  를 포함한 작은 선택 영역으로 표현하면 된다.
- 하단의 정답/해설 텍스트는 문제 본문과 다른 스타일로 분리해 배치하는 것이 좋다.
- 그림의 핵심 시각 정보는 **오른쪽 접시가 아래로 기운 저울 상태**이므로, DSL에서 저울의 기울기 방향을 표현해야 한다.

# [주의할 점]
- 문제를 풀어서 정답을 추론하는 방식으로 작성하지 말아야 한다.
- 보이는 문장과 그림 상태를 그대로 설명하고, 비교 판단은 문제 문구 수준에서만 다뤄야 한다.
- `고구마가 감자보다 더 무겁다`는 문장은 화면에 **이미 적혀 있는 문장**으로 취급해야 하며, 이를 새로 계산해 단정적으로 재서술하지 않도록 주의한다.
- `( ○, × )`는 실제 선택지이므로, 특정 기호를 빈칸처럼 채워 넣지 말아야 한다.
- 하단의 `(정답)○`는 시각적으로 보이지만, DSL 작성 목적상 정답을 새로 생성하는 용도로 사용하지 않도록 한다.

# [불확실한 부분]
- 저울 그림의 세부적인 기울기 각도와 받침대 장식은 해상도 때문에 정확히 단정하기 어렵다.
- 라벨 박스의 정확한 크기와 여백은 이미지상 대략적이다.
- 하단 `(정답)○`가 편집용 정답 표기인지, 인쇄된 해설 일부인지 화면만으로는 완전히 구분되지 않는다.
