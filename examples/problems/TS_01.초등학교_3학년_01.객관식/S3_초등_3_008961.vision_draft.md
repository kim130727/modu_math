[전체적인 구성]
- 화면 상단에는 문제 번호와 문제 문장이 있고, 그 아래 중앙에 **혈액형별 학생 수를 그림그래프로 나타낸 표**가 배치되어 있습니다.
- 표 아래에는 **그림 기호의 의미**가 따로 제시되어 있고, 그 아래에는 **선택지 ①~⑤**가 두 줄로 나뉘어 배치되어 있습니다.
- 좌하단에는 **(정답) ③** 표시가 보이며, 맨 아래에는 **해설 문장**이 여러 줄로 이어집니다.
- 전체적으로 **위쪽: 문제 제시 / 가운데: 그림표 / 아래쪽: 보기와 해설** 구조입니다.

[보이는 텍스트]
- 문제 번호: `□ 31.`
- 문제 문장: `정호네 학교 학생 1250명의 혈액형을 조사하여 그림그래프로 나타내었습니다. 학생 수가 많은 혈액형부터 차례대로 나타낸 것을 고르세요.`
- 표 제목: `학생들의 혈액형`
- 표의 열 제목: `혈액형`, `학생 수`
- 행의 혈액형 표기: `A`, `B`, `AB`, `O`
- 기호 설명: `🙂 100명`, `😶 10명`  
  - 실제로는 웃는 얼굴 아이콘 하나와 작은 얼굴 아이콘 하나가 각각 숫자 의미와 함께 표시됨
- 선택지:
  - `① A, B, O, AB`
  - `② AB, O, B, A`
  - `③ A, O, B, AB`
  - `④ O, A, B, AB`
  - `⑤ B, O, A, AB`
- 정답 표시: `(정답) ③`
- 해설 시작: `(해설)` 이후 설명 문장들이 이어짐

[반복되는 시각 패턴]
- 표의 각 행에는 혈액형 이름 왼쪽과 오른쪽에 학생 수를 나타내는 **같은 모양의 아이콘이 반복**됩니다.
- 큰 아이콘과 작은 아이콘이 섞여 있으며, **큰 아이콘 1개 = 100명**, **작은 아이콘 1개 = 10명**이라는 대응 관계가 아래에 제시되어 있습니다.
- 각 혈액형 행에서 아이콘의 개수를 비교하도록 구성되어 있어, **행마다 반복되는 시각 단위**를 세어 비교하는 형태입니다.
- 선택지는 모두 혈액형 네 개를 **쉼표로 나열한 순서형 보기**이며, 순서만 다르게 반복 제시됩니다.

[수식 단계]
- 이 문제 화면에는 **등호를 포함한 수식 변형 단계가 보이지 않습니다.**
- 따라서 식의 전개나 변환 과정은 확인되지 않습니다.

[등호별 변환 의미]
- 해당 없음.  
- 화면에는 등호를 사용한 계산식이 제시되지 않았습니다.

[세부 내용]
- 중앙 표는 연한 살구색 계열의 배경과 테두리로 구성된 **2열 표**입니다.
- 맨 위에는 표 전체를 설명하는 제목 상자 `학생들의 혈액형`이 별도 박스처럼 떠 있는 형태로 놓여 있습니다.
- 표의 왼쪽 열은 혈액형 이름을 적는 칸이며, 오른쪽 넓은 칸에는 각 혈액형에 해당하는 아이콘들이 가로로 나열되어 있습니다.
- 혈액형 행은 위에서부터 `A`, `B`, `AB`, `O` 순서로 보입니다.
- 각 행의 아이콘은 모두 왼쪽에서 오른쪽으로 배열되어 있고, 행마다 개수가 다릅니다.
- 표 아래의 기호 설명은 아이콘과 텍스트가 한 줄에 붙어 있으며, 아이콘 크기 차이로 의미를 구분합니다.
- 선택지는 페이지 하단에 배치되어 있고, ①~⑤ 번호가 파란색 동그라미로 표시되어 있습니다.
- 선택지 ③은 왼쪽 아래와 가운데 아래가 아니라, **오른쪽 아래 보기 영역의 첫 줄 오른쪽**에 배치된 형태입니다.
- 해설 영역은 문장형 설명으로, 아이콘 수를 비교해 순서를 판단하는 과정을 글로 적고 있습니다.

[Python DSL 작성 힌트]
- 전체 레이아웃은 다음과 같이 나누어 그리면 좋습니다.
  - 상단 문제 문장 영역
  - 중앙 그림표 영역
  - 표 아래 기호 설명 영역
  - 하단 선택지 영역
  - 최하단 해설 영역
- 중앙 표는:
  - 큰 직사각형 테두리 1개
  - 상단 제목 박스 1개
  - 열 제목 행 1개
  - 혈액형 행 4개
  - 왼쪽 좁은 열과 오른쪽 넓은 열로 분할
- 각 혈액형 행의 오른쪽 칸에는 **큰 아이콘과 작은 아이콘의 반복 배치**를 그대로 표현하면 됩니다.
- 선택지는 2열처럼 보이도록 배치되어 있어, 왼쪽에 ①, ④, ⑤ 일부가 있고, 가운데/오른쪽에 ②, ③이 보입니다.
- 해설 텍스트는 줄바꿈이 포함된 문단형 배치로 처리하는 것이 좋습니다.
- 문제 문장과 해설 문장은 폰트가 비교적 큰 편이라, 표보다 넓게 문단 영역을 잡는 것이 자연스럽습니다.

[주의할 점]
- 학생 수의 실제 합이나 혈액형별 정확한 인원은 **절대 계산해서 채우지 마세요.**
- 아이콘 개수를 보더라도 그 값을 수치로 단정해 적지 말고, **보이는 개수 차이만 설명**하세요.
- 선택지 중 정답을 따로 추론해 다시 쓰지 말고, 화면에 보이는 순서 그대로 기록하세요.
- 아이콘이 실제로는 웃는 얼굴/작은 얼굴처럼 보이지만, DSL 작성 시에는 **두 종류의 기호**로만 구분해도 됩니다.
- 표의 각 행에서 아이콘 개수가 서로 다르다는 점은 보이지만, 그 차이를 수치화해 확정하지 않도록 주의하세요.

[불확실한 부분]
- 아이콘의 정확한 종류를 문자열로 어떻게 명명할지는 약간 불확실합니다. 화면상으로는 **큰 웃는 얼굴 아이콘**과 **작은 얼굴 아이콘** 두 종류로 보입니다.
- 표의 테두리 색과 제목 박스 색은 연한 살구색 계열로 보이지만, 화면 캡처 환경에 따라 약간 다르게 인식될 수 있습니다.
- 선택지의 배치는 두 줄로 보이나, 정확한 좌우 정렬 간격은 이미지 해상도에 따라 달라질 수 있습니다.
- `(정답) ③` 표시는 문제 본문에 포함된 인쇄 요소인지, 별도 주석인지 시각적으로는 함께 보이지만, DSL에서는 보조 텍스트로 처리하는 것이 적절할 수 있습니다.
