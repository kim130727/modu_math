# [문제 개요]
- 문제 번호는 **70.**
- 제목/안내 문장은 **“정사각형을 알아보려고 합니다. 물음에 답하세요.”** 로 보입니다.
- 본문 문항은 **“(1) 네 각이 모두 직각인 사각형을 모두 찾아 기호를 선택하세요.”** 입니다.
- 화면에는 **도형 5개를 보고 조건에 맞는 것의 기호를 고르는 선택형 문제**가 제시되어 있습니다.
- 원본 이미지 하단에는 작은 글씨의 해설/정답처럼 보이는 문장이 함께 보입니다.

# [원본에 보이는 요소]
## 보이는 사실
- 상단 왼쪽에 **70.** 이 있습니다.
- 상단에 **“정사각형을 알아보려고 합니다. 물음에 답하세요.”** 문장이 있습니다.
- 중앙에는 **연한 파란색 격자 배경** 위에 도형 5개가 **왼쪽에서 오른쪽으로** 나란히 배치되어 있습니다.
- 각 도형 안에는 각각 **가, 나, 다, 라, 마** 가 적혀 있습니다.
- 문항 아래쪽에 선택지처럼 **“( 가, 나, 다, 라, 마 )”** 가 큼직하게 제시되어 있습니다.
- 하단 왼쪽에는 작은 글씨로 **“(1) 네 각이 모두 직각인 사각형을 찾습니다.”** 와 **“(1) 가, 라”** 가 보입니다.

## 추론되는 수학적 의미
- 이 문제는 **정사각형/직사각형과 관련된 도형 분류**를 묻는 것으로 보입니다.
- “네 각이 모두 직각인 사각형”은 **직사각형 계열의 도형**을 찾는 조건입니다.
- 다만 답을 추론해서 확정해서는 안 됩니다.

## 렌더링/레이아웃 힌트
- 격자 배경은 **연한 하늘색 점선/실선 격자** 느낌으로 크게 잡습니다.
- 도형 5개는 **동일한 기준선 위에 좌→우 순서**로 배치합니다.
- 각 도형 내부 글자는 **중앙 부근 정렬**로 둡니다.
- 문항 문장은 도형 영역 **아래쪽 왼편 정렬**로 배치합니다.
- 선택지 괄호 표기는 문항의 **오른쪽 아래 쪽**에 크게 배치되어 있습니다.
- 하단의 작은 글씨는 본문과 분리된 **보조 텍스트 영역**으로 다루는 것이 안전합니다.

# [수식/관계 구조]
- 수식이나 등호 `=` 를 포함한 변형식은 보이지 않습니다.
- 따라서 이 문항에는 **등호 연쇄 변환 구조가 없습니다.**
- 구조는 다음과 같이 이해할 수 있습니다:
  - 조건: **네 각이 모두 직각**
  - 대상: **사각형들**
  - 작업: **조건을 만족하는 도형 기호 선택**

# [등호별 변환 의미]
- 해당 없음.
- 등호가 없으므로 단계적 변환 설명도 없습니다.

# [빈칸 정책]
- 본 문항에는 **숫자 빈칸**이 아니라 **기호를 고르는 선택 구조**가 보입니다.
- 하단의 작은 글씨에 보이는 **“가, 라”** 를 정답으로 단정하지 않습니다.
- 원본에 보이는 선택지나 해설처럼 보여도, **추론값을 채워 넣지 않고 보이는 그대로 유지**합니다.
- 만약 DSL에서 정답 표시 칸이 필요하더라도, 원문 근거 없이 **빈칸을 임의로 채우지 않습니다.**

# [반복 패턴]
- **도형 5개가 반복적으로 나열**된 구조입니다.
- 각 도형은 서로 다른 모양이며, 내부에 **한 글자 기호**가 들어 있습니다.
- 반복 패턴의 핵심은 **“기호 1개 + 도형 1개”** 조합이 5번 반복된다는 점입니다.
- 좌→우 순서가 문제 해결의 기준이 될 수 있습니다.
- 선택지는 **괄호 안에 기호를 나열하는 방식**으로 반복 패턴과 같은 기호 집합을 사용합니다.

# [DSL 작성 힌트]
- 상단 텍스트:
  - 문제 번호 **70.**
  - 제목 문장 **“정사각형을 알아보려고 합니다. 물음에 답하세요.”**
- 중앙 도형 영역:
  - 연한 파란색 격자 배경
  - 도형 5개를 가로로 배치
  - 각 도형은 검은 외곽선
  - 내부 글자는 **가 / 나 / 다 / 라 / 마**
- 도형 외형은 보이는 대로 구분해서 배치:
  - **가**: 세로로 긴 직사각형처럼 보임
  - **나**: 마름모 형태
  - **다**: 윗변과 아랫변이 평행한 사다리꼴 형태
  - **라**: 정사각형처럼 보이는 네모
  - **마**: 오른쪽으로 기울어진 사각형/사다리꼴처럼 보임
- 문항 텍스트:
  - **“(1) 네 각이 모두 직각인 사각형을 모두 찾아 기호를 선택하세요.”**
- 선택지:
  - **“( 가, 나, 다, 라, 마 )”**
- 하단의 작은 글씨는 별도 주석/해설 영역처럼 분리해 두는 것이 좋습니다.
- 도형 간 간격은 일정하게 유지하고, 각 도형의 중심에 기호를 배치합니다.
- 격자선은 도형 정렬 기준 역할을 하므로, 배경 요소로 유지합니다.

# [semantic/solvable 힌트]
- 조건 문장인 **“네 각이 모두 직각인 사각형”** 은 도형의 성질을 판단하는 기준입니다.
- 문제는 **도형의 이름을 직접 쓰는 것보다, 해당 조건을 만족하는 기호를 고르는 구조**입니다.
- 하단 작은 글씨는 해설 또는 채점용 표기일 가능성이 있으나, 화면이 일부 잘려 있어 확정할 수 없습니다.
- 정답은 원본에 인쇄된 부분만 관찰해야 하며, 여기서는 **답을 산출하지 않습니다.**
