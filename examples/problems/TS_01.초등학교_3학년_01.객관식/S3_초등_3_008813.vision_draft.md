# [세부 내용]
- 문제 문장은 **우유병에 물을 가득 채운 뒤 컵에 옮겨 담았더니 우유병에 물이 남았다**는 상황을 설명합니다.
- 이 문장 아래 중앙에 그림이 있으며:
  - 왼쪽: **우유병 그림**
    - 병 모양의 용기 내부에 물을 나타내는 파란색 부분이 아래쪽에 들어 있습니다.
    - 병 몸체 중앙 부근에 연두색 띠처럼 보이는 라벨 영역이 있고, 그 위에 **“우유”**라는 글자가 있습니다.
  - 가운데: **회색 오른쪽 화살표**
    - 왼쪽 병에서 오른쪽 컵으로 이동하는 방향을 나타냅니다.
  - 오른쪽: **컵 그림**
    - 주황색 컵 모양이며, 윗부분 안쪽에 물 표시에 보이는 파란색이 있습니다.
- 그림 아래의 명칭은 각각 **우유병 / 컵**으로, 두 대상의 비교를 돕는 레이블입니다.
- 오른쪽 중간의 `( 우유병, 컵 )`는 보기 목록처럼 보이며, 두 대상 중 하나를 고르는 형식입니다.
- 하단의 `(정답)컵`은 화면상 정답 표시로 보입니다.
- `(해설)` 뒤 문장은 컵을 먼저 가득 채웠는데도 우유병에 물이 남았다는 점을 근거로 **컵의 들이가 더 적다**고 설명합니다.
- 다만, 이는 화면에 적힌 해설 내용의 의미를 정리한 것이며, 문제 해결 과정 자체를 새로 계산한 것은 아닙니다.

# [Python DSL 작성 힌트]
- 전체 레이아웃을 크게 다음처럼 나누면 좋습니다.
  1. 상단 문제 문장 영역
  2. 중앙 비교 그림 영역
  3. 우측의 보기 영역
  4. 하단 정답/해설 영역
- 중앙 그림은 좌우 정렬로 배치:
  - 왼쪽: 병 아이콘 + 아래 캡션 `우유병`
  - 가운데: 오른쪽 방향 화살표
  - 오른쪽: 컵 아이콘 + 아래 캡션 `컵`
- 텍스트는 줄바꿈이 포함된 문장형으로 넣고, 문항 번호 `46.`을 앞에 둡니다.
- 보기 텍스트 `( 우유병, 컵 )`은 괄호 포함 문자열로 처리합니다.
- 정답/해설 영역은 별도 텍스트 박스로 배치하는 것이 자연스럽습니다.
- 그림은 실제 계산 요소가 아니라 **비교 설명용 삽화**로 취급하면 됩니다.
- 우유병과 컵의 상대적 크기 비교를 암시하는 구성이라, 캡션과 아이콘의 위치 관계를 정확히 맞추는 것이 중요합니다.
