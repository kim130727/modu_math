# [전체적인 구성]
- 흰 바탕의 한 페이지형 문제 화면이다.
- 좌상단에 문제 번호와 본문 문장이 있고, 그 아래 중앙 쪽에 바나나와 오이의 그림이 나란히 배치되어 있다.
- 그림 아래에는 선택지 2개가 세로로 배치되어 있으며, 각 선택지 앞에 동그라미 숫자 표시가 있다.
- 더 아래 왼쪽에는 `(정답)` 표시와 동그라미 `②`가 작게 보이고, 그 아래에 `(해설)` 문단이 이어진다.
- 전체적으로 **문제 설명 → 그림 → 선택지 → 정답 표시 → 해설** 순서로 구성되어 있다.

- 선택지 ②:
  - `② 저울의 양쪽 접시에 바나나와 오이를 각각 올려놓고 무게를 비교해. 아래`
  - 다음 줄에 이어서: `로 내려간 쪽이 더 무거워.`
- 하단:
  - `(정답)②`
  - `(해설)바나나와 오이는 길이가 같더라도 무게가 다를 수 있으므로 길이를 재어 무게를 비교할 수는 없습니다. 양손에 들어 보았을 때 무게가 비슷한 두 물건의 무게를 비교할 때에는 저울을 사용합니다.`

# [반복되는 시각 패턴]
- 선택지 앞의 `①`, `②`는 파란색 동그라미 숫자 아이콘처럼 보인다.
- 본문 아래에 선택지가 2개가 연속 배치되어 있어, **보기형 문제** 구조임을 알 수 있다.
- 하단의 `(정답)`과 `(해설)`은 각각 괄호로 시작하는 라벨 형식이 반복된다.
- 그림은 2개 물체를 비교하는 상황을 보여 주는 시각 패턴이며, 두 물체가 가운데 영역에 적당한 간격을 두고 배치되어 있다.

# [세부 내용]
- 문제 문장은 “바나나와 오이를 양손에 들어 보니 무게가 비슷하여 어느 것이 더 무거운지 알 수 없습니다.”라고 되어 있어, **겉으로 드러난 무게감이 비슷한 상황**을 설명한다.
- 이어서 “바나나와 오이의 무게를 비교할 수 있는 방법을 바로” 다음 줄 “게 말한 것을 선택해 보세요.”로 이어지며, 문장이 줄바꿈되어 있다.
- 그림은 바나나와 오이의 색과 형태를 구분되게 그려 놓았고, 직접 비교할 대상이 무엇인지 시각적으로 강조한다.
- 선택지 ①은 **길이를 재어 비교**하는 방법을 말하고, `길이가 더 긴 쪽이 더 무거워.`라는 문장이 함께 있다.
- 선택지 ②는 **저울의 양쪽 접시**를 사용하여 비교하는 방법을 말하고, `아래로 내려간 쪽이 더 무거워.`라는 문장이 함께 있다.
- 하단 해설은 “길이가 같더라도 무게가 다를 수 있다”는 설명과 함께, 무게 비교에는 **저울 사용**이 적절하다는 점을 서술한다.
- `(정답)②` 표시는 선택지 2번이 정답임을 알려 주지만, 이 문서는 시각 분석 초안이므로 내용 자체를 풀이로 확장하지는 않는다.

# [Python DSL 작성 힌트]
- 전체 레이아웃을 상단 본문, 중앙 그림, 하단 선택지, 맨 아래 정답/해설의 4개 영역으로 나누어 배치하면 좋다.
- 본문은 좌상단 시작, 2줄로 줄바꿈된 긴 문장 형태로 처리한다.
- 그림은 중앙 상단부에 바나나와 오이를 좌우로 나란히 두고, 각각 별도 아이콘/도형 요소로 표현한다.
- 선택지는 세로 목록 형태로, 각 항목 앞에 동그라미 숫자 라벨을 붙인다.
- 선택지 ①, ②의 본문은 줄바꿈이 들어가 있으므로, 텍스트 박스를 2줄로 나눠 배치하는 방식이 자연스럽다.
- 하단의 `(정답)②`는 작은 크기의 텍스트로 좌측에 배치한다.
- `(해설)`은 별도 문단으로 길게 이어지므로, 작은 글씨 크기와 여러 줄 텍스트 박스가 필요하다.

# [주의할 점]
- 정답은 시각적으로 보이지만, 문제 풀이 과정에서 임의로 추론해 확장하지 말아야 한다.
- 선택지 내용을 그대로 옮길 때, 문장 끝의 줄바꿈 위치를 대략 반영하면 좋다.
- 그림 속 바나나와 오이의 크기, 각도, 세부 곡선은 실제 드로잉과 완전히 일치하지 않을 수 있으므로, **대략적인 위치와 종류** 중심으로 표현하는 것이 안전하다.
- “무게가 비슷하다”는 말은 문제 상황 설명일 뿐, 실제 무게값이나 비교 결과를 새로 추론하면 안 된다.
- 저울 그림은 텍스트로 직접 보이지 않으므로, 선택지 문장에 등장하는 개념만 그대로 반영하고 별도 장식적 저울 이미지는 추가 해석하지 않는 것이 좋다.

# [불확실한 부분]
- 바나나와 오이 그림의 정확한 픽셀 위치와 크기는 화면만으로는 정밀하게 특정하기 어렵다.
- 좌상단의 `□` 기호가 체크박스인지, 문제 번호 표식의 일부인지 단정하기 어렵다.
- 본문 마지막 줄의 줄바꿈 위치는 화면 해상도에 따라 다르게 해석될 수 있다.
- `(정답)②`의 동그라미가 실제 인쇄된 기호인지, 편집상 추가된 표식인지 확실하지 않다.
- 해설 문단의 줄 간격과 정확한 문장 분리 지점은 이미지 상에서 약간 흐릿하여 세부 재현에 유의가 필요하다.
