[전체적인 구성]
- 페이지 상단에 문제 번호 **86.**이 보이고, 그 오른쪽으로 긴 문장형 문제가 한 줄 이상으로 배치되어 있습니다.
- 본문 아래에는 **연보라색 테두리의 큰 박스**가 있으며, 그 안에 **물병 그림 3개**가 좌→우로 나란히 배치되어 있습니다.
- 각 물병은 위쪽에 **원형 표시(가, 나, 다)**가 붙어 있고, 각 그림 아래에는 해당 물병의 용량이 **“1 L 700 mL”, “1 L 100 mL”, “1 L 300 mL”**처럼 텍스트로 적혀 있습니다.
- 큰 박스의 오른쪽 아래 바깥쪽에는 **괄호 안 선택지 형태**로 “( 가 물병, 나 물병, 다 물병 )”이 보입니다.
- 그 아래에는 **정답 표기**와 **해설 문장**이 인쇄되어 있습니다.

[보이는 텍스트]
- 문제 번호: **86.**
- 문제 문장: **“다음과 같이 물병에 물이 가득 채워져 있습니다. 두 물병의 물을 모아 3 L가 되게 하려면 어느 물병을 골라야 하는지 모두 선택하세요.”**
- 그림 안 표기:
  - **가**
  - **나**
  - **다**
- 각 물병 아래 용량 표기:
  - **1 L 700 mL**
  - **1 L 100 mL**
  - **1 L 300 mL**
- 선택지 표기:
  - **( 가 물병, 나 물병, 다 물병 )**
- 정답/해설로 보이는 인쇄 텍스트:
  - **(정답) 가 물병, 다 물병**
  - **(해설) 1 L 700 mL + 1 L 300 mL = 2 L + 1000 mL = 3 L이므로 가와 다 물병을 골라야 합니다.**

[반복되는 시각 패턴]
- 물병 그림이 **3개** 반복되며, 각각 비슷한 형태의 용기 그림에 용량만 다르게 적혀 있습니다.
- 각 물병 위에는 **동그라미 안 한글 라벨**이 반복되어 붙어 있습니다: **가 / 나 / 다**
- 선택지 표현도 **동일한 형식의 문구 3개**가 쉼표로 구분되어 반복됩니다.
- 용량 표기는 모두 **“1 L + mL” 구조**를 따르며, 숫자만 달라집니다.
- 시각적으로는 **큰 박스 안의 3개 그림**과, **박스 밖의 문자형 선택지**가 분리되어 있습니다.

[수식 단계] (해당될 때)
- 해설 부분에 등호가 이어진 식이 보입니다.
- 식의 표기는 다음과 같은 구조입니다:
  - **1 L 700 mL + 1 L 300 mL = 2 L + 1000 mL = 3 L**
- 각 단계는 같은 합을 다른 단위 표현으로 바꾸어 적은 형태로 보입니다.
- 이 문제 자체의 본문에는 식이 직접 제시되기보다, 아래 해설에서 수식 형태가 나타납니다.

[등호별 변환 의미] (해당될 때)
- 첫 번째 식:
  - **1 L 700 mL + 1 L 300 mL**
  - 두 물병의 용량을 더한 형태로 보입니다.
- 두 번째 식:
  - **= 2 L + 1000 mL**
  - 리터끼리, 밀리리터끼리 묶어 쓴 형태로 보입니다.
- 세 번째 식:
  - **= 3 L**
  - 앞 단계의 합을 최종적으로 3 L로 정리한 형태로 보입니다.
- 단, 이 문서는 **풀이 자체를 확정하지 않고**, 화면에 보이는 해설의 변환 순서만 기록합니다.

[세부 내용]
- 문제 문장은 “**두 물병의 물을 모아 3 L가 되게 하려면**”이라는 조건을 제시하고 있습니다.
- 그림 속 물병은 모두 **물이 가득 찬 상태**로 표현되어 있습니다.
- 각 물병의 용량은 병 아래쪽에 **텍스트로 직접 표기**되어 있어, 그림 크기만으로 판단하지 않도록 구성되어 있습니다.
- 선택해야 하는 대상은 물병 이름이 아니라, **라벨 가/나/다에 해당하는 물병**입니다.
- 오른쪽 아래의 선택지 문구는 괄호로 둘러싸인 목록 형태이며, 각 항목은 **“가 물병”**처럼 라벨과 ‘물병’이 결합되어 있습니다.
- 정답 및 해설은 문제 본문과 분리되어 아래쪽에 작게 배치되어 있습니다.

[Python DSL 작성 힌트]
- 큰 외곽 박스는 **연보라색 얇은 선**의 직사각형으로 구현하면 좋습니다.
- 박스 내부에 물병 3개를 **좌-중-우** 정렬로 배치하고, 각 물병 상단 근처에 **작은 원형 라벨**을 넣습니다.
- 각 물병 아래에는 가운데 정렬로 용량 텍스트를 배치합니다.
- 문제 문장은 상단 좌측에서 시작하는 **가로 문장형 텍스트**로 두고, 줄바꿈 위치를 자연스럽게 맞춥니다.
- 선택지는 박스 바깥 오른쪽 아래에 **괄호 포함 텍스트**로 배치합니다.
- 해설 영역에는 아래쪽에 **정답 표시 문구**와 **수식 해설 문장**을 별도로 넣는 구성을 고려합니다.
- 수식 해설을 DSL로 옮길 경우, **“A + B = C = D”**처럼 등호가 연속된 구조를 그대로 유지할 수 있습니다.
- 그림은 실제 측정값이 아니라 **시각적 아이콘**이므로, 병의 형태 차이보다 **라벨과 용량 텍스트**가 핵심입니다.

[주의할 점]
- **정답을 추론해서 새로 적지 마세요.** 화면에 있는 정답 표기는 참고용으로만 기록하고, 풀이 과정은 생성하지 않습니다.
- **빈칸을 채우는 형태는 보이지 않습니다.** 따라서 답칸을 임의로 생성하지 마세요.
- 물병의 정확한 형태는 스타일 요소가 포함되어 있으므로, **모양을 지나치게 세부적으로 단정하지 말고** 배치와 라벨 중심으로 기술하는 것이 안전합니다.
- 해설의 등호식은 화면에 적힌 대로만 기록하고, **단위 변환의 중간 논리를 추가로 보강하지 마세요.**
- 본문은 “모두 선택하세요” 형태이므로, **복수 선택 문제처럼 보이지만 실제 선택 결과는 시각 자료에 종속**됩니다.

[불확실한 부분]
- 각 물병 그림의 세부 디자인(뚜껑 모양, 손잡이 곡선, 병 몸통의 정확한 굴곡)은 이미지 해상도에 따라 약간 다르게 인식될 수 있습니다.
- 문제 본문이 줄바꿈된 정확한 위치는 화면 폭에 따라 달라 보일 수 있습니다.
- “가 물병, 나 물병, 다 물병”의 선택지와 아래 정답/해설 문구는 **인쇄된 해설 영역**으로 보이며, 실제 학습용 문제판에서는 숨겨질 가능성이 있습니다.
- 연보라색 박스의 정확한 선 두께와 색상은 화면 표시 환경에 따라 다소 달라질 수 있습니다.
