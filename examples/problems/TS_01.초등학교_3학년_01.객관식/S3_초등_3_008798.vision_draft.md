- 전체적으로 “그림 + 문장 + 정답/해설”로 이루어진 초등 수학 단원형 보기 문제 형태입니다.

- 아래쪽 `(정답) L`은 정답을 직접 제시하는 영역으로 보이며, 같은 줄에서 `L`이 강조된 형태입니다.
- `(해설)` 뒤 문장은 본문을 답에 맞게 다시 쓴 형태로 보입니다.
