[문제 개요]
- 화면은 **초등 수학 도형 판별형 1문항**이다.
- 문항 번호는 `14.`로 보인다.
- 지시문은 `직각이면 ○, 직각이 아니면 ×를 선택하세요.`이다.
- 가운데에는 **각의 그림 1개**가 있고, 아래쪽에는 **선택 기호 `○`, `×`**, 하단에는 **`(정답) ×`**, **`(해설)` 문장**이 보인다.
- 이 문항은 **그림을 보고 직각 여부를 판단하는 선택형 문제**로 보인다.

[원본에 보이는 요소]
- 상단 왼쪽에 `14.`가 보인다.
- 그 오른쪽에 지시문 `직각이면 ○, 직각이 아니면 ×를 선택하세요.`가 한 줄로 배치되어 있다.
- 문제 중앙에는 한 꼭짓점에서 두 선분이 만나는 각의 그림이 있다.
- 아래쪽 왼편에 파란색 `○` 표시가 1개 보인다.
- 도형 아래 쪽 중앙 부근에 파란색 `×` 표시가 1개 보인다.
- 그 아래에 `(정답) ×`가 인쇄되어 있다.
- 더 아래에 `(해설) 직각 삼각자의 직각 부분을 대어 보았을 때 꼭 맞게 겹쳐지는 각이 직각입니다.`가 보인다.
- 수식, 등호 연쇄, 분수식은 보이지 않는다.

[수식/관계 구조]
- 해당 없음.
- 이 문항은 **수식 변형 문제**가 아니라 **도형 판별 문제**이다.
- 보이는 관계는 다음과 같이 정리할 수 있다.
  - 지시문: `직각이면 ○, 직각이 아니면 ×`
  - 그림: 각 1개
  - 선택 표시: `○`, `×`
  - 하단 표기: `(정답) ×`
  - 해설: 직각삼각자의 직각 부분을 대어 확인하는 방법 설명

[등호별 변환 의미]
- 해당 없음.
- `=`가 있는 등호 연쇄나 변형 과정은 보이지 않는다.

[빈칸 정책]
- 빈칸은 보이지 않는다.
- 따라서 **추론으로 빈칸 값을 채우지 않는다.**
- 원본에 인쇄된 기호나 문구만 관찰 사실로 옮긴다.
- 정답 표기가 보이더라도, 이는 **관찰 사실로만 기록**하고 새로 계산하거나 추론한 값으로 바꾸지 않는다.

[반복 패턴]
- 선택형 문제에서 흔히 보이는 구성이다.
  - 상단에 문항 번호와 지시문
  - 중앙에 도형
  - 하단에 선택 기호
  - 별도의 정답 표기
  - 해설 문장
- 도형은 **한 꼭짓점에서 두 선분이 만나는 각**의 형태로 반복적으로 사용된다.
- 한 선분은 **오른쪽으로 거의 수평**으로 뻗고,
- 다른 선분은 **오른쪽 위 방향으로 기울어져** 뻗는다.
- 선택 기호 `○`, `×`는 각각 화면상 별도 배치되어 있으며, 문제의 응답 형식을 안내하는 요소로 보인다.

[DSL 작성 힌트]
- 요소를 다음처럼 분리해 기록하면 좋다.
  1. **문항 번호/지시문 텍스트**
  2. **각 그림**
  3. **선택 기호 `○`, `×`**
  4. **정답 표기**
  5. **해설 텍스트**
- 렌더링 관점 힌트:
  - 문항 번호는 좌상단 정렬
  - 지시문은 번호 오른쪽에 이어지는 한 줄 텍스트
  - 각 그림은 화면 중앙보다 약간 오른쪽에 위치
  - `○`는 왼쪽 아래 쪽에 작게 배치
  - `×`는 도형 아래 중앙 부근에 작게 배치
  - `(정답) ×`는 좌하단 쪽 텍스트
  - `(해설)`은 그 아래에 문장형 텍스트로 이어짐
- 도형 구성 힌트:
  - 꼭짓점 1개
  - 꼭짓점에서 뻗는 선분 2개
  - 아래쪽 선분은 오른쪽으로 거의 수평
  - 다른 선분은 오른쪽 위로 사선
- 텍스트와 도형 사이 간격이 비교적 넓게 보이므로, **세로 배치 간격**을 분리해서 저장하는 것이 좋다.

[semantic/solvable 힌트]
- 보이는 해설은 직각의 판정 기준을 설명한다.
- 해설 문장:
  - `직각 삼각자의 직각 부분을 대어 보았을 때 꼭 맞게 겹쳐지는 각이 직각입니다.`
- 의미상, 이 문제는 **직각인지 아닌지 판별하는 학습용 문항**이다.
- 다만, 답을 새로 추론해 쓰는 것은 금지한다.
- 원본에는 `(정답) ×`가 **인쇄되어 보이므로**, 이것은 관찰 사실로는 기록 가능하다.
