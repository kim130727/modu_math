# [전체적인 구성]
- 한 장짜리 초등 수학/논리형 문제 화면이다.
- 상단에는 문제 번호와 문제 지문이 왼쪽 정렬로 배치되어 있다.
- 지문 아래쪽 중앙에는 두 개의 물그릇 그림이 나란히 배치되어 있으며, 각각 `가`, `나`로 표시되어 있다.
- 화면 오른쪽 중간에는 선택지 형태로 보이는 `(`  `가 , 나`  `)` 뒤에 `그릇`이라는 텍스트가 있다.
- 왼쪽 중간에는 `(정답) 가 그릇`이 보이고, 그 아래에 `(해설)`이 표시된다.
- 하단에는 다시 `가`, `나` 그림이 나란히 나타나며, 맨 아래에 해설 문장이 이어진다.

# [보이는 텍스트]
- 문제 번호: `48.`
- 지문:
  - `가 그릇과 나 그릇에 물을 가득 채운 후 모양과 크기가 같은 그릇에 옮겨 담았더니 그림과 같이 물이 채워졌습니다. 둘이 더 많은 그릇을 선택해 보세요.`
- 그림 위의 표시:
  - `가`
  - `나`
- 오른쪽의 보기처럼 보이는 텍스트:
  - `( 가 , 나 ) 그릇`
- 정답 표시:
  - `(정답) 가 그릇`
- 해설 표시:
  - `(해설)`
- 하단 해설 문장:
  - `가 그릇의 들이가 더 많습니다.`
- 화면 좌상단에는 작은 네모 체크박스처럼 보이는 빈 사각형이 있다.

# [반복되는 시각 패턴]
- 물그릇 그림이 2개 반복된다.
  - 위쪽 중앙: 비교용 그림 2개
  - 아래쪽 왼쪽: 해설용 그림 2개
- 각 물그릇은 같은 외곽선 스타일을 사용한다.
  - 타원형 입구
  - 둥근 몸통
  - 내부 물 표면은 연한 파란색 그라데이션으로 표현
- 각 그림 옆에 원 안의 한글 표기 `가`, `나`가 반복된다.
- `가`와 `나`는 항상 그림의 왼쪽 위쪽에 작은 원형 표식으로 붙어 있다.
- 선택지/정답 영역에서도 `가, 나`가 다시 텍스트로 반복된다.

# [수식 단계]
- 해당 사항 없음.
- 등호(=)나 계산식 변형은 보이지 않는다.

# [등호별 변환 의미]
- 해당 사항 없음.

# [세부 내용]
- 위쪽의 두 그릇은 같은 종류의 용기처럼 보이지만, 물 높이가 다르게 표현되어 있다.
- `가` 그릇은 물이 더 높게 차 있고, `나` 그릇은 상대적으로 더 낮게 차 있다.
- 문제 지문은 “가 그릇과 나 그릇에 물을 가득 채운 후”라는 조건을 제시한 뒤, “모양과 크기가 같은 그릇에 옮겨 담았더니”라고 설명한다.
- 그림은 두 그릇의 물이 같은 기준 그릇에 옮겨졌을 때의 채워진 정도를 시각적으로 비교하는 상황으로 보인다.
- 오른쪽의 `( 가 , 나 ) 그릇`은 둘 중 하나를 고르는 형식의 답안 영역처럼 보인다.
- 아래 해설에는 다시 두 그릇 그림이 제시되고, 마지막 문장으로 `가 그릇의 들이가 더 많습니다.`라고 적혀 있다.
- 단, 이 문장은 화면에 적힌 해설 텍스트를 옮긴 것이며, 그 이유를 수학적으로 증명하거나 풀이한 것은 아니다.

# [Python DSL 작성 힌트]
- 상단 문제문은 한 줄 또는 두 줄로 나눠 배치하면 좋다.
- 왼쪽 상단에 `48.`과 작은 빈 사각형을 함께 넣는다.
- 본문 그림은 중앙 상단에 `가`, `나` 두 개의 물그릇 아이콘을 수평으로 정렬한다.
- 각 물그릇은 다음 요소로 구성하면 된다.
  - 바깥 윤곽선
  - 입구의 타원
  - 내부 물높이 영역
  - 원형 라벨 `가` 또는 `나`
- 오른쪽 중간의 보기 텍스트는 `(`, `가`, `,`, `나`, `)`와 `그릇`을 별도 요소로 배치할 수 있다.
- 왼쪽 중간의 `(정답) 가 그릇`은 정답 표시용 텍스트로 따로 배치한다.
- 하단 해설용 그림은 상단 그림과 같은 스타일로 재사용하고, 그 아래에 해설 문장을 넣는다.
- 해설 문장과 그림은 충분한 간격을 두어 겹치지 않게 배치한다.

# [주의할 점]
- 답을 추론해서 채우지 말아야 한다. 이미지에 이미 적힌 정답 표시만 사실로 기록한다.
- `가`, `나`의 상대적 크기나 물의 많고 적음을 설명할 때는 화면에 보이는 표현만 기준으로 써야 한다.
- `들이`는 용량을 뜻하는 문맥이지만, 풀이 과정이나 계산식은 만들지 않는다.
- 오른쪽의 `( 가 , 나 ) 그릇`은 선택지인지, 빈칸인지가 이미지상 완전히 명확하지 않을 수 있으므로 형태만 기술한다.
- 물그릇 내부의 정확한 물 높이는 픽셀 단위로 매우 정밀하게 단정하지 말고, 대략적인 시각 특징만 설명한다.

# [불확실한 부분]
- 오른쪽의 `( 가 , 나 ) 그릇`이 실제로 선택지인지, 정답 기입란인지 화면만으로는 완전히 단정하기 어렵다.
- `그릇` 앞의 괄호 안 구성 요소가 정확히 어떤 입력 형식을 의미하는지 확실하지 않다.
- 아래 해설의 문장 `가 그릇의 들이가 더 많습니다.`는 화면에 보이는 그대로 옮긴 것이며, 문장 내 표현이 교재 의도와 정확히 일치하는지는 추가 맥락이 없으면 단정할 수 없다.
