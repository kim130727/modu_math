- 하단의 `(정답)2`와 `(해설)`은 본문과 분리된 **작은 주석형 텍스트**처럼 배치되어 있습니다.

# [세부 내용]
- 문제 지문은 **“알맞은 것을 선택하세요.”**라고 되어 있어, 선택형 문제임을 알 수 있습니다.
- 표의 제목은 **목장별 우유 생산량**이며, 목장 이름이 `가, 나, 다, 라`로 구분되어 있습니다.
- 각 목장의 생산량은 각각 숫자로 제시되어 있고, 마지막 `합계` 열에는 총량으로 보이는 값 `135`가 들어 있습니다.
- 표 아래의 문장은 **“우유 생산량이 두 자리 수이므로 그림을 ( 1, 2 )가지로 나타냅니다.”**라고 적혀 있으며, 괄호 안은 선택지처럼 보이지만 실제로는 인쇄된 문장 안의 빈칸형 표기입니다.
- 하단의 `(정답)2`, `(해설)`은 학습지/해설지에서 보이는 보조 표기입니다.
- 해설 문장은 본문 문장과 같은 내용을 더 직접적으로 반복하고 있습니다.

# [Python DSL 작성 힌트]
- 표는 **2행 6열 구조**로 잡는 것이 적절합니다.
- 첫 행: `목장 / 가 / 나 / 다 / 라 / 합계`
- 둘째 행: `생산량(kg) / 36 / 31 / 25 / 43 / 135`
- 제목 `목장별 우유 생산량`은 표 상단 가운데에 배치합니다.
- 표의 셀 배경과 테두리는 **연한 주황색 테마**로 표현하면 원본 느낌에 가깝습니다.
- 지문 문장은 표 위쪽에 좌측 정렬로 넣고, 표 아래의 설명 문장은 가운데 정렬에 가깝게 배치합니다.
- 하단의 `(정답)2`, `(해설)` 텍스트는 본문과 분리해서 아래쪽에 작은 글씨로 배치합니다.
- 표 안 숫자들은 **좌우 여백이 충분한 중앙 정렬 텍스트**로 두는 것이 좋습니다.
- 괄호 `( 1, 2 )`는 답을 채우지 말고, **보이는 그대로 빈칸/선택 표기**로 유지합니다.
