[문제 개요]
- 문제 번호 `54.`가 좌상단에 표시되어 있다.
- 문제 문장은 상단에 2줄 정도로 배치되어 있으며, 그림그래프를 보고 조건에 맞는 농장을 고르는 내용이다.
- 중앙에는 `농장별 토마토 생산량`이라는 제목의 그림그래프 표가 있다.
- 표 오른쪽에는 토마토 그림 크기와 수량을 대응시키는 범례가 있다.
- 표 아래에는 선택 대상 후보를 나열한 괄호 문구가 있고, 그 아래에 `(정답)`, `(해설)` 문장이 보인다.

[원본에 보이는 요소]
- 보이는 텍스트
  - `54.`
  - `진아네 마을의 농장에서 하루 동안 생신힌 토마토의 양을 조사하여 그림그래프로 나타내었습니다. 토마토 생산량이 다 붕장보다 적은 농장을 모두 선택해 보세요.`
  - 제목: `농장별 토마토 생산량`
  - 표 머리글: `농장`, `토마토 생산량`
  - 농장 이름: `가`, `나`, `다`, `라`, `마`
  - 범례: 큰 토마토 그림 옆 `10상자`, 작은 토마토 그림 옆 `1상자`
  - 괄호 문구: `( 가 농장, 나 농장, 다 농장, 라 농장, 마 농장 )`
  - 하단 문구:
    - `(정답)나 농장, 마 농장`
    - `(해설)큰 그림 수가 2개보다 적은 농장을 찾고, 큰 그림 수가 2개인 경우 작은 그림의 수가 3개보다 적은 농장을 찾으면 나 농장과 마 농장입니다.`
- 시각 요소
  - 연한 초록색 계열의 표 테두리와 배경이 보인다.
  - 표는 왼쪽 좁은 열과 오른쪽 넓은 열로 나뉜다.
  - 행은 `가, 나, 다, 라, 마` 순으로 세로 정렬되어 있다.
  - 각 행 오른쪽 칸에는 토마토 그림이 가로로 반복 배치되어 있다.
  - 토마토 그림은 큰 것과 작은 것 두 종류이다.
  - 표 오른쪽 바깥에 범례가 세로로 배치되어 있다.
  - 표 아래 괄호 문구는 가운데쯤, 정답/해설은 더 아래에 배치되어 있다.

[수식/관계 구조]
- 이 문제는 등호로 이어지는 수식 전개가 아니라, 그림그래프의 범례를 통해 수량을 읽는 구조이다.
- 보이는 관계는 다음과 같다.
  - 큰 토마토 1개 = `10상자`
  - 작은 토마토 1개 = `1상자`
- 각 농장 행은 큰 토마토와 작은 토마토의 반복 조합으로 생산량을 나타낸다.
- 표는 “농장별 생산량 비교” 구조이며, 특정 기준보다 적은 농장을 찾는 선택형 문제이다.
- 보이는 정답 문구가 있으나, 문제를 새로 계산해서 답을 추론하지는 않는다.

[등호별 변환 의미]
- 해당 없음.
- 원본에는 등호 연쇄 변형이나 식 변형 과정이 보이지 않는다.

[빈칸 정책]
- 이 원본에는 직접적인 빈칸 입력란은 보이지 않는다.
- 괄호로 묶인 후보 목록은 보이지만, 추론으로 항목을 채워 넣지 않는다.
- 정답 칸이 따로 있다면, 원문에 인쇄되어 보이는 경우에만 사실로 기록하고 임의 보완은 하지 않는다.

[반복 패턴]
- 각 농장 행마다 토마토 그림이 여러 개 반복된다.
- 반복 단위는 두 종류의 아이콘이다.
  - 큰 토마토
  - 작은 토마토
- 큰 토마토와 작은 토마토의 크기 차이가 수량 단위를 구분하는 반복 패턴으로 쓰인다.
- 행별로 아이콘 개수와 조합이 달라 보이며, 왼쪽의 농장명과 1:1로 대응한다.
- 표 전체는 `가 → 나 → 다 → 라 → 마` 순으로 같은 구조가 반복되는 5행 구성이다.

[DSL 작성 힌트]
- 표 전체를 하나의 큰 그룹으로 구성하고, 위쪽에 제목을 중앙 정렬로 배치하는 방식이 자연스럽다.
- 표는 2열×5행 구조로 보되, 상단에 머리글 행이 따로 있는 형태로 처리하면 좋다.
- 왼쪽 열은 좁게, 오른쪽 열은 넓게 두어 농장명과 그림 영역을 분리한다.
- 오른쪽 열에는 토마토 아이콘들을 가로 방향으로 나열하고, 행마다 아이콘 묶음을 따로 그룹화한다.
- 범례는 표 오른쪽 바깥에 세로로 배치하고, 큰 토마토 아이콘 옆에 `10상자`, 작은 토마토 아이콘 옆에 `1상자`를 붙인다.
- 표 테두리와 셀 경계선이 연한 초록색이므로, 선색과 배경색을 구분해 설정하면 좋다.
- 표 아래의 후보 목록은 별도 텍스트 박스로 둔다.
- `(정답)`, `(해설)`은 본문 그림과 분리된 보조 텍스트로 처리하는 것이 좋다.
- 토마토 아이콘은 크기 차이가 명확하게 드러나도록 두 가지 렌더링 요소로 분리하는 편이 좋다.

[semantic/solvable 힌트]
- 보이는 의미상, 범례를 통해 큰 토마토와 작은 토마토가 서로 다른 수량 단위를 나타낸다.
- 문제는 각 농장의 그림 조합을 비교하여 조건에 맞는 농장을 고르는 선택형이다.
- 해설 문구에는 “큰 그림 수가 2개보다 적은 경우”, “큰 그림 수가 2개인 경우 작은 그림 수가 3개보다 적은 경우”라는 비교 기준이 보인다.
- 다만 이 비교 기준을 바탕으로 새로 판정하거나 답을 재계산해서 적지는 않는다.
- 원문에 `(정답)나 농장, 마 농장`이 보이지만, DSL에서는 이를 정답 추론의 근거로 사용하지 않고 관찰 텍스트로만 취급한다.
