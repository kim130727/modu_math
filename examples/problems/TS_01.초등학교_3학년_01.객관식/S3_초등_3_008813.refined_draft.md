# [문제 개요]
- 문항 번호는 `46.`으로 보인다.
- 문제는 초등 수학의 **비교/들이(용량)** 관련 선택형 문항으로 보인다.
- 상단에 문제 문장이 1~2줄로 배치되어 있고, 가운데에는 그림 비교가 있다.
- 하단에는 보기와 정답/해설 표기가 따로 보인다.
- 원본 이미지 기준으로, 화면 하단에 `(정답)컵`이 **인쇄된 듯 보이는 텍스트로 실제로 보인다**.

# [원본에 보이는 요소]
## 보이는 사실
- `46.`
- 문제 문장:
  - `우유병에 물을 가득 채운 후 컵에 옮겨 담았더니 우유병에 물이 남았습니다. 우유병과 컵 중에서 들이가 더 적은 것을 선택해 보세요.`
- 중앙 그림:
  - 왼쪽에 병 그림
  - 가운데에 오른쪽 방향 화살표
  - 오른쪽에 컵 그림
- 그림 아래 캡션:
  - `우유병`
  - `컵`
- 보기처럼 보이는 텍스트:
  - `( 우유병, 컵 )`
- 하단 표기:
  - `(정답)컵`
- 해설 표기:
  - `(해설)컵을 가득 채우고도 우유병에 물이 남았으므로 컵의 들이가 더 적습니다.`

## 추론되는 수학적 의미
- 병에서 컵으로 옮겨 담는 상황을 통해 두 용기의 들이를 비교하는 문제로 해석된다.
- “들이가 더 적은 것”을 고르는 형식으로 보인다.
- 다만 이 문서는 **풀이를 새로 계산하지 않고**, 화면에 보이는 내용을 구조화하는 데 집중한다.

# [수식/관계 구조]
## 보이는 사실
- 등호 `=`가 포함된 수식은 보이지 않는다.
- 분수, 나눗셈식, 계산식, 변형식은 보이지 않는다.
- 비교 구조는 그림과 문장으로 표현되어 있다.

## 추론되는 수학적 의미
- 관계는 대략 다음의 언어적 비교 구조로 읽힌다.
  - `우유병에 물을 가득 채움`
  - `그 물을 컵에 옮겨 담음`
  - `옮겨 담은 뒤에도 우유병에 물이 남음`
  - `따라서 컵의 들이가 더 적음`
- 그러나 이 문서에서는 답을 추론한 서술이 아니라, **원문 해설에 적힌 의미**로만 다룬다.

# [등호별 변환 의미]
- 해당 없음.
- 원본에 등호 연쇄나 식 변형이 보이지 않는다.

# [빈칸 정책]
- 빈칸, 미확정 보기, 미표시 답안 칸은 **추론값으로 채우지 않는다**.
- 원본에 실제로 인쇄되어 보이는 경우에만 문자 그대로 기록한다.
- 이 문제에는 별도의 빈칸 입력란이 보이지 않는다.
- `(정답)컵`은 빈칸 채우기가 아니라 **보이는 정답 표기**로만 취급한다.

# [반복 패턴]
- `문항 번호 + 본문 문장 + 삽화 + 보기 + 정답 + 해설`의 순서로 구성된다.
- 중앙 삽화는 좌우 비교형 레이아웃이다.
  - 왼쪽: 용기 그림
  - 가운데: 오른쪽 화살표
  - 오른쪽: 다른 용기 그림
- 캡션은 각 그림의 바로 아래에 짧게 배치되어 있다.
- 보기 텍스트는 괄호로 묶인 2항목 나열 형태이다.
- 정답과 해설은 하단에 별도 텍스트로 분리되어 있다.

# [DSL 작성 힌트]
## 레이아웃 힌트
- 전체를 다음 영역으로 나누면 자연스럽다.
  1. 상단 문제 문장 영역
  2. 중앙 삽화 영역
  3. 우측 보기 영역
  4. 하단 정답/해설 영역
- 중앙 삽화는 좌우 정렬로 배치한다.
  - 왼쪽: 병 아이콘
  - 가운데: 오른쪽 화살표 아이콘
  - 오른쪽: 컵 아이콘
- 각 아이콘 아래에 캡션을 둔다.
  - 병 아래: `우유병`
  - 컵 아래: `컵`

## 렌더링/배치 힌트
- 문제 문장은 여러 줄 줄바꿈을 반영해 배치한다.
- `46.`은 본문 시작 앞에 붙는다.
- `( 우유병, 컵 )`은 괄호와 쉼표, 공백을 포함한 문자열로 보존하는 편이 안전하다.
- 정답과 해설은 본문과 시각적으로 분리된 텍스트 박스로 처리한다.
- 그림은 계산 요소가 아니라 **비교 설명용 삽화**로 취급한다.
- 좌우 크기 비교를 직접 계산하기보다, **형태적 관계**만 표현하는 것이 좋다.

## 구성 요소 힌트
- 병 그림은 내용물 색이 일부 들어간 삽화로 보인다.
- 컵 그림은 위가 열린 컵 형태로 보인다.
- 가운데 화살표는 왼쪽에서 오른쪽으로의 이동/옮김을 암시한다.
- 단, 화살표의 정확한 의미는 문맥상 해석일 뿐이므로 과도하게 단정하지 않는다.

# [semantic/solvable 힌트]
## 보이는 사실
- 문제는 “우유병과 컵 중에서 들이가 더 적은 것”을 고르라고 묻는다.
- 해설에는 `컵을 가득 채우고도 우유병에 물이 남았으므로 컵의 들이가 더 적습니다.`라고 적혀 있다.
- 화면 하단의 `(정답)컵`이 보인다.

## 추론되는 수학적 의미
- 해설 문장은 비교 결과를 설명하고 있으나, 이 문서는 그 결과를 새로 계산하지 않는다.
- 따라서 `컵`이 정답이라는 사실은 **보이는 텍스트**로만 기록한다.
- 문제 해결 과정을 재구성할 때는 “가득 채움”, “옮겨 담음”, “남음”의 관계를 서술형으로 정리하면 된다.

# [주의 및 TODO]
- 정답을 추론해서 채우지 말고, 보이는 값만 기록한다.
- 빈칸이 없는지 최종 확인하되, 비어 있는 부분이 있더라도 임의로 채우지 않는다.
- 원본 이미지에서 병 내부 색, 화살표 의미, 정답/해설 표기의 성격은 일부 해석 여지가 있다.
- 현재 문서에서는 다음 항목을 확정하지 않는다:
  - 병 안의 파란색이 실제 물 높이인지 여부
  - 화살표가 단순 방향 표시인지, 옮겨 담기 과정의 표시인지 여부
  - `(정답)컵`, `(해설)`이 인쇄물 본문인지 시스템 오버레이인지 여부
- 필요한 경우 DSL에서는 이미지 요소를 **비교 삽화**로만 배치하고, 수학식 요소는 추가하지 않는 방향이 안전하다.
