[문제 개요]
- 문제 번호는 **86.**이며, 지시문은 **“알맞은 단위를 골라 보세요.”**이다.
- 본문은 **100원짜리 동전의 무게**에 대한 문장형 문제이다.
- 문장 중간에 **단위 선택용 괄호**가 있고, 선택지는 **g, kg, t**로 보인다.
- 상단 중앙에는 **100원짜리 동전 사진**이 1개 배치되어 있다.
- 하단에는 **(정답) g**와 **(해설)** 문장이 인쇄되어 있다.
- 계산식, 도형, 등호 연쇄는 보이지 않는다.

[원본에 보이는 요소]
- 보이는 텍스트:
  - `86. 알맞은 단위를 골라 보세요.`
  - `100원짜리 동전의 무게는 약 5( g , kg , t )입니다.`
  - `(정답)g`
  - `(해설)100원짜리 동전의 무게는 5 kg보다 더 가벼우므로 약 5 g이 알맞습니다.`
- 그림 요소:
  - 100원짜리 동전 사진 1개
- 배치 요소:
  - 문제 번호와 지시문은 좌상단
  - 동전 사진은 상단 중앙 쪽
  - 본문 문장은 중앙 부근의 큰 글씨
  - 정답/해설은 하단 좌측에 작은 글씨로 배치

[수식/관계 구조]
- 이 문제는 **수식 전개 문제**가 아니라 **단위 선택 문제**이다.
- 문장 구조는 다음과 같이 읽힌다:
  - `100원짜리 동전의 무게는 약 5( g , kg , t )입니다.`
- 괄호 안의 후보 단위는 **나열형 선택지**이다.
- 수량 `5`와 단위 후보가 결합되는 형태이므로, DSL에서는 **숫자 + 단위 선택 슬롯** 구조로 다루면 적절하다.
- 해설 문장에 `5 kg보다 더 가벼우므로`가 보이지만, 이것은 **설명 텍스트**이며 별도의 계산식 변형은 아니다.

[등호별 변환 의미]
- 화면에 **등호(=)를 포함한 변환식**은 보이지 않는다.
- 따라서 등호별 단계 변환 설명은 적용되지 않는다.
- 해설은 단순 비교 설명으로 보이며, 식 변형 연쇄는 없다.

[빈칸 정책]
- 문장 중간의 괄호는 **빈칸/선택 영역**으로 취급한다.
- **빈칸에는 추론값을 채우지 않는다.**
- 원본에 정답이 인쇄되어 보이더라도, DSL용 본문에는 **빈칸은 빈칸 상태로 유지**하는 정책을 우선한다.
- 선택지 `g, kg, t`의 순서는 원본 그대로 유지하는 것이 안전하다.
- 하단의 `(정답)g`는 **보이는 사실**로만 기록하고, 빈칸 채우기용 값으로 자동 반영하지 않는다.

[반복 패턴]
- 상단: 문제 번호 + 지시문
- 중단: 참고 그림 1개
- 본문: 문장형 질문 1개
- 본문 내부: 괄호로 묶인 단위 후보 3개
- 하단: 정답 표시 + 해설 문장
- 이 문제는 **학습지에서 자주 보이는 “단위 알기” 패턴**이다.
- 선택지는 쉼표로 구분된 텍스트 목록처럼 보이며, 체크박스보다는 **괄호 내 후보 나열** 형태이다.

[DSL 작성 힌트]
- 구성 요소를 다음처럼 분리하는 것이 좋다:
  1. 문제 번호 `86`
  2. 지시문 `알맞은 단위를 골라 보세요.`
  3. 동전 사진 1개
  4. 본문 문장
  5. 선택지 괄호 `g, kg, t`
  6. 하단 정답/해설 텍스트
- 본문 문장은 **문장형 텍스트 + 중간 선택 슬롯**으로 표현한다.
- 괄호 부분은 **수식 슬롯이 아니라 단위 후보 슬롯**으로 처리하는 것이 적절하다.
- 사진은 본문 위쪽 중앙에 따로 배치된 **참고 이미지**로 처리한다.
- 하단의 `(정답)`과 `(해설)`은 문제 본문과 분리된 **보조 텍스트 블록**으로 두는 편이 좋다.
- 렌더링 시:
  - 문제 번호는 좌측 상단에 작게
  - 동전 사진은 중앙 상단에 단독 배치
  - 본문은 그 아래 가운데 정렬에 가깝게
  - 정답/해설은 하단 좌측에 작은 글씨로 배치
- 선택지 괄호는 한 줄에 이어서 보여 주고, 후보 사이 간격은 원본처럼 좁게 유지한다.

[semantic/solvable 힌트]
- 이 문제의 의미는 **무게 단위가 무엇인지 고르는 것**이다.
- 해설에는 `5 kg보다 더 가볍다`는 비교가 적혀 있지만, 이것은 **보이는 설명**일 뿐이다.
- 정답으로 보이는 글자는 하단의 `(정답)g`이다.
- 다만 DSL 작성 시에는 정답을 자동으로 확정해 넣지 말고, **보이는 사실만 기록**하는 방식이 안전하다.
- 문제 본문은 단위 선택형이므로, 계산이나 변형 없이도 문장 구조만 유지하면 된다.
