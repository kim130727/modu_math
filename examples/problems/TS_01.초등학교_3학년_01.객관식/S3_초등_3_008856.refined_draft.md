[문제 개요]
- 문제 번호 `2.`가 상단 왼쪽에 보입니다.
- 질문 문장은 상단에 2줄로 배치되어 있습니다.
- 내용은 `가위, 풀, 지우개`의 무게를 비교하여 `가장 무거운 것`을 묻는 서술형 문제입니다.
- 화면 중간에는 남자아이 그림과 비교 설명 말풍선이 함께 배치되어 있습니다.
- 하단에는 `(정답)` 표기와 `(해설)` 문장이 인쇄되어 있습니다.
- 원본 이미지에 정답이 **실제로 인쇄되어 보입니다**: `(정답)가위`

[원본에 보이는 요소]
- 상단 문제 문장:
  - `2. 은우가 가위, 풀, 지우개의 무게를 다음과 같이 비교하였습니다. 가위, 풀, 지우개 중 가장 무거운 것은 어느 것인가요?`
- 인물 그림:
  - 남자아이 캐릭터가 왼쪽에 있습니다.
  - 인물 아래 이름으로 보이는 텍스트 `은우`가 있습니다.
- 비교 설명 상자:
  - 연한 하늘색 배경의 큰 상자가 오른쪽에 있습니다.
  - 상자 안에는 4줄의 문장이 있습니다.
- 상자 오른쪽 아래의 괄호 텍스트:
  - `( 가위 , 풀 , 지우개 )`
- 하단 정답:
  - `(정답)가위`
- 하단 해설:
  - `(해설)손에 힘이 더 많이 들어가는 쪽이 더 무거우므로`
  - `가위와 풀 중 가위가 더 무겁고, 풀과 지우개 중 풀이 더 무겁습니다.`
  - `무거운 것부터 순서대로 쓰면 가위, 풀, 지우개이므로 가장 무거운 것은 가위입니다.`

[수식/관계 구조]
- 등호 `=`가 있는 수식은 보이지 않습니다.
- 대신 비교 관계가 문장 형태로 제시됩니다.
- 비교 1:
  - `가위와 풀`을 양손에 각각 든 상황
  - 더 힘이 많이 드는 쪽: `가위`
- 비교 2:
  - `풀과 지우개`를 양손에 각각 든 상황
  - 더 힘이 많이 드는 쪽: `풀`
- 해설은 위 두 비교를 연결하여 무게 순서를 정리합니다.
- 관계 해석:
  - `가위`는 `풀`보다 무겁다.
  - `풀`은 `지우개`보다 무겁다.
  - 따라서 순서는 `가위 > 풀 > 지우개`로 읽힙니다.
- 단, 위 순서는 해설 문장에 이미 드러난 관계를 정리한 것이며, 새로운 계산을 만든 것은 아닙니다.

[등호별 변환 의미]
- 해당 사항 없음.
- 등호 연쇄 변환이나 식 변형은 보이지 않습니다.
- 비교 문장을 수식으로 바꾸는 단계도 화면에는 직접 표시되어 있지 않습니다.

[빈칸 정책]
- 빈칸이 보이지 않습니다.
- 따라서 추론값을 채워 넣을 대상도 없습니다.
- 만약 DSL에서 빈칸 슬롯을 만들더라도, 원본에 보이지 않는 값을 미리 채우지 말고 빈칸 상태를 유지해야 합니다.

[반복 패턴]
- 비교 문장 구조가 2번 반복됩니다.
  - `가위와 풀을 양손에 각각 들면 ...`
  - `풀과 지우개를 양손에 각각 들면 ...`
- 각 반복은 같은 틀을 사용합니다.
  - 대상 2개 제시
  - 양손에 각각 든 상황 설명
  - 더 힘이 많이 드는 쪽을 알려 줌
- 해설에서도 같은 비교 관계를 다시 반복해 요약합니다.
- 시각적으로는
  - 왼쪽: 인물 그림
  - 오른쪽: 비교 설명 상자
  - 아래: 정답/해설
  의 반복적 정보 흐름을 가집니다.

[DSL 작성 힌트]
- 상단 문제 문장은 긴 한 줄 텍스트로 두되, 실제 렌더링에서는 2줄 줄바꿈이 필요할 수 있습니다.
- 문제 번호 `2.`는 문장과 같은 영역에 붙어 있습니다.
- 인물 그림은 왼쪽, 설명 상자는 오른쪽의 가로 배치로 구성하는 것이 자연스럽습니다.
- 설명 상자는 연한 하늘색 배경의 직사각형 박스로 표현하는 것이 적절합니다.
- 상자 내부 텍스트는 중앙 정렬에 가깝게 4줄로 배치합니다.
- 상자 오른쪽 아래의 `( 가위 , 풀 , 지우개 )`는 본문과 분리된 별도 텍스트 객체로 두는 것이 좋습니다.
- 하단의 `(정답)`은 작은 라벨처럼 보이며, 바로 뒤에 답 텍스트가 이어집니다.
- 하단 해설은 여러 줄 텍스트 블록으로 분리해 배치하는 것이 좋습니다.
- 전체 구성은 위에서 아래로 흐르는 학습지형 레이아웃입니다.
- 그림, 말풍선, 정답, 해설 사이의 세로 간격을 충분히 두면 원본과 유사합니다.

[semantic/solvable 힌트]
- 이 문제는 수치 계산 문제가 아니라, 비교 문장을 통해 상대적 무게 순서를 읽는 문제입니다.
- “손에 힘이 더 많이 들어가는 쪽이 더 무겁다”는 문장이 핵심 규칙입니다.
- 원본 해설에는 이미 순서가 문장으로 정리되어 있습니다.
- 원본에 인쇄된 정답 표기는 `(정답)가위`입니다.
- 다만 DSL 작성에서는 정답을 임의 생성하는 방식이 아니라, **보이는 텍스트를 그대로 옮기는 것**에 집중해야 합니다.
