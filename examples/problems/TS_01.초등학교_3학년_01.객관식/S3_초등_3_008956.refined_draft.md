[문제 개요]
- 문제 번호 `26.`이 보인다.
- 문장형 질문이 1개 있으며, 내용은 “은호네 모둠 학생들이 좋아하는 과일을 조사하여 나타낸 자료와 표 중에서 어느 것이 더 편리할까요?”로 읽힌다.
- 화면에는 위쪽의 그림 자료와 아래쪽의 요약 표가 함께 제시되어 있다.
- 아래쪽에는 `(정답)`, `(해설)` 표기가 따로 보이며, 본문과 분리된 안내/해설 영역처럼 배치되어 있다.

[원본에 보이는 요소]
- 보이는 텍스트
  - `26.`
  - `은호네 모둠 학생들이 좋아하는 과일을 조사하여 나타낸 자료와 표 중에서 어느 것이 더 편리할까요?`
  - 위쪽 제목: `친구들이 좋아하는 과일`
  - 아래쪽 표 제목: `친구들이 좋아하는 과일`
  - 범례: `사과, 포도, 딸기, 바나나`
  - 표 머리글: `과일, 사과, 포도, 딸기, 바나나, 합계`
  - 행 이름: `학생 수(명)`
  - 수치: `5, 3, 6, 1, 15`
  - 오른쪽 아래: `( 자료 , 표 )`
  - 하단: `(정답)표`
  - 하단 해설: `(해설)조사한 학생 수는 표의 합계를 보면 쉽게 알 수 있습니다.`
- 위쪽 그림 자료
  - 3행 × 5열의 이름-그림 대응 표처럼 보인다.
  - 이름: `은호, 민지, 정민, 준수, 호진 / 선아, 효성, 영준, 규진, 현영 / 승희, 선미, 은수, 재민, 은채`
  - 각 칸에는 학생 이름 1개와 과일 그림 1개가 함께 있다.
  - 과일 그림은 사과, 포도, 딸기, 바나나 아이콘으로 보인다.
- 아래쪽 표
  - 2행 표로 보이며, 첫 열은 항목명, 나머지 열은 과일별 수와 합계를 나타낸다.
  - `합계` 칸에 `15`가 인쇄되어 보인다.

[수식/관계 구조]
- 등호로 연결된 수식이나 계산식은 보이지 않는다.
- 보이는 관계는 다음과 같다.
  - 위쪽 그림 자료는 학생 15명의 과일 선호를 개별 그림으로 나타낸 것처럼 보인다.
  - 아래쪽 표는 같은 정보를 과일 종류별 개수로 정리한 요약 형태처럼 보인다.
  - 표의 열 순서는 `사과 → 포도 → 딸기 → 바나나 → 합계`이다.
  - 숫자 `5, 3, 6, 1, 15`는 각각 위 열에 대응하는 학생 수로 보인다.
- 추론되는 수학적 의미
  - 위쪽 그림의 각 과일 아이콘 개수가 아래 표의 해당 수치와 대응되는 것으로 보인다.
  - `합계 15`는 전체 학생 수를 나타내는 값처럼 읽힌다.

[반복 패턴]
- 위쪽 자료는 3행 × 5열 격자 패턴으로 반복된다.
  - 각 칸마다 이름 1개 + 과일 아이콘 1개가 대응된다.
- 과일 아이콘은 범례에 따라 반복 사용된다.
  - 사과 아이콘
  - 포도 아이콘
  - 딸기 아이콘
  - 바나나 아이콘
- 아래쪽 표는 2행 요약 표 패턴이다.
  - 첫 행: 열 제목
  - 둘째 행: 수치 입력 행
- 제목 `친구들이 좋아하는 과일`이 위쪽 자료와 아래쪽 표에 각각 한 번씩 반복된다.

[DSL 작성 힌트]
- 전체 배치는 세로 방향으로 위쪽 그림 자료, 아래쪽 표, 하단 안내문이 순서대로 놓인 구조로 잡으면 좋다.
- 위쪽 영역
  - 제목 박스 1개
  - 3행 × 5열 격자
  - 각 셀 내부의 이름 텍스트
  - 각 셀 내부의 과일 아이콘
  - 범례용 아이콘 + 텍스트 가로 배열
- 아래쪽 영역
  - 제목 박스 1개
  - 2행 표 1개
  - 첫 열의 항목명 셀
  - 과일별 수치 셀
  - `합계` 셀
- 렌더링/레이아웃 힌트
  - 위쪽 표는 둥근 테두리의 분홍색 박스처럼 보인다.
  - 내부는 점선 격자로 나뉘어 있다.
  - 아래쪽 표는 파란색 테두리와 연한 파란색 머리글 배경을 가진다.
  - 숫자는 셀 가운데 정렬로 배치하는 것이 자연스럽다.
  - 제목은 각각 별도의 배경 박스 안에 중앙 정렬되어 있다.
  - 범례는 과일 아이콘과 텍스트를 좌우로 한 줄에 정렬한다.
- 재사용 힌트

  - 아래쪽 표의 `합계`는 의미상 다른 열보다 강조 가능성이 있다.
