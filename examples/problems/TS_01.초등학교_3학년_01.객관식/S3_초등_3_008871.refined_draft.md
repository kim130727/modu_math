[문제 개요]
- 1페이지형 초등 수학 문제이다.
- 문제 번호 `24.`가 상단 왼쪽에 보인다.
- 문장형 질문 뒤에 3개의 그림 묶음이 가로로 배치되어 있고, 각 묶음은 `가`, `나`, `다` 중 하나에 대응된다.
- 하단에는 `(정답)`과 `(해설)`이 인쇄되어 있다.
- 원본 이미지에 정답이 **보이지만**, 아래에서는 관찰 사실만 기록하고 추론 계산은 하지 않는다.

[원본에 보이는 요소]
- 상단 문장:
  - `가 그릇, 나 그릇, 다 그릇에 물을 가득 채운 후 모양과 크기가 같은 그릇에 옮겨 담았습니다. 그림과 같이 물을 채웠을 때에 가 그릇, 나 그릇, 다 그릇 중 들이가 가장 많은 것은 어느 것인지 선택하세요.`
- 라벨:
  - 원 안의 `가`
  - 원 안의 `나`
  - 원 안의 `다`
- 가운데 그림 3묶음:
  - 각 묶음은 왼쪽의 빈 그릇, 가운데의 오른쪽 방향 화살표, 오른쪽의 물이 든 그릇으로 구성됨
- 아래 중앙 텍스트:
  - `( 가 그릇, 나 그릇, 다 그릇 )`
- 하단 정답:
  - `(정답) 다 그릇`
- 하단 해설:
  - `(해설) 그릇에 담긴 물의 높이를 비교하면 다 그릇이 가장 높으므로 다 그릇의 들이가 가장 많습니다.`

[수식/관계 구조]
- 이 문제는 등식 계산식이 아니라, 그림 비교를 통해 `들이`를 비교하는 구조이다.
- 세 대상 `가`, `나`, `다`가 각각 하나의 그림 묶음과 대응된다.
- 각 묶음의 관계는 다음처럼 읽힌다.
  - `가 그릇` → 물을 가득 채움 → 모양과 크기가 같은 그릇에 옮겨 담은 결과를 그림으로 표시
  - `나 그릇` → 동일한 방식
  - `다 그릇` → 동일한 방식
- 수학적 의미:
  - 물의 높이 비교를 통해 들이의 상대적 크기를 판단하는 문제로 보인다.
- 단, 실제 답을 새로 계산해서 적지는 않는다.

[등호별 변환 의미]
- 등호 기호나 수식 변형 연쇄는 보이지 않는다.
- 따라서 단계별 등호 변환 설명은 해당 없음이다.
- 그림에서의 변화는 등호가 아니라 “그릇에 물을 채운 뒤 다른 같은 그릇으로 옮겨 담는 과정”으로 표현된다.

[빈칸 정책]
- 빈칸을 채우는 방식의 추론은 하지 않는다.
- 원문에 빈칸이 따로 보이지 않으면, 빈칸 후보를 임의로 만들지 않는다.
- 원문에 정답이 인쇄되어 보여도, DSL용 초안에서는 정답을 빈칸에 넣지 않고 **보이는 텍스트 그대로의 위치 정보**만 기록한다.
- 따라서 이 문제는 `정답: 다 그릇`이 보이더라도, 빈칸을 채운 것으로 처리하지 않는다.

[반복 패턴]
- 동일한 구조의 그림 묶음이 3회 반복된다.
- 각 반복은 다음 구성으로 보인다.
  - 원형 라벨 1개
  - 왼쪽 빈 그릇 1개
  - 오른쪽 방향 화살표 1개
  - 오른쪽 물이 찬 그릇 1개
- 반복 라벨은 순서대로 `가`, `나`, `다`이다.
- 세 묶음은 좌→우로 배치되어 비교 대상으로 읽힌다.
- 그릇 색/형태가 서로 다르게 표현되어 식별을 돕는다.
  - `가`: 갈색 계열의 작은 컵형
  - `나`: 연두색 계열의 낮은 그릇형
  - `다`: 파란색 계열의 넓은 그릇형
- 화살표는 모두 오른쪽 방향이며, 옮겨 담기 흐름을 시각적으로 나타낸다.

[DSL 작성 힌트]
- 상단의 긴 문제 문장은 좌측 정렬 텍스트 블록으로 둔다.
- 가운데는 가로 3칸 구조로 배치한다.
  - 각 칸은 서로 같은 폭의 독립 영역으로 잡는다.
  - 각 칸의 위쪽에 원형 라벨을 둔다.
  - 그 아래에 왼쪽 그릇, 가운데 화살표, 오른쪽 물이 찬 그릇을 순서대로 놓는다.
- 각 묶음 사이 간격은 충분히 둬서 3개가 분리되어 보이게 한다.
- 그릇은 각각 색과 윤곽이 다르게 표현되도록 한다.
- 물은 반투명한 파란색 채움으로 보이게 하고, 그릇 테두리와 물 표면을 분리해 그린다.
- 화살표는 회색 계열의 굵은 오른쪽 화살표로 표현하면 원문 느낌에 가깝다.
- 그림 아래 중앙의 `( 가 그릇, 나 그릇, 다 그릇 )`는 별도 텍스트 줄로 둔다.
- 하단의 `(정답)`과 `(해설)`은 작은 텍스트 블록으로 배치한다.
- 정답과 해설은 원문에 보이는 위치를 유지하되, 새로운 해석을 덧붙이지 않는다.

[semantic/solvable 힌트]
- 이 문제의 핵심 의미는 “물의 높이 비교”와 “들이 비교”이다.
- 그림에서 물 높이가 더 높아 보이는 대상이 들이가 더 많다는 해설이 인쇄되어 있다.
- 그러나 이 초안에서는 그 결론을 새로 도출하거나 계산하지 않는다.
- 보이는 해설 문장은 사실로 기록만 한다.
- 원문에서 정답이 이미 인쇄되어 보이므로, 시각적 사실로는 `다 그릇`이 표시되어 있음을 적을 수 있다.

[주의 및 TODO]
- 정답을 빈칸에 추론해서 넣지 말 것.
- 그림의 실제 비율은 인쇄 상태에 따라 달라질 수 있으므로, 절대 치수처럼 쓰지 말 것.
- `가`, `나`, `다`의 높이 비교는 “그림상 그렇게 보인다” 수준으로만 서술할 것.
- 각 그릇의 실제 실물 명칭(컵, 대야 등)은 원문에 명시되지 않았으므로 확정하지 말 것.
- 오른쪽 물 표면의 곡선은 해상도상 다소 흐리므로, 정확한 형태는 TODO로 남긴다.
- 등호 연쇄나 분수 변환식이 없으므로 관련 DSL 규칙은 적용되지 않는다.
