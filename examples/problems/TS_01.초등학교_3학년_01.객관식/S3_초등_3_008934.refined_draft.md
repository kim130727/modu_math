[문제 개요]
- 화면 상단에 문제 번호 `2.`와 지시문이 있으며, 아래에 **숫자 표 1개**, **그림그래프 표 1개**, **범례**, **말풍선형 설명 상자**, **오른쪽 아래 인물 그림**, **하단 정답/해설 텍스트**가 배치되어 있다.
- 문제의 핵심은 **그림그래프에서 책 그림의 의미(10권, 1권)가 바뀐 잘못**을 찾는 것이다.
- 원본 이미지 기준으로 정답 표시는 `(정답)바꿔서`로 인쇄되어 보인다.

[원본에 보이는 요소]
- 상단 지시문:
  - `2. 표를 보고 그림그래프로 나타낸 것입니다. 잘못된 부분을 찾아 선택해 보세`
  - 다음 줄에 `요.`가 이어져 보임
- 첫 번째 표 제목:
  - `종류별 책의 수`
- 첫 번째 표의 항목:
  - 머리글: `종류 / 동화책 / 위인전 / 과학책 / 합계`
  - 수치 행: `책의 수(권) / 40 / 25 / 14 / 79`
- 두 번째 표 제목:
  - `종류별 책의 수`
- 두 번째 표의 항목:
  - 머리글: `종류 / 책의 수`
  - 행 이름: `동화책 / 위인전 / 과학책`
- 범례:
  - 책 그림 옆에 `10권`
  - 작은 책 그림 옆에 `1권`
- 말풍선/설명 상자:
  - `10권과 1권을 나타내는 그림을 ( 둘러서 , 바꿔서 ) 그렸어.`
- 하단 정답 표시:
  - `(정답)바꿔서`
- 하단 해설 문장:
  - `(해설) [큰 책 그림]은 10권, [작은 책 그림]은 1권을 나타내는데 10권과 1권의 그림이 바뀌었습니다.`

[수식/관계 구조]
- 이 문제는 등호로 이어지는 수식 변형이 아니라, **표의 수치와 그림의 대응 관계**를 비교하는 구조이다.
- 위쪽 숫자 표는 실제 책의 수를 직접 제시한다.
- 아래쪽 그림표는 책 그림의 개수와 크기를 이용해 수를 표현한다.
- 범례가 `큰 책 그림 = 10권`, `작은 책 그림 = 1권`의 의미를 갖도록 보이지만, 말풍선과 해설에서 **두 그림의 역할이 서로 바뀌었다**고 지적한다.

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 식의 단계적 변형은 보이지 않는다.

[빈칸 정책]
- 빈칸을 추론값으로 채우지 않는다.
- 문제의 선택형/괄호형 표현이 있더라도, 원본에 인쇄된 텍스트만 그대로 옮기고 부족한 부분은 비워 둔다.
- 이 문항에서는 숫자나 개수를 새로 계산해 빈칸을 메우지 않는다.
- `( 둘러서 , 바꿔서 )`는 원본에 보이는 선택지 형태이므로, 추론으로 다른 말을 추가하지 않는다.

[반복 패턴]
- 책 그림 아이콘이 여러 번 반복되어 각 종류의 책 수를 나타낸다.
- 아래 표의 각 행은 `종류명 + 가로로 나열된 책 그림`의 반복 패턴으로 구성된다.
- 범례에서도 책 그림이 2개 반복되며, 각각 다른 크기로 제시된다.
- 상단 숫자 표와 하단 그림표가 같은 제목 `종류별 책의 수`를 공유해 대응 구조를 만든다.

[DSL 작성 힌트]
- 상단 지시문은 좌상단에 텍스트 블록으로 배치한다. 줄바꿈은 원본처럼 나뉘어 있어도 되지만, 읽기 쉽게 한 줄로 정리해도 된다.
- 첫 번째 표는 2행 5열 구조로 처리한다.
  - 제목은 표 위 중앙에 둔다.
  - 첫 행: `종류 / 동화책 / 위인전 / 과학책 / 합계`
  - 둘째 행: `책의 수(권) / 40 / 25 / 14 / 79`
- 두 번째 표는 4행 2열 구조에 가깝게 보인다.
  - 제목은 표 위 중앙에 둔다.
  - 왼쪽 열은 `종류`, `동화책`, `위인전`, `과학책`
  - 오른쪽 열은 각 행마다 책 그림이 가로로 반복 배치된다.
- 책 그림은 같은 모양이 반복되지만, **큰 책 그림**과 **작은 책 그림**을 서로 다른 이미지 객체로 두는 것이 좋다.
- 범례는 표 아래쪽에 배치하고, 큰 책 그림 옆에 `10권`, 작은 책 그림 옆에 `1권`을 붙인다.
- 말풍선/설명 상자는 연한 분홍색 계열의 둥근 직사각형 느낌으로 배치하면 원본과 유사하다.
- 오른쪽 아래 인물 그림은 별도 삽화 요소로 둔다.
- 하단의 `(정답)바꿔서`, `(해설) ...`은 문제 외 설명 텍스트로 별도 배치한다.

[semantic/solvable 힌트]
- 위쪽 표는 기준 데이터이고, 아래쪽 표는 그 데이터를 그림으로 나타내려는 도식이다.
- 문제는 **어떤 점이 잘못되었는지 찾는 것**에 초점이 있다.
- 원본에 이미 `(정답)바꿔서`가 보이므로, 관찰 사실로서 그 선택지가 정답 표시임을 언급할 수 있다.
- 해설 문장도 보이므로, 관찰 사실로서 `큰 책 그림`과 `작은 책 그림`의 역할이 바뀌었다고 기록할 수 있다.
- 다만, 아래 표의 각 행에 실제로 몇 개의 아이콘이 들어 있는지까지 계산해 확정하지는 않는다.
