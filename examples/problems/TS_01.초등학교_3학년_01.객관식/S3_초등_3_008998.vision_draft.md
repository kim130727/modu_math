# [전체적인 구성]
- 상단 왼쪽에 문제 번호와 지시문이 있고, 그 아래 중앙 상단에 도형 2개가 나란히 배치되어 있습니다.
- 왼쪽 도형은 직사각형, 오른쪽 도형은 정사각형으로 보이며, 각각 아래에 이름표가 붙어 있습니다.
- 도형 아래에는 두 문장의 빈칸형 설명문이 크게 배치되어 있고, 그 아래 왼쪽에는 정답 표기, 하단에는 해설 문장이 있습니다.
- 전체적으로 “그림을 보고 알맞은 말을 선택”하는 형태의 객관식/선택형 문항 구조입니다.  
  - 다만 실제 선택지는 보이지 않고, 문장 중 괄호 안에 선택할 말이 들어가는 형식입니다.

# [보이는 텍스트]
- `□ 36. 그림을 보고 알맞은 말을 선택하세요.`
- 도형 아래 라벨:
  - `직사각형`
  - `정사각형`
- 본문 문장:
  - `직사각형은 마주 보는 ( 두, 네 ) 변의 길이가 같습니다.`
  - `정사각형은 ( 두, 네 ) 변의 길이가 모두 같습니다.`
- 아래쪽 표기:
  - `(정답) 두, 네`
  - `(해설) 직사각형은 마주 보는 두 변의 길이가 같고, 정사각형은 네 변의 길이가 모두 같습니다.`

# [반복되는 시각 패턴]
- 두 개의 도형이 비교용으로 나란히 배치되어 있습니다.
- 각 도형의 네 꼭짓점 쪽에 분홍색의 작은 직각 표시가 있어, 사각형임을 강조하는 시각 패턴이 보입니다.
- 두 도형 아래에는 각각 한 단어짜리 도형 이름 라벨이 있고, 같은 스타일로 정렬되어 있습니다.
- 본문 문장에서는 두 번 같은 괄호 패턴 `( 두, 네 )`가 반복됩니다.
- 아래 해설 문장에서도 `두 변`, `네 변`의 표현이 다시 반복되어 문항 내용을 되짚는 구조입니다.

# [세부 내용]
- 왼쪽에는 직사각형 그림이 있으며, 가로가 세로보다 길어 보입니다.
- 오른쪽에는 정사각형 그림이 있으며, 네 변이 같아 보이는 비율로 그려져 있습니다.
- 두 도형 모두 선은 검은색이고, 꼭짓점 안쪽에 분홍색 직각 표시가 있어 각이 직각임을 시각적으로 나타냅니다.
- 각 도형 바로 아래에 회색빛/연한 글씨로 도형 이름이 붙어 있어, 그림과 용어를 대응시키는 역할을 합니다.
- 본문은 두 개의 빈칸형 문장으로 구성되어 있으며, 괄호 안에 `두, 네` 중 하나를 선택하게 되어 있습니다.
- 아래쪽의 `(정답)`과 `(해설)`은 학습용 설명으로 보이며, 문제 본문과는 별도로 배치되어 있습니다.

# [Python DSL 작성 힌트]
- 상단 문제 번호와 지시문을 한 줄로 배치하는 텍스트 객체가 필요합니다.
- 중앙에는 도형 2개를 수평 정렬로 배치합니다.
  - 왼쪽: 직사각형 외곽선 + 네 꼭짓점의 직각 표시 + 아래 라벨 `직사각형`
  - 오른쪽: 정사각형 외곽선 + 네 꼭짓점의 직각 표시 + 아래 라벨 `정사각형`
- 본문 문장은 가운데 정렬에 가깝게 2줄로 배치합니다.
  - 각 문장에는 괄호 `(` `)`와 쉼표 `,`가 포함된 선택형 빈칸을 그대로 표시합니다.
- 하단 왼쪽에 `(정답) 두, 네`를 작은 글씨로 넣고, 그 아래 또는 옆에 `(해설)` 문장을 배치합니다.
- 도형과 문장 사이의 간격이 꽤 넓으므로 세로 여백을 충분히 두는 것이 좋습니다.
- 분홍색 직각 표시는 꼭짓점 근처의 작은 L자/ㄴ자 형태 마커로 처리하면 됩니다.

# [주의할 점]
- 빈칸에 들어갈 말을 추론해서 별도의 강조 텍스트로 확정 표기하지 마세요.
- 문제 본문에서는 선택지 괄호 안의 글자 배열을 그대로 유지해야 합니다.
- 도형의 정확한 가로세로 비율은 사진상 약간 다르게 보일 수 있으므로, “직사각형/정사각형”의 형태적 차이만 반영하면 됩니다.
- 해설 문장은 문제의 일부가 아니라 아래 설명 영역이므로, 본문과 구분해서 배치해야 합니다.
- 왼쪽 상단의 작은 네모 표시는 문제 번호 앞의 기호처럼 보이나, 인쇄물의 장식/표식일 수 있어 의미를 단정하지 않는 것이 좋습니다.

# [불확실한 부분]
- 문제 번호 앞의 작은 네모 기호(`□`)는 체크박스인지 장식인지 화면만으로 확실하지 않습니다.
- 도형 내부의 분홍색 표시가 정확히 직각 기호인지, 혹은 인쇄용 보조 표시인지 100% 단정하기는 어렵습니다.
- 본문 문장의 글꼴 굵기와 줄간격은 이미지 해상도에 따라 다르게 구현될 수 있습니다.
- `(정답) 두, 네`는 화면에 보이지만, DSL에서 이 영역을 반드시 포함해야 하는지 여부는 작성 목적에 따라 달라질 수 있습니다.
