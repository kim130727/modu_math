[문제 개요]
- 초등 수학 문제 1개.
- 주어진 두 털실의 길이를 수직선으로 비교하는 형식이다.
- 화면에는 문제 설명, 문항 안내, 수직선 2개, 비교 문장, 그리고 아래쪽의 일부 잘린 추가 표기가 함께 보인다.
- 이미지 기준으로는 문제 번호 `94.`가 보인다.

[원본에 보이는 요소]
- 상단 본문:
  - `재영이는 노란색 털실을 9/7 m 가지고 있고 보라색 털실을 1 3/7 m 가지고 있습니다.`
  - `더 긴 털실은 무슨 색인지 알아보려고 합니다. 물음에 답하세요.`
- 문항 안내:
  - `(1) 수직선을 보고, 더 큰 수를 선택해 보세요.`
- 수직선 2개:
  - 첫 번째 수직선 왼쪽 라벨: `9/7 m`
  - 두 번째 수직선 왼쪽 라벨: `1 3/7 m`
  - 각 수직선에 `0`, `1`, `2(m)` 눈금/라벨이 보임
  - `1` 근처 위쪽에 `7/7` 표기가 각각 보임
  - 파란색 선분이 각 수직선의 0에서 오른쪽으로 이어져 있음
- 하단 문장:
  - `(1) 수직선에 나타난 부분을 비교하면 9/7 < 1 3/7 입니다.`
- 맨 아래쪽:
  - `1 3/7`처럼 보이는 일부 잘린 표기가 추가로 보임
- 화면 왼쪽 위에는 작은 사각형 기호처럼 보이는 빈 표시가 함께 보임.

[수식/관계 구조]
- 직접적인 수식 변형 사슬은 보이지 않는다.
- 보이는 비교 관계는 다음 1개이다:
  - `9/7 < 1 3/7`
- 이 비교는 수직선에서 표시된 길이의 상대적 위치를 바탕으로 제시된 것으로 보인다.
- 두 수는 각각 털실의 길이를 나타내며, 단위 `m`가 붙어 있다.
- 두 수직선 모두 같은 기준 구간 `0 ~ 2(m)` 위에 놓여 있어 비교 구조가 명확하다.

[등호별 변환 의미]
- 등호 연쇄나 식 변형은 보이지 않는다.
- 보이는 것은 부등호 비교식 하나뿐이다.
- 각 단계의 변환이 있는 형태가 아니므로, 별도 변환 의미를 적을 수 없다.
- 단, 화면 하단의 문장 `수직선에 나타난 부분을 비교하면 ...`은 수직선 상 위치 비교를 문자식으로 옮긴 결과로 해석된다.

[빈칸 정책]
- 이 화면에는 풀이용 빈칸이 명확하게 보이지 않는다.
- 아래쪽에 `1 3/7`처럼 일부 잘린 표기가 있지만, 이것을 빈칸으로 단정하지 않는다.
- 보이지 않거나 잘린 부분은 추론으로 채우지 않고, 그대로 미확정 상태로 둔다.
- 만약 DSL에서 빈칸 슬롯이 필요해도, 원본에 보이는 범위만 반영하고 추정값은 넣지 않는다.

[반복 패턴]
- 동일한 형태의 수직선이 2개 반복된다.
- 반복되는 공통 요소:
  - 가로 수직선 축
  - `0`, `1`, `2(m)` 라벨
  - `1` 위쪽의 `7/7` 라벨
  - 0에서 오른쪽으로 뻗는 파란 강조 선분
  - 왼쪽의 길이 표기 텍스트
- 두 수직선의 눈금 간격과 구조는 거의 동일해 보인다.
- `7/7` 표기가 두 개 모두 같은 높이에 반복되어 있어, `1 = 7/7` 기준을 강조하는 레이아웃으로 보인다.
- 각 수직선의 파란 선분 끝점은 서로 다르며, 두 길이의 차이를 시각적으로 비교하게 한다.

[DSL 작성 힌트]
- 상단 본문은 텍스트 박스 2개로 분리하는 편이 좋다.
- 문항 번호 `(1)`은 본문과 분리된 작은 라벨 요소로 두면 좋다.
- 수직선 2개는 별도 컴포넌트로 두되, 아래 요소를 공통 템플릿처럼 맞추면 좋다:
  - 기준선 1개
  - 0, 1, 2(m) 라벨
  - 0~2 사이 등간격 눈금들
  - `1` 위의 `7/7` 라벨
  - 왼쪽에 분수/대분수 길이 표기
  - 0부터 특정 지점까지의 파란 선분
- 두 수직선은 세로로 배치하고 좌측 라벨과 축의 시작점을 같은 x축 기준에 맞추는 것이 좋다.
- `2(m)`는 숫자와 단위를 함께 보이게 처리해야 한다.
- 하단 비교 문장은 별도 텍스트 박스로 배치하고, 비교식 `9/7 < 1 3/7`은 문장 안의 인라인 수식처럼 넣는 구성이 적합하다.
- 화면 하단의 잘린 추가 표기는 별도 주석 또는 TODO 영역으로 분리하는 것이 안전하다.
- 정렬 힌트:
  - 수직선 두 개는 수평 길이와 눈금 위치를 동일 스케일로 유지
  - `7/7`은 두 수직선 모두 `1` 위 중앙 정렬
  - 왼쪽 길이 표기는 각 수직선의 중앙 높이에 맞춤

[semantic/solvable 힌트]
- 문제 의도는 두 털실의 길이를 수직선으로 비교하여 더 큰 수를 찾는 것이다.
- 원본에 실제 정답 선택 결과나 색 이름이 인쇄되어 보이지는 않는다.
- 비교식은 원본에 보이는 사실이므로 기록 가능하지만, 그로부터 답을 새로 계산해 확정하지는 않는다.
- 수직선에서 `0~2(m)` 범위 안의 위치 차이를 통해 두 값의 크기 비교가 이루어지는 구조다.
- `9/7`과 `1 3/7`은 분수/대분수 표현으로 함께 제시되어 있다.

[주의 및 TODO]
- 정답 추론 금지: 더 긴 털실의 색 이름을 단정하지 말 것.
- 하단의 잘린 `1 3/7` 표기는 문장 전체인지, 별도 예시인지, 혹은 캡처 잔상인지 확정하지 말 것.
- 파란 선분의 정확한 끝점이 어느 세부 눈금인지는 해상도상 약간 불확실하므로, DSL에서는 “대략 해당 위치” 정도로 처리하거나 TODO로 남길 것.
- `7/7` 표기는 보이지만 픽셀 단위 정렬까지는 확정하지 말 것.
- 화면 왼쪽 위의 작은 사각형 기호는 체크박스인지 단순 인쇄 마크인지 불명확하므로 TODO로 둘 것.
- 비교 문장 `9/7 < 1 3/7`은 화면에 보이는 사실로만 기록하고, 이를 추가 계산으로 확장하지 말 것.
