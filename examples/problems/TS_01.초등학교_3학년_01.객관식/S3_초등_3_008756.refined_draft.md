# [문제 개요]
- 문제 번호는 `53.`이다.
- 상단 문장은 `500 mL 생수병을 보고 들이를 어림해 보려고 합니다. 알맞은 것을 선택하세요.`로 보인다.
- 가운데에는 비교 대상 그림 2개가 좌우로 배치되어 있다.
  - 왼쪽: 생수병
  - 오른쪽: 종이컵
- 아래에는 빈칸 2개가 있는 문장이 이어지고, 선택지 형태의 괄호 안 표현이 함께 보인다.
- 하단에는 `(정답)`, `(해설)`이 실제로 인쇄되어 있다.
- 화면 전체 구조는 `문제 제시 → 그림 비교 → 빈칸 선택 → 정답/해설` 순서이다.

# [원본에 보이는 요소]
## 보이는 사실
- 상단 텍스트:
  - `53. 500 mL 생수병을 보고 들이를 어림해 보려고 합니다. 알맞은 것을 선택하세요.`
- 왼쪽 그림 아래:
  - `생수병`
  - `500 mL`
- 오른쪽 그림 아래:
  - `종이컵`
- 본문 문장:
  - `종이컵의 들이는 500 mL 생수병의 들이보다 ( 많습니다, 적습니다 ).`
  - `→ 종이컵의 들이는 약 ( 180 mL , 180 L )입니다.`
- 하단:
  - `(정답)적습니다, 180 mL`
  - `(해설)종이컵의 들이는 500 mL 생수병의 들이보다 적습니다. 따라서 500 mL보다 적은 180 mL로 어림할 수 있습니다.`

## 렌더링/레이아웃 힌트
- 상단 문장은 한 줄 텍스트로 배치되어 있다.
- 중앙의 두 그림은 좌우로 충분히 떨어져 있다.
- 왼쪽 그림은 병 아이콘, 오른쪽 그림은 종이컵 아이콘이다.
- 각 그림 아래에 라벨이 세로로 배치된다.
  - 왼쪽은 `생수병` 다음에 `500 mL`
  - 오른쪽은 `종이컵`
- 본문은 2줄 구조이며, 1줄째 문장 뒤에 화살표 `→`가 있고 2줄째 문장이 이어진다.
- 괄호 안의 `많습니다, 적습니다`, `180 mL , 180 L`는 선택지처럼 보이는 텍스트 슬롯이다.
- 하단의 `(정답)`, `(해설)`은 본문보다 작게 배치된 별도 영역처럼 보인다.
- 해설 문단은 줄바꿈되어 두 줄로 보인다.

# [수식/관계 구조]
## 보이는 사실
- 화면에는 계산식, 등식, 연산 기호는 보이지 않는다.
- 보이는 관계는 크기 비교 문장과 단위 선택이다.
- 같은 단위 `mL`가 본문과 정답/해설에 반복된다.

## 추론되는 수학적 의미
- `500 mL` 생수병을 기준으로 종이컵의 들이를 어림하는 비교 문제이다.
- 첫 번째 빈칸은 `많습니다 / 적습니다` 중 하나를 고르는 비교 판단이다.
- 두 번째 빈칸은 `180 mL / 180 L` 중 하나를 고르는 어림값 선택이다.
- 다만 정답 판단은 여기서 새로 계산하지 않는다.

# [등호별 변환 의미]
- 해당 없음.
- 화면에 등호 `=`를 이용한 변환 연쇄는 보이지 않는다.
- 따라서 단계별 등호 변형 설명은 없다.

# [빈칸 정책]
## 보이는 사실
- 첫 번째 빈칸은 `( 많습니다, 적습니다 )` 형태이다.
- 두 번째 빈칸은 `( 180 mL , 180 L )` 형태이다.

## 정책
- 빈칸은 추론값으로 채우지 않는다.
- 원본에 정답이 하단에 인쇄되어 보이더라도, DSL용 재작성에서는 빈칸 자체를 빈칸으로 유지하는 것이 원칙이다.
- 선택지 텍스트는 보이는 그대로 기록하되, 무엇이 정답인지 여기서 단정하지 않는다.

# [반복 패턴]
- `들이`라는 표현이 문제 문장, 비교 문장, 해설에 반복된다.
- `500 mL`가 기준 용량으로 여러 번 반복된다.
- 괄호 선택지 형식이 2회 반복된다.
  - 첫 번째: 비교 판단
  - 두 번째: 어림한 수치 선택
- `약`이 들어가 있어 정확값보다 근삿값 선택 구조임을 보여준다.
- 화살표 `→`가 앞 문장과 뒤 문장의 연결을 시각적으로 강조한다.
- 하단의 `(정답)`, `(해설)` 형식은 다른 문제들에서도 재사용 가능한 반복 패턴이다.

# [DSL 작성 힌트]
## 구성 배치
- 상단에 문제 번호와 문제 문장을 한 줄 텍스트로 둔다.
- 중앙에는 두 그림을 가로 배치한다.
  - 왼쪽: 생수병 아이콘
  - 오른쪽: 종이컵 아이콘
- 각 아이콘 아래에 라벨 텍스트를 둔다.
  - 왼쪽: `생수병`
  - 왼쪽 아래: `500 mL`
  - 오른쪽: `종이컵`
- 본문은 두 줄 텍스트로 구성하고, 중간에 `→`를 넣는다.
- 하단에는 `(정답)`과 `(해설)`을 별도 텍스트 블록으로 둔다.

## 렌더링 힌트
- 그림은 실제 사진보다 설명용 아이콘 수준으로 단순화해도 된다.
- 좌우 간격을 넓게 두어 비교 대상임이 분명하게 보이게 한다.
- 선택지는 괄호 안 텍스트로 표현한다.
- 해설은 작은 글씨 또는 아래쪽 보조 텍스트 영역에 배치한다.
- 정답 표시는 원문처럼 그대로 노출하되, 빈칸에 정답을 미리 입력하지 않는다.

# [semantic/solvable 힌트]
## 보이는 사실
- 문제는 그림을 보고 용량을 비교해 어림하는 내용이다.
- `500 mL`와 `180 mL`가 함께 보인다.
- `180 L`도 선택지로 보이지만, 이는 선택지 중 하나일 뿐이다.

## 추론되는 수학적 의미
- 비교 대상은 `500 mL` 생수병과 종이컵이다.
- 종이컵의 들이는 생수병보다 작다고 해석되는 구조이다.
- 어림값은 `mL` 단위로 선택하는 흐름이다.
