# [전체적인 구성]
- 상단에 **문항 번호 27번**이 있고, 그 오른쪽으로 문제 지문이 한 줄~두 줄에 걸쳐 이어집니다.
- 지문 아래 중앙에는 **2행 3열 형태의 표**가 배치되어 있습니다.
- 표 아래에는 **괄호로 묶인 선택 표기**가 따로 있으며, 그 아래쪽 왼편에 **(정답)**, **(해설)** 표기가 이어집니다.
- 전체적으로 **문제 제시 → 표 정보 → 정답/해설 텍스트** 순서의 세로 배치입니다.

# [보이는 텍스트]
- `□ 27.`
- `민선이와 경호는 실제 들이가 1700 mL인 주전자의 들이를 다르게 아림하였습니다. 주전자의 실제 들이에 더 가깝게 아림한 사람은 누구인지 선택해 보세요.`
- 표의 첫 줄:
  - `이름`
  - `민선`
  - `경호`
- 표의 둘째 줄:
  - `어림한 들이`
  - `약 1600 mL`
  - `약 1400 mL`
- 표 아래:
  - `( 민선, 경호 )`
- 아래쪽 텍스트:
  - `(정답)민선`
  - `(해설)민선이가 어림한 들이는 1700 mL에서 약 100 mL가 모자라고,`
  - `경호가 어림한 들이는 1700 mL에서 약 300 mL가 모자랍니다.`
  - `따라서 민선이가 1700 mL에 더 가깝게 어림하였습니다.`

# [반복되는 시각 패턴]
- **이름 비교 표**가 사용되어 두 사람의 값을 나란히 비교하게 되어 있습니다.
- 표의 첫 행은 **항목명(이름)**, 두 번째 행은 **측정/어림 값(어림한 들이)** 구조입니다.
- 값 앞에 모두 `약`이 붙어 있어 **정확한 수가 아니라 어림값**임을 반복해서 보여 줍니다.
- 표 아래의 `( 민선, 경호 )`는 **선택지 형식**으로, 둘 중 하나를 고르게 만드는 패턴입니다.
- 해설 부분에서는 `1700 mL에서 약 100 mL`, `1700 mL에서 약 300 mL`처럼 **기준값과 차이**를 비교하는 문장이 반복됩니다.

# [수식 단계]

- 따라서 별도의 수식 변형 단계는 없습니다.

# [세부 내용]
- 지문은 **실제 들이가 1700 mL인 주전자**를 기준으로, 두 사람의 어림값이 얼마나 가까운지 비교하라고 묻고 있습니다.
- 표의 핵심 정보는:
  - 민선: `약 1600 mL`
  - 경호: `약 1400 mL`
- 이 정보는 **두 값이 실제값 1700 mL와 얼마나 차이나는지 비교해야 한다**는 의미를 전달합니다.
- 표 아래 `( 민선, 경호 )`는 정답을 적거나 선택하는 자리처럼 보입니다.
- 아래의 `(정답)민선`과 `(해설)`은 **문항의 해설 영역**으로 보이며, 이미 정답이 표시된 상태입니다.
- 해설 문장은 각 어림값이 1700 mL와 비교했을 때 **부족한 양**을 설명하는 형태입니다.
- 다만 이 분석에서는 그 차이를 직접 계산해 서술하지 않고, **화면에 적힌 비교 표현만** 설명합니다.

# [Python DSL 작성 힌트]
- 상단 문항은 `문항번호 + 지문` 형태로 배치하면 좋습니다.
- 중앙에는 **표 1개**를 두고, 열은 3개, 행은 2개로 구성합니다.
  - 1행: `이름 | 민선 | 경호`
  - 2행: `어림한 들이 | 약 1600 mL | 약 1400 mL`
- 표 아래에는 선택지처럼 보이는 `( 민선, 경호 )` 텍스트를 가운데 정렬로 배치합니다.
- 하단에는 `(정답)`과 `(해설)`을 별도 텍스트 블록으로 두고, 해설은 줄바꿈을 포함한 문장형으로 배치합니다.
- 표의 테두리는 연한 색의 얇은 선처럼 보이며, 셀 배경은 아주 옅은 베이지/회색 계열로 보입니다.
- 숫자 `1600`, `1400`, `1700`은 `mL` 단위와 함께 일반 본문보다 약간 강조된 크기로 보입니다.
- 전체 페이지는 왼쪽 정렬이 기본이지만, 표는 본문보다 조금 오른쪽에 위치한 중앙형 배치입니다.

# [주의할 점]
- **정답을 추론해서 빈칸에 직접 채우지 마세요.**
- 이 문항은 비교 문제이므로, 분석 단계에서는 **누가 더 가깝다**를 계산 결과로 단정하지 말고 화면에 있는 정보 중심으로만 적습니다.
- 표의 `약`은 **어림값 표시**이므로 정확한 값으로 해석하지 마세요.
- `( 민선, 경호 )`는 보기/선택 영역처럼 보이지만, 실제 입력형 빈칸인지 단순 선택지인지는 화면만으로 완전히 단정하지 마세요.
- 해설의 내용은 이미 제시된 텍스트이므로, 문제를 새로 푸는 설명으로 바꾸지 말고 **보이는 문장 그대로의 구조**를 중심으로 적습니다.

# [불확실한 부분]
- 맨 앞의 작은 네모 기호 `□`가 **체크박스인지 장식 기호인지**는 화면만으로 확정하기 어렵습니다.
- 표 아래의 `( 민선, 경호 )`가 **실제 선택 표시 영역인지, 단순 예시 텍스트인지**는 약간 불확실합니다.
- 하단의 `(정답)민선`은 화면에 직접 보이지만, 이것이 **문제 본문 일부인지 편집용 표시인지**는 문맥상 해설 영역으로 보이나 완전히 단정하기는 어렵습니다.
