[전체적인 구성]
- 상단에 문제 번호와 지시문이 있는 **그림형 객관식/선택형 문제**로 보입니다.
- 화면 중앙에는 **두 명의 어린이 캐릭터**가 좌우로 나뉘어 배치되어 있고, 각 인물 위에 **말풍선**이 하나씩 있습니다.
- 왼쪽은 **나래**, 오른쪽은 **재석**으로 이름표가 각각 인물 아래에 박스 형태로 붙어 있습니다.
- 하단 왼쪽에는 **(정답)** 표기와 함께 짧은 정답 표시가 보이고, 그 아래에는 **(해설)** 문장이 한 줄로 제시되어 있습니다.
- 전체적으로 “누가 더 적절한지 선택”하는 형태의 시각 자료이며, 계산 과정을 여러 단계로 보여주는 구조는 아닙니다.

[보이는 텍스트]
- 상단 문제 문장:
  - `69. 보온병의 둘이를 더 적절히 어린 친구는 누구인지 선택해 보세요.`
  - ※ 문장 중간의 표현은 이미지상 약간 어색하게 보일 수 있으나, 보이는 그대로 옮기면 위와 같습니다.
- 왼쪽 말풍선:
  - `보온병에 200 mL 우유갑으로`
  - `4번쯤 들어갈 것 같아.`
  - `들이는 약 600 mL야.`
- 오른쪽 말풍선:
  - `보온병은 1 L 우유갑과`
  - `들이가 비슷할 것 같아.`
  - `들이는 약 1000 mL야.`
- 왼쪽 인물 이름표:
  - `나래`
- 오른쪽 인물 이름표:
  - `재석`
- 하단 표기:
  - `(정답)재석`
  - `(해설)나래: 200 mL 우유갑으로 4번쯤 들어가면 들이는 약 800 mL 입니다.`

[반복되는 시각 패턴]
- 말풍선이 **좌우 대칭처럼 두 개** 배치되어 있습니다.
- 각 말풍선 아래에 **인물 1명 + 이름표 1개**가 대응되는 구조입니다.
- 이름표는 둘 다 **둥근 모서리의 작은 사각형 박스** 안에 텍스트가 들어간 형태로 보입니다.
- 하단의 `(정답)`, `(해설)` 표시는 모두 **괄호가 포함된 라벨 + 텍스트**의 반복 패턴입니다.
- 문제 해결을 위한 표, 화살표, 빈칸, 분수칸, 등식 변형은 보이지 않습니다.

[수식 단계]
- 별도의 수식 변형 단계는 보이지 않습니다.
- 다만 말풍선 안에 수치 비교가 포함되어 있습니다.
  - 왼쪽: `200 mL × 4번쯤`이라는 의미로 읽히는 설명
  - 오른쪽: `1 L`과 `1000 mL`를 서로 비슷한 것으로 보는 설명
- 하단 해설에는 `200 mL`에 `4번쯤`이라는 표현이 다시 등장하지만, 이는 계산식이 아니라 설명 문장 형태입니다.

[등호별 변환 의미]
- 해당 없음.
- 화면에 `=` 등호가 있는 수식 단계는 보이지 않습니다.

[세부 내용]
- 왼쪽 인물은 분홍색 옷을 입은 여자아이 캐릭터로 보이며, 말풍선에는 보온병의 들이와 관련해 **200 mL 우유갑이 4번쯤 들어갈 것 같고, 그래서 약 600 mL라고 생각함**이라는 취지의 말이 적혀 있습니다.
- 오른쪽 인물은 노란 옷과 안경을 쓴 남자아이 캐릭터로 보이며, 말풍선에는 **1 L 우유갑과 비슷할 것 같고, 그래서 약 1000 mL라고 생각함**이라는 취지의 말이 적혀 있습니다.
- 아래 해설 문장은 왼쪽 주장에 대해 **200 mL가 4번쯤이면 약 800 mL**라는 식의 설명을 적고 있습니다.  
  - 다만 이 문장은 문제 화면에 인쇄된 해설 텍스트를 옮긴 것이며, 여기서 실제 계산을 검토하거나 정답을 재판단하지는 않습니다.
- 전체 그림의 의도는 “보온병의 들이(용량)를 더 적절하게 말한 친구를 고르기”입니다.

[Python DSL 작성 힌트]
- 문제 제목, 말풍선 2개, 인물 2명, 이름표 2개, 하단 정답/해설 텍스트를 **각각 독립 요소**로 배치하는 구성이 적절합니다.
- 좌우 구조를 살려서:
  - 왼쪽 그룹: 말풍선 + 나래 캐릭터 + 이름표
  - 오른쪽 그룹: 말풍선 + 재석 캐릭터 + 이름표
- 말풍선은 위쪽에 크고 둥근 타원형으로, 아래쪽 꼬리가 인물 쪽을 향하도록 배치하면 좋습니다.
- 이름표는 인물 바로 아래, 작고 둥근 사각형 박스로 넣는 구성이 자연스럽습니다.
- 하단에는 `(정답)`과 `(해설)`을 별도 텍스트로 처리하되, 해설은 한 줄 문장 형태로 길게 배치하는 것이 보입니다.
- 수식 박스나 빈칸 입력칸은 이 이미지에는 없으므로 별도 생성이 필요 없어 보입니다.

[주의할 점]
- 정답을 추론해서 다시 쓰지 마세요. 화면에 적힌 정답 표기는 그대로 다루되, 그 옳고 그름을 재계산하지 않는 것이 좋습니다.
- 말풍선 안의 문장은 줄바꿈이 섞여 있으므로, DSL에서는 **줄 단위 텍스트 배치**를 고려해야 합니다.
- 이미지 상단의 문제 문장은 일부 글자가 어색하게 보일 수 있으므로, OCR 결과를 그대로 쓰기보다 **보이는 모양을 우선** 확인하는 편이 좋습니다.
- 하단 해설 문장은 화면에 인쇄된 내용과 의미가 완전히 일치하지 않을 수 있어 보이므로, 해설 영역은 **텍스트 원문 그대로** 다루는 것이 안전합니다.
- 인물 캐릭터의 세부 표정/소품은 수업용 일러스트 요소로 보이며, 수학 정보 자체는 말풍선과 이름표에 집중되어 있습니다.

[불확실한 부분]
- 상단 문제 문장의 일부 표현은 이미지 해상도 때문에 다소 흐리게 보여 정확한 OCR이 어려울 수 있습니다.
- 왼쪽 말풍선의 마지막 줄은 `들이는 약 600 mL야.`로 읽히지만, 일부 글자는 선명도가 낮아 약간의 오독 가능성이 있습니다.
- 오른쪽 말풍선의 가운데 줄은 `들이가 비슷할 것 같아.`처럼 보이지만, 문맥상 글자 일부가 흐릿합니다.
- 하단 해설의 수치 설명은 화면에 보이는 그대로 옮겼으나, 실제 의도와 문장 구성은 추가 확인이 필요해 보입니다.
