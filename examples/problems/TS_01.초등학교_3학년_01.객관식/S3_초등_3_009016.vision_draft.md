# [전체적인 구성]
- 한 장짜리 초등 수학 문제 화면이다.
- 왼쪽 상단에 문제 번호와 지시문이 있고, 그 아래에는 **그림 3개**와 **정답 표시 1개**, **해설 문장 1줄**이 배치되어 있다.
- 전체적으로 “직선”을 **선분**과 비교하는 선택형 문제로 보이며, 오른쪽에는 길게 휘어진 곡선 모양의 도형이 제시되어 있다.
- 화면 하단 왼쪽에는 `(정답)` 표시가 붙은 예시/정답 위치처럼 보이는 그림이 따로 있다.
- 가장 아래에는 `(해설)` 문장으로 정의가 적혀 있다.

# [보이는 텍스트]
- `□ 54. 직선을 선택하세요.`
- `(정답)`
- `(해설) 직선은 선분을 양쪽으로 끝없이 늘인 곧은 선입니다.`

# [반복되는 시각 패턴]
- 화면 중앙 상단에는 **작은 점이 찍힌 선 모양**들이 여러 개 보인다.
- 비슷한 형태의 선 그림이 **왼쪽**, **가운데**, **오른쪽**에 각각 하나씩 배치되어 있다.
- 왼쪽과 가운데의 선 그림은 **꺾이거나 방향이 달라지는 형태**이고, 오른쪽은 **S자처럼 휘어진 곡선**이다.
- 왼쪽 아래에는 `(정답)` 표시와 함께 **한 번 더 비슷한 선 그림**이 제시되어 있어, 위의 그림들과 구별되는 정답 예시처럼 보인다.

# [세부 내용]
- 문제 문구 `직선을 선택하세요.`는 선택지에서 **직선에 해당하는 그림을 고르는 과제**로 보인다.
- 그림은 선의 양 끝 또는 중간에 **작은 검은 점**이 표시되어 있어, 각 선의 위치나 끝점을 강조한 것처럼 보인다.
- 왼쪽 위 그림:
  - 왼쪽에서 오른쪽 위로 올라가는 선분처럼 보이다가, 꼭짓점에서 방향이 꺾여 오른쪽 아래로 내려가는 형태이다.
  - 따라서 전체적으로는 **꺾인 선**처럼 보인다.
- 가운데 그림:
  - 왼쪽 아래에서 오른쪽 위로 뻗는 하나의 곧은 선처럼 보이며, 선 위에 점이 두 개 있다.
  - 다만 양 끝이 화면 안에서 잘려 보이거나 짧게 표현되어 있어, 완전한 직선인지 판단할 때 주의가 필요하다.
- 오른쪽 그림:
  - 길게 휘어진 **곡선** 형태로 보이며, 직선은 아니다.
- 왼쪽 아래 `(정답)` 옆 그림:
  - 왼쪽 아래에서 오른쪽 위로 향하는 **곧은 선** 형태로 보이고, 점이 두 개 찍혀 있다.
  - 해설 문장과 함께 배치되어 있어 정답 예시 또는 정답 위치를 나타내는 것으로 해석된다.
- 해설 문장:
  - 직선을 “선분을 양쪽으로 끝없이 늘인 곧은 선”이라고 설명한다.
  - 이는 문제의 목적이 직선의 정의를 보고 그림 중 직선을 고르게 하는 것임을 보여준다.

# [Python DSL 작성 힌트]
- 상단 제목 영역과 본문 그림 영역을 분리해서 배치하면 좋다.
- `직선을 선택하세요.` 문장은 문제 번호 `54.`와 같은 줄에 두고, 왼쪽 상단에 정렬한다.
- 중앙의 보기 그림들은 가로 방향으로 3개를 나란히 배치하되, 각 그림 사이 간격을 넉넉히 둔다.
- 왼쪽 아래의 `(정답)` 표시는 작은 텍스트로, 그 오른쪽 또는 아래에 별도의 선 그림을 배치한다.
- 해설 문장은 하단 전체 폭에 가깝게 한 줄로 두되, `(해설)` 라벨을 먼저 표시하고 그 뒤에 문장을 이어 쓰면 좋다.
- 그림 요소는 다음처럼 개별 객체로 다루면 좋다:
  - 꺾인 선 1개
  - 곧은 선 1개
  - 곡선 1개
  - 정답 예시용 곧은 선 1개
- 선 위의 점은 작은 원형 점으로 표시하고, 각 그림에 대응되게 위치를 맞춘다.

# [주의할 점]
- **정답을 추론해서 텍스트로 확정하지 말아야 한다.**
- 화면에 보이는 정답 표시가 있어도, 실제로 어떤 보기를 정답으로 처리할지는 DSL 작성 단계에서 다시 확인해야 한다.
- 왼쪽 위 그림은 꺾인 부분이 있어 직선과 다르게 보일 수 있으므로, 단순한 선분과 혼동하지 않도록 한다.
- 가운데 그림은 곧은 선처럼 보이지만, 화면 해상도나 잘림 때문에 끝점이 정확히 보이지 않을 수 있다.
- 오른쪽 그림은 분명한 곡선 형태로 보이므로 직선이 아님을 시각적으로 구분해야 한다.
- 문제 번호 앞의 작은 빈 네모 `□`는 체크박스처럼 보일 수 있으나, 의미가 확실하지 않으면 장식 요소로 취급한다.

# [불확실한 부분]
- 가운데 그림이 **완전한 직선인지**, 아니면 일부가 잘려 보이는 선분인지 화면만으로는 엄밀히 단정하기 어렵다.
- 왼쪽 아래 `(정답)` 옆 그림이 실제 정답 표시인지, 아니면 설명용 예시인지 시각적으로는 정답처럼 보이지만, 파일 구조상 별도 처리 여부는 확정할 수 없다.
- 상단 제목 왼쪽의 작은 `□` 기호는 체크박스인지 문제 번호 표식인지 불확실하다.
