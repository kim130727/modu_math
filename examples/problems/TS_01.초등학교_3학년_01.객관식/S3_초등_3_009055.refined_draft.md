[문제 개요]
- 문제 번호는 `93.`으로 보인다.
- 지시문은 `직각이 있는 삼각형을 모두 선택하세요.`이다.
- 상단에는 점선 격자 배경 위에 삼각형 6개가 배치되어 있다.
- 삼각형 내부에는 각각 `가, 나, 다, 라, 마, 바`라는 라벨이 보인다.
- 하단에는 `(정답) 나, 바`라는 문구와 `(해설)` 표시가 따로 보이며, 같은 삼각형들이 다시 배치되어 있다.

  - 하단 해설용 그림 1개 묶음

  - 연한 파란색 계열의 점선 격자 배경

[수식/관계 구조]
- 등호로 연결된 수식은 보이지 않는다.
- 수치 계산식도 보이지 않는다.
- 문제는 도형 선택형이며, 삼각형의 직각 여부를 판단하는 시각적 분류 구조로 보인다.
- 보이는 사실과 추론되는 수학적 의미를 분리하면:
  - 보이는 사실: 삼각형 6개 중 일부가 직각처럼 보이도록 배치되어 있다.
  - 추론되는 수학적 의미: 직각이 있는 삼각형을 고르는 활동형 문제이다.
- 단, 실제 정답 판정은 여기서 새로 하지 않는다.

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 변형 단계가 없다.

[빈칸 정책]
- 빈칸은 보이는 대로 빈칸으로 유지한다.
- 보이지 않는 정답, 좌표, 각도, 숨은 도형 정보는 추론하여 채우지 않는다.
- 원본에 인쇄된 `정답) 나, 바`는 보이는 텍스트로만 기록하고, 빈칸을 메우는 근거로 사용하지 않는다.

[반복 패턴]
- 삼각형 6개가 반복적으로 등장한다.
- 상단 문제 그림과 하단 해설 그림은 같은 유형의 삼각형 배치를 다시 보여 주는 구조로 보인다.
- 각 삼각형은 모두 선분 3개로 이루어진다.
- 내부 라벨은 각 도형 중앙 부근에 한 글자씩 들어간다.
- 점선 격자 배경이 상단/하단 모두에 반복된다.
- 해설 영역은 문제 그림을 재배치하거나 축소한 유사 구성으로 보인다.

[DSL 작성 힌트]
- 상단 문제 영역과 하단 해설 영역을 서로 분리된 그룹 또는 캔버스로 나누는 것이 좋다.
- 배경은 연한 파란색 점선 격자 스타일로 처리한다.
- 삼각형은 각각 독립 도형으로 두고, 각 변을 선분 3개로 표현한다.
- 내부 글자 `가, 나, 다, 라, 마, 바`는 삼각형 중심 근처에 배치한다.
- `93.`과 지시문은 상단 좌측 텍스트로 묶되, 문제 번호와 문장을 한 줄 구성으로 맞추는 것이 좋다.
- `(정답) 나, 바`는 상단 그림 아래 좌측에 별도 텍스트로 둔다.
- `(해설)`은 하단 그림의 좌측 또는 상단 좌측에 작은 라벨처럼 배치한다.
- 삼각형 위치는 격자선에 맞춘 상대 좌표 배치가 적절하다.
- 일부 변은 수평/수직에 가깝게 보이므로, 완전한 직선 정렬 여부는 너무 엄격하게 고정하지 않는 편이 안전하다.

- 이 문제는 도형의 성질을 보고 직각이 있는 삼각형을 고르는 유형으로 해석된다.
- 화면에 `(정답) 나, 바`가 인쇄되어 보이지만, 이는 관찰 사실일 뿐 여기서 답을 새로 계산한 것은 아니다.
- 직각 여부를 판정하는 기준은 도형의 변이 서로 수직인지 여부로 추정된다.
- 다만 실제 판정 로직을 여기서 단정하거나 재계산하지 않는다.
