[문제 개요]
- 초등 수학 문제 54번.
- 가, 나, 다 3개의 그릇을 물로 가득 채운 뒤, 모양이 같은 그릇에 옮겨 담았을 때의 그림을 보고, **들이가 가장 적은 그릇**을 고르는 선택형 문제.
- 원본 이미지에는 문제 문장, 3개의 그림 비교, 보기 형식의 문자열, 정답 표시, 해설 문장이 함께 보인다.
- **보이는 사실**: 정답과 해설이 이미지 안에 인쇄되어 있음.
- **추론되는 수학적 의미**: 물의 높이를 비교하여 그릇의 들이(용량)를 상대적으로 판단하는 상황.

[원본에 보이는 요소]
- 좌상단에 `□ 54.`가 보인다.
- 본문 문장:
  - `가 그릇, 나 그릇, 다 그릇에 물을 가득 채운 후 모양과 같은 그릇에 옮겨 담았습니다.`
  - `그림과 같이 물을 채웠을 때 가 그릇, 나 그릇, 다 그릇 중 들이가 가장 적은 것은 어느 것인지 선택하세요.`
- 가운데에 3개의 그림 묶음이 가로로 배치되어 있다.
  - 각 묶음은 `비어 있는 그릇` → `오른쪽 화살표` → `물이 채워진 그릇` 구조.
  - 각 묶음 위쪽에 동그라미 친 문자 `가`, `나`, `다`가 각각 표시됨.
- 오른쪽 아래 쪽에 `(가 그릇, 나 그릇, 다 그릇)` 문구가 보인다.
- 하단 왼쪽에 `(정답) 나 그릇`이 인쇄되어 보인다.
- 그 아래 해설:
  - `(해설) 그릇에 담긴 물의 높이를 비교하면 나 그릇이 가장 낮으므로 나 그릇의 들이가 가장 적습니다.`
- **보이는 사실**: 해설 문장에는 `나 그릇`이 직접 적혀 있다.
- **주의**: 그러나 DSL용 정리에서는 정답을 새로 추론해 채우지 말고, 보이는 텍스트를 그대로 기록하는 수준으로만 다룬다.

[수식/관계 구조]
- 이 문제에는 등호 `=` 를 포함한 계산식이나 식 변형 연쇄가 보이지 않는다.
- 대신 **비교 구조**가 중심이다.
  - 3개의 그릇 각각을 같은 방식으로 물을 채운 뒤,
  - 채워진 물의 **높이**를 서로 비교함.
- **추론되는 수학적 의미**
  - 같은 양의 물을 넣었을 때 물 높이가 낮을수록 그릇의 들이가 더 크거나, 반대로 같은 조건 비교에서 문제 의도상 “들이다가 가장 적은 그릇”을 물 높이로 판단하게 구성되어 있다.
  - 다만 이 해석은 문제의 그림 비교를 읽는 의미이며, 계산식은 없음.

[등호별 변환 의미]
- 해당 없음.
- 원본에 등호 연결식, 단계적 변형, 분수식 변환이 보이지 않는다.
- 따라서 각 단계의 수학적 변환 의미를 설명할 대상이 없다.

[빈칸 정책]
- 원본 이미지에서 **빈칸을 채우는 형태의 서술형 칸은 보이지 않는다**.
- 따라서 별도의 답칸을 추론해 작성하지 않는다.
- **빈칸 정책 원칙**
  - 보이지 않는 값은 추론하여 채우지 않는다.
  - 정답이 이미지에 인쇄되어 보이더라도, DSL용 정리에서는 관찰 사실로만 적고 빈칸을 메우지 않는다.
  - 필요하면 `빈칸 없음` 또는 `해당 없음`으로 표시한다.

[반복 패턴]
- 동일한 비교 구조가 3회 반복된다.
  1. `가` 그릇: 왼쪽 비어 있는 그릇 → 화살표 → 오른쪽 채워진 그릇
  2. `나` 그릇: 왼쪽 비어 있는 그릇 → 화살표 → 오른쪽 채워진 그릇
  3. `다` 그릇: 왼쪽 비어 있는 그릇 → 화살표 → 오른쪽 채워진 그릇
- 반복 요소:
  - 그릇 1개
  - 수평 화살표 1개
  - 결과 그릇 1개
  - 위쪽 라벨 1개
- 색상 차이로 각 항목을 구분:
  - `가`: 파란색 계열
  - `나`: 연두색 계열
  - `다`: 갈색/붉은색 계열
- **렌더링 관찰**
  - 물 채움은 원통형 투명 용기 내부의 물 높이로 표현되어 있다.
  - 세 그림은 같은 행에서 간격을 두고 배치되어 있다.
  - 화살표는 각 초기 그릇과 채워진 그릇 사이를 연결하는 수평 방향 요소이다.

[DSL 작성 힌트]
- 레이아웃은 다음 순서로 분리하는 것이 좋다.
  1. 상단 문제 번호와 문장 텍스트
  2. 중앙의 3개 비교 그림 세트
  3. 각 그림 위의 `가/나/다` 라벨
  4. 각 세트 사이의 화살표
  5. 우측 하단의 `(가 그릇, 나 그릇, 다 그릇)` 문구
  6. 하단의 정답/해설 텍스트 블록
- 그림 배치 힌트:
  - 세 세트는 가로로 균등 배치.
  - 각 세트 안에서 `왼쪽 그릇`과 `오른쪽 채워진 그릇` 사이에 화살표를 둔다.
  - 라벨은 해당 그릇 묶음의 왼쪽 위 또는 상단에 작은 원형 문자로 표시한다.
- 도형 표현 힌트:
  - 그릇은 단순한 컵/원통 윤곽선으로 표현 가능.
  - 채워진 상태는 내부 물 영역 색으로 구분.
  - 물 높이 차이는 높이 비율로 시각적으로 드러내면 된다.
- 텍스트 처리 힌트:
  - 본문은 줄바꿈이 2줄로 나뉘어 보인다.
  - `(정답)`, `(해설)`은 별도 텍스트 블록으로 분리하면 좋다.
- **보이는 사실과 레이아웃 힌트 분리**
  - 보이는 사실: 3개의 그릇, 화살표, 라벨, 보기 문구, 정답/해설.
  - 레이아웃 힌트: 가로 정렬, 간격, 위쪽 라벨, 하단 설명 분리.

[semantic/solvable 힌트]
- 이 문제는 계산 문제가 아니라 **시각적 비교를 통해 선택하는 문제**이다.
- 핵심 의미는 `물의 높이 비교 → 들이의 상대적 판단`이다.
- 문제 문장상 요구되는 것은 “들이가 가장 적은 것”을 고르는 것이지만, 이 문서는 풀이를 완성하지 않는다.
- 원본 해설은 `나 그릇`을 지시하고 있으나, DSL 작성에서는 이를 정답으로 확정해 채우지 말고 **이미지에 인쇄된 관찰 사실**로만 기록한다.
- **보이는 사실**: 해설 문장에 `나 그릇이 가장 낮으므로`라는 표현이 있다.
- **추론되는 수학적 의미**: 가장 낮은 물 높이의 그릇을 작은 들이로 판단하도록 의도한 문제 구성.
