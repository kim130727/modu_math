# 시각 분석 초안

## [전체적인 구성]
- 문제 번호 **71.**이 왼쪽 상단에 있고, 그 오른쪽에 문제 문장이 한 줄로 배치되어 있습니다.
- 화면 중앙에는 **정현이라는 캐릭터 소개**와 함께, 말풍선 형태의 설명 문장, 그리고 아래쪽에 **저울 그림 2개(비교 전/후)**가 좌우로 배치되어 있습니다.
- 저울 그림 사이에는 **오른쪽을 가리키는 회색 화살표**가 있어, 왼쪽 그림에서 오른쪽 그림으로 상황이 바뀌는 흐름을 나타냅니다.
- 하단에는 문장형 답안/해설이 텍스트로 배치되어 있습니다.

## [보이는 텍스트]
- 상단 문제 문장:
  - **“71. 정현이가 자두와 배의 무게를 비교하려고 합니다. 알맞은 말을 선택하세요.”**
- 말풍선 안 문장:
  - **“저울을 사용하여 자두와 배의 무게를 비교하자.”**
- 저울 위 안내 문장(초록색):
  - **“윗접시저울로 무게를 비교하려면 먼저 저울의 영점을 맞춰야 해요.”**
- 사물 이름 라벨:
  - **“자두”**
  - **“배”**
- 하단 문장:
  - **“배를 올려놓은 접시가 아래로 내려갔으므로 ( 자두 , 배 )가 더 무겁습니다.”**
- 정답 표시:
  - **“(정답)배”**
- 해설 문장:
  - **“배를 올려놓은 접시가 아래로 내려갔으므로 배가 더 무겁습니다.”**

## [반복되는 시각 패턴]
- 저울 그림이 **두 번** 등장합니다.
  - 왼쪽: 빈 접시 상태의 저울
  - 오른쪽: 과일을 올려 비교한 저울
- 오른쪽 그림에서는
  - 왼쪽 접시 위에 **자두**
  - 오른쪽 접시 위에 **배**
  가 각각 올려져 있으며, 각 과일 위에 **작은 색상 라벨 상자**가 붙어 있습니다.
- 말풍선과 초록색 안내 문장은 모두 **설명용 텍스트 블록**으로, 문제 풀이보다 상황 이해를 돕는 역할입니다.
- 문제 하단에는 **빈칸을 선택하는 형태**로 보이는 괄호 **( 자두 , 배 )**가 들어가 있습니다.  
  - 이 괄호는 보기 선택 또는 빈칸 채우기용 표현처럼 보입니다.

## [수식 단계]
- 해당 없음.
- 이 문제 화면에는 등호를 이용한 수식 변형 단계가 보이지 않습니다.

## [등호별 변환 의미]
- 해당 없음.
- 등호를 포함한 식이 제시되어 있지 않습니다.

## [세부 내용]
- 화면 왼쪽 상단에는 작은 네모 체크박스처럼 보이는 표시가 있고, 그 옆에 문제 번호가 있습니다.
- 정현 캐릭터는 말풍선 왼쪽 근처에 배치되어 있으며, 이름표 **“정현”**이 연한 초록색 둥근 사각형 안에 표시되어 있습니다.
- 말풍선은 연분홍색의 큰 말풍선 형태이고, 정현이 말하는 내용처럼 보입니다.
- 중앙 위쪽의 초록색 문장은 **곡선 화살표**로 저울의 중심 부분을 가리키고 있어, 저울의 영점을 맞추는 위치를 안내하는 시각적 요소로 읽힙니다.
- 아래의 두 저울 그림은 비교 전후를 보여주는 구조입니다.
  - 왼쪽 저울: 양쪽 접시가 비어 있고 균형이 맞은 상태처럼 보입니다.
  - 오른쪽 저울: 한쪽에는 자두, 다른 쪽에는 배가 올려져 있으며, 한쪽 접시가 내려간 모습입니다.
- 하단 문장은 관찰 결과를 서술하는 형태로 구성되어 있고, 괄호 안에 **“자두, 배”** 두 보기 중 하나를 고르는 구조처럼 보입니다.
- 정답과 해설이 별도로 아래쪽에 제시되어 있어, 학습용 정답 확인 화면처럼 보입니다.

## [Python DSL 작성 힌트]
- 전체 화면을 **상단 문제문 + 중앙 설명/그림 + 하단 답/해설**의 3단 구성으로 나누어 배치하면 좋습니다.
- 저울 그림은 **왼쪽(비교 전) / 오른쪽(비교 후)**의 2패널 구조로 구현할 수 있습니다.
- 오른쪽 저울 그림에서는:
  - 왼쪽 접시 위 과일 1개
  - 오른쪽 접시 위 과일 1개
  - 각 과일 위의 라벨 박스
  - 두 그림 사이의 방향 화살표
  를 별도 객체로 두면 좋습니다.
- 초록색 안내 문장은 저울 중심부를 향하는 **곡선 화살표**와 함께 배치해야 합니다.
- 하단 문장에는 **괄호로 묶인 선택 항목**이 포함되므로, 텍스트 객체에서 괄호와 쉼표 위치를 그대로 유지하는 것이 중요합니다.
- 정답과 해설은 문제 본문과 시각적으로 분리된 아래쪽 텍스트 블록으로 처리하면 됩니다.

## [주의할 점]
- **정답을 추론해서 빈칸에 미리 채우지 마세요.**
- “( 자두 , 배 )”는 화면에 보이는 선택형 표현이므로, 실제로 무엇이 정답인지 별도로 판단해 채워 넣지 않는 것이 좋습니다.
- 저울 그림의 세부 비율이나 색은 참고용이므로, DSL에서는 **배치 관계와 구성 요소**를 우선 반영하세요.
- 오른쪽 저울에서 어느 접시가 더 낮은지에 대한 해석은 보이지만, 이 초안에서는 **결론을 문제풀이 형태로 단정하지 않고** 시각 요소만 설명해야 합니다.
- “윗접시저울”, “영점” 같은 용어는 화면에 명시되어 있으나, 그림만으로 추가 의미를 과도하게 확장하지 않도록 주의하세요.

## [불확실한 부분]
- 왼쪽 상단의 작은 네모 표시가 **체크박스인지, 인쇄용 문제 표시인지**는 정확히 단정하기 어렵습니다.
- 하단의 **“(자두, 배)”**가 정확히 단일 선택지인지, 문장 속 삽입형 빈칸인지 화면만으로는 다소 애매합니다.
- 저울의 실제 수평 상태나 접시 높이 차이는 그림 스타일상 강조된 연출일 수 있어, 수치적 해석은 하지 않는 것이 안전합니다.
