[문제 개요]
- 문제 번호는 **86.**으로 보인다.
- 상단에는 물병 그림을 보고 **두 물병의 물을 모아 3 L가 되게 하는 물병을 모두 고르라**는 내용의 문장형 문제가 있다.
- 본문 아래에는 연보라색 테두리의 큰 박스 안에 물병 3개가 좌→우로 배치되어 있다.
- 박스 오른쪽 아래 바깥쪽에는 선택지 형태의 문구가 있다.
- 하단에는 **정답 표기**와 **해설 문장**이 인쇄되어 보인다.

[원본에 보이는 요소]
- 문제 번호: **86.**
- 문제 문장: **다음과 같이 물병에 물이 가득 채워져 있습니다. 두 물병의 물을 모아 3 L가 되게 하려면 어느 물병을 골라야 하는지 모두 선택하세요.**
- 그림 속 라벨:
  - **가**
  - **나**
  - **다**
- 각 물병 아래 용량 표기:
  - **1 L 700 mL**
  - **1 L 100 mL**
  - **1 L 300 mL**
- 선택지 표기:
  - **( 가 물병, 나 물병, 다 물병 )**
- 하단 인쇄 텍스트로 보이는 내용:
  - **(정답) 가 물병, 다 물병**
  - **(해설) 1 L 700 mL + 1 L 300 mL = 2 L + 1000 mL = 3 L이므로 가와 다 물병을 골라야 합니다.**

[수식/관계 구조]
- 본문 자체에는 계산식이 직접 제시되지 않고, 아래 해설 영역에 등호 연쇄식이 보인다.
- 보이는 식의 구조는 다음과 같다:
  - **1 L 700 mL + 1 L 300 mL = 2 L + 1000 mL = 3 L**
- 이는 두 물병의 용량 합을 여러 단위 표현으로 순서대로 적은 형태로 보인다.
- 단, 이 문서는 풀이를 새로 만들지 않고 **화면에 인쇄된 식의 형태만 기록**한다.

[등호별 변환 의미]
- **1 L 700 mL + 1 L 300 mL**
  - 두 물병의 용량을 더한 첫 단계로 보인다.
- **= 2 L + 1000 mL**
  - 리터 단위끼리 묶고, 밀리리터 단위끼리 묶어 적은 것으로 보인다.
- **= 3 L**
  - 앞 단계의 합을 최종적으로 3 L로 정리한 것으로 보인다.
- 해설에 따라, 두 물병의 합이 3 L가 되는 조합을 설명하는 흐름으로 읽힌다.

[빈칸 정책]
- 화면에는 **빈칸 입력칸이 보이지 않는다.**
- 따라서 답을 적는 빈칸을 새로 만들거나, 빈칸을 추론값으로 채우지 않는다.
- 인쇄된 정답 표기가 보이더라도, 이는 **관찰 사실**로만 기록하고 빈칸 대체값으로 사용하지 않는다.

[반복 패턴]
- 물병 그림이 **3개** 반복된다.
- 각 물병 위에는 동그라미 안 한글 라벨 **가 / 나 / 다**가 반복된다.
- 각 물병 아래에는 **“1 L + mL”** 형태의 용량 표기가 반복된다.
- 선택지에서도 **“○ 물병”** 형식이 3개 나열된다.
- 전체적으로 **좌→우 3분할 배치**가 반복 구조의 핵심이다.

[DSL 작성 힌트]
- 상단 문제 문장은 한 줄 또는 두 줄의 **가로 문장형 텍스트**로 배치하면 된다.
- 그 아래 큰 박스는 **연보라색 얇은 테두리의 직사각형**으로 표현하는 것이 적절하다.
- 박스 안에는 물병 3개를 **좌-중-우 정렬**로 균등 배치한다.
- 각 물병 상단 근처에 **작은 원형 라벨(가/나/다)**을 넣는다.
- 각 물병 아래에는 가운데 정렬로 용량 텍스트를 둔다.
- 물병 그림은 정밀한 실물 묘사보다 **아이콘/삽화 수준**으로 처리하고, 핵심은 라벨과 용량 텍스트이다.
- 선택지는 박스 바깥 오른쪽 아래에 **괄호 포함 텍스트**로 배치한다.
- 하단 해설 영역은 정답 표기와 해설 문장을 **서로 분리된 텍스트 블록**으로 두는 구성이 적절하다.
- 등호가 연속된 수식은 **하나의 긴 수식 문자열**로 유지해도 된다.

[semantic/solvable 힌트]
- 문제 의미는 “주어진 물병들 중 두 개를 골라 물의 합이 3 L가 되는 조합을 찾기”로 읽힌다.
- 물병마다 용량이 숫자 텍스트로 직접 제시되어 있으므로, **그림 크기만 보고 판단하지 않도록** 해야 한다.
- 본문은 복수 선택 형태이지만, 실제 선택 결과는 **그림 아래의 용량 표기**를 기준으로 이해된다.
- 해설에 인쇄된 등호식은 합이 3 L가 되는 예시를 보여 주는 것으로 보인다.
