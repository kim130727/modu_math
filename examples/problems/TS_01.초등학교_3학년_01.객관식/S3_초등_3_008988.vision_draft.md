# [Python DSL 작성 힌트]
- 이 문제는 **도형 판별형 단일 선택 문제**로 작성하면 됩니다.
- DSL에는 다음 요소가 필요해 보입니다.
  - 문제 번호 텍스트
  - 지시문 텍스트
  - 삼각형 도형 1개
  - 선택 기호 `○`, `×` 안내
  - 정답/해설 텍스트 영역
- 도형은 **삼각형 윤곽선**으로만 표현하고, 내부에 직각 표시가 없다는 점을 반영하세요.
- 좌우/상하 위치를 구분해서:
  - 상단: 문제 문장
  - 우측 상단: 도형
  - 좌측 하단: 정답/해설
  로 배치하는 것이 자연스럽습니다.
- 선택 기호는 **빈 선택지**처럼 보이게 처리할 수 있습니다. 다만 실제 정답은 추론하지 말고, 그림에 보이는 표식을 그대로 옮기세요.

# [주의할 점]
- **문제를 풀어 정답을 확정하지 마세요.** 화면에 이미 ` (정답)× `가 보이더라도, 분석에서는 “보인다” 수준으로만 서술하는 것이 안전합니다.
- 삼각형이 **직각삼각형인지 아닌지**를 해석하더라도, 그 판단 근거를 새로 계산해 적지 마세요.
- 파란색 원형 기호는 **편집용 표시인지 실제 선택지인지 불명확**하므로 단정하지 마세요.
- 도형 내부의 각도나 선분 길이는 화면 해상도상 정확히 판독되지 않을 수 있으니, **보이는 외곽선 형태 중심**으로 설명하세요.
- `○`와 `×`는 지시문에 명시된 기호이므로, 이를 별도의 수식이나 답안 칸으로 오해하지 않도록 주의하세요.
