[문제 개요]
- 문제 번호는 `66.`로 보인다.
- 상단에 “저울을 사용하여 공책과 필통의 무게를 비교하려고 합니다. 알맞은 말을 선택하세요.”라는 문장이 있다.
- 가운데에는 양팔저울 그림이 있고, 저울 위 물체에 `공책`, `필통` 라벨이 붙어 있다.
- 아래에는 선택형 문장 `공책이 필통보다 더 ( 무겁습니다, 가볍습니다 ).`가 제시된다.
- 하단에는 `(정답) 가볍습니다`, `(해설) 공책을 올려놓은 접시가 위로 올라갔으므로 공책이 필통보다 더 가볍습니다.`가 보인다.

[원본에 보이는 요소]
- 보이는 사실
  - `□ 66.` 표시가 왼쪽 위에 있다.
  - 문제 문장이 두 줄로 배치되어 있다.
  - 중앙에 저울 그림이 있다.
  - 저울의 왼쪽 물체는 `공책`, 오른쪽 물체는 `필통`으로 라벨링되어 있다.
  - 선택 문장에는 괄호 안에 두 선택어 `무겁습니다`, `가볍습니다`가 나란히 있다.
  - 아래에 `(정답)`과 `(해설)` 텍스트가 별도 줄로 있다.
- 추론되는 수학적 의미
  - 저울의 기울기와 접시 높이 차이로 두 물체의 무게 비교를 묻는 문제로 보인다.
  - 다만 본 답안 문서에서는 결과를 새로 추론해 서술하기보다, 이미지에 적힌 문구만 관찰 사실로 다루는 것이 안전하다.

[수식/관계 구조]
- 해당 문제는 숫자 계산식이나 등호 연쇄가 없다.
- 비교 관계는 다음과 같은 문장 구조로만 제시된다.
  - `공책이 필통보다 더 ( 무겁습니다, 가볍습니다 ).`
- 수식 변환 단계는 없음.
- 분수, 나눗셈, 등호, 부등호 표기는 보이지 않는다.

[등호별 변환 의미]
- 해당 없음.
- 등호를 포함한 변형식이 제시되지 않는다.
- 문장 선택형 비교 문제이므로 등호별 단계 설명 대상이 없다.

[빈칸 정책]
- 선택지 괄호 `(` `)` 안의 문구는 빈칸 성격으로 취급한다.
- DSL 작성 시에는 추론값을 채워 넣지 말고, 원문에 보이는 선택어 목록을 그대로 유지한다.
- 특히 `무겁습니다/가볍습니다` 중 어느 쪽을 정답으로 확정하여 본문에 삽입하지 않는다.
- 하단에 `(정답) 가볍습니다`가 인쇄되어 보이지만, 이는 원본 관찰 사실로만 기록하고 새로 계산해 넣는 방식은 피한다.

[반복 패턴]
- 물체 이름 라벨이 작은 둥근 사각형 박스 안에 들어가 있다.
- 라벨 박스는 각 물체 위쪽에 배치되어 있으며, 저울의 좌우 물체와 대응된다.
- 양팔저울은 좌우 대칭 구조로 표현되어 있다.
- 선택 문장은 본문 문장 뒤에 괄호 선택지가 붙는 형식이다.
- 하단의 정답/해설은 문제 본문과 시각적으로 분리된 별도 블록으로 반복 배치되는 형식이다.

[DSL 작성 힌트]
- 화면을 크게 다음 3영역으로 나누면 좋다.
  1. 상단 문제 문장 영역
  2. 중앙 그림 영역
  3. 하단 선택문장 및 정답/해설 영역
- 저울 그림은 구성 요소를 분리해 표현하기 좋다.
  - 중앙 받침대
  - 중앙 기둥
  - 좌우 팔
  - 좌우 접시
  - 왼쪽 접시 위 공책
  - 오른쪽 접시 위 필통
  - 각 물체 위 라벨 박스
- 그림의 핵심 레이아웃 힌트
  - 저울은 중앙 기준으로 좌우 균형형 배치
  - 왼쪽 접시가 위로, 오른쪽 접시가 아래로 기울어진 상태
  - 물체는 각각 접시 위에 놓여 있음
- 텍스트 레이아웃 힌트
  - 문제 문장은 상단 좌측 정렬
  - 선택 문장은 그림 아래 중앙에 크게 배치
  - `(정답)`, `(해설)`은 그 아래 별도 줄로 작게 배치
- 선택지는 하나의 문장 안에 포함되며, 괄호 안 두 항목은 텍스트 상으로 함께 보여 주는 방식이 자연스럽다.

[semantic/solvable 힌트]
- 보이는 그림과 해설 문구는 “공책 쪽 접시가 위에 있다”는 상황을 나타낸다.
- 저울에서 위로 올라간 쪽이 더 가벼운 쪽이라는 해설 문장이 함께 인쇄되어 있다.
- 따라서 이 문항은 비교 어휘 선택 문제로 이해된다.
- 다만 DSL 문서에서는 정답 판정 과정을 새로 계산해 서술하기보다, 원문에 인쇄된 표현을 그대로 기록하는 것이 좋다.
