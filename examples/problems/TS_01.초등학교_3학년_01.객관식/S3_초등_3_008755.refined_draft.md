[문제 개요]
- 초등 수학 1문항 형태의 단위 선택 문제이다.
- 주스병과 우유갑에 표시된 용량을 보고, 각각 어떤 단위인지 고르는 구조이다.
- 화면에는 상단 문제 문장, 가운데의 물체 그림 2개, 말풍선 설명, 아래쪽의 단위 선택 문장, 그리고 하단의 `(정답)`, `(해설)` 표기가 함께 보인다.
- **보이는 사실**: 주스병에는 `1L`, 우유갑에는 `200mL`가 적혀 있다.
- **추론되는 수학적 의미**: `L`과 `mL`는 각각 리터, 밀리리터 단위를 뜻하며, 물체의 용량 단위를 연결하는 문제로 보인다.

[원본에 보이는 요소]
- 상단 문장: `52. 주스병과 우유갑에는 각각 어떤 단위가 표시되어 있는지 선택해 보세요.`
- 그림 왼쪽: 주스병 그림과 아래 이름 `주스병`
- 그림 가운데: 우유갑 그림과 아래 이름 `우유갑`
- 말풍선: `들이의 단위에는 리터와 밀리리터 등이 있어.`
- 단위 선택 문장: `주스병 → ( L, mL ), 우유갑 → ( L, mL )`
- 하단 표기:
  - `(정답) L, mL`
  - `(해설) 주스병 → L, 우유갑 → mL`
- 그림 안/확대 원 안의 표시:
  - 주스병 쪽 `1L`
  - 우유갑 쪽 `200mL`
- 색 강조:
  - 말풍선 안의 `리터`, `밀리리터`가 다른 색으로 강조되어 있음
- **보이는 사실**: `정답`, `해설` 텍스트가 화면에 인쇄되어 있다.
- **주의**: 하단의 정답/해설이 보이더라도, 답을 새로 추론해 채우는 방식은 피해야 한다.

[수식/관계 구조]
- 등호를 포함한 계산식은 보이지 않는다.
- 이 문항은 계산 과정보다 **대상-단위 대응 구조**가 핵심이다.
- 구조적으로는 다음과 같은 대응을 읽을 수 있다.
  - `주스병` ↔ `L`
  - `우유갑` ↔ `mL`
- 괄호 안 선택지 구조:
  - `주스병 → ( L, mL )`
  - `우유갑 → ( L, mL )`
- 화살표는 “대상에 알맞은 단위를 고르기”를 나타내는 시각 요소로 해석된다.
- **보이는 사실**: 주스병의 용량 표시는 `1L`, 우유갑의 용량 표시는 `200mL`이다.
- **추론되는 수학적 의미**: 큰 용량에는 `L`, 비교적 작은 용량에는 `mL`가 쓰인다는 개념을 확인시키는 문제로 보인다.

[등호별 변환 의미]
- 해당 없음.
- 등호로 이어지는 변환식이나 단계별 계산식이 없다.
- 따라서 “어떤 항이 어떻게 바뀌는지”를 설명할 변환 단계도 없다.

[빈칸 정책]
- 이 문제에는 수학식의 빈칸이 아니라, 선택형 괄호 구조가 있다.
- 괄호 안의 `L`, `mL`는 **보기(선택지)** 로 보이며, 이를 임의로 채워 넣지 않는다.
- 하단의 `(정답) L, mL`, `(해설) 주스병 → L, 우유갑 → mL`는 이미지에 **인쇄되어 보이는 텍스트**이지만, DSL 작성에서는 이를 정답 확정 근거로 사용하지 않고 **관찰 사실**로만 기록한다.
- 빈칸이 따로 주어진 것으로 보이지 않으므로, 추론값을 별도로 생성해 채우지 않는다.

[반복 패턴]
- 원형 확대 표시가 2개 반복된다.
  - 왼쪽 원: 주스병의 `1L` 확대
  - 오른쪽 원: 우유갑의 `200mL` 확대
- 각 확대 원에서 해당 물체의 표시 위치로 점선 연결선이 이어진다.
- 물체명 아래에 이름 라벨이 붙는 패턴이 반복된다.
- 단위 선택 문장에서 `→`와 `( L, mL )`가 두 번 반복되어, 두 대상에 대해 동일한 선택 구조를 만든다.
- 말풍선 안에서 `리터`, `밀리리터`가 강조되어 반복적으로 단위 개념을 상기시킨다.
- **렌더링/레이아웃 힌트**:
  - 좌측 주스병, 중앙 우유갑, 우측 말풍선+인물의 3분할 배치가 자연스럽다.
  - 확대 원은 물체의 용량 표기를 크게 보여주는 보조 요소로 따로 배치한다.
  - 점선 연결은 원형 확대 요소와 실제 표시 위치를 연결하는 방향으로 둔다.
  - 아래쪽 단위 선택 문장은 한 줄 텍스트로 길게 배치되며, 화살표 기호가 중간중간 삽입된다.

[DSL 작성 힌트]
- 문제 텍스트는 상단에 한 줄 또는 두 줄로 배치한다.
- 물체 그림은 왼쪽부터 주스병, 우유갑 순으로 두고, 각 아래에 물체 이름 라벨을 둔다.
- 각 물체의 용량 표시는 본체 내부 또는 상단 근처에 작게 배치한다.
- 용량 표기를 확대 원으로 한 번 더 보여주고, 점선 연결선을 추가하면 원본 구조와 유사해진다.
- 말풍선은 오른쪽 상단에 배치하고, 그 옆에 여자아이 캐릭터를 둔다.
- 말풍선 내부 텍스트는 두 줄 느낌으로 배치할 수 있으며, `리터`, `밀리리터`만 색 강조해도 된다.
- 하단의 `(정답)`, `(해설)`은 별도 텍스트 블록으로 분리해 배치한다.
- 단위 선택 문장은 `대상 → ( L, mL )` 형태의 텍스트 구조를 그대로 유지하는 것이 좋다.
- **보이는 사실**: `L`과 `mL`는 괄호 안에서 쉼표로 나란히 제시되어 있다.
- **레이아웃 힌트**: 단위 선택 문장은 다른 요소보다 약간 크게 보여서, 본문보다 눈에 띄는 위치에 놓는 것이 자연스럽다.

[semantic/solvable 힌트]
- 이 문항은 계산형이 아니라, 그림 속 용량 표기를 보고 단위를 대응시키는 문제이다.
- **보이는 사실**: 주스병에는 `1L`, 우유갑에는 `200mL`가 인쇄되어 있다.
- **추론되는 수학적 의미**: `L`은 리터, `mL`은 밀리리터이므로, 각 물체의 단위 인식을 묻는 형식이다.
- 단, DSL/정제 초안에서는 실제 정답을 새로 계산하거나 다시 서술하지 말고, 원본에 적힌 표시와 구조만 기록한다.
- 원본 하단에 정답/해설이 인쇄되어 있으나, 이를 문제 풀이 결과로 확장해 설명하지 않는다.
