# [전체적인 구성]
- 한 페이지짜리 초등 수학 객관식 문제로 보입니다.
- 상단에는 문제 번호 `50.`과 함께 서술형 지문이 왼쪽에서 오른쪽으로 이어집니다.
- 가운데에는 **두 개의 입체도형 그림**이 나란히 배치되어 있고, 각각 원 안에 `가`, `나`가 표시되어 있습니다.
- 오른쪽 아래 쪽에는 선택지/답 표기 영역처럼 보이는 `(` `가`, `나` `)` 형태의 문자열과 `그릇`이라는 단어가 있습니다.
- 하단 왼쪽에는 `(정답)` 표기와 함께 `나 그릇`이 보이고, 그 아래에는 `(해설)`로 시작하는 설명 문장이 있습니다.

# [보이는 텍스트]
보이는 텍스트를 있는 그대로 적으면 다음과 같습니다.

- `50.`
- `가 그릇과 나 그릇에 물을 가득 채운 후 모양과 크기가 같은 그릇에 옮겨 담았더니 그림과 같이 물이 채워졌습니다. 들어가 더 많은 그릇을 선택해 보세요.`
- 원 안 글자: `가`, `나`
- 오른쪽의 표기: `( 가, 나 ) 그릇`
- `(정답) 나 그릇`
- `(해설) 모양과 크기가 같은 그릇 담았을 때 물의 높이가 더 높은 나 그릇이 들어가 더 많은 그릇입니다.`

※ 마지막 해설 문장은 화면상 띄어쓰기와 줄바꿈이 다소 부자연스럽게 보이지만, 의미상 위처럼 읽힙니다.

# [반복되는 시각 패턴]
- **그릇 그림 2개가 반복 구조**로 배치되어 있습니다.
  - 왼쪽: `가`
  - 오른쪽: `나`
- 두 그림 모두 같은 형태의 투명한 직육면체 그릇으로 그려졌고, 물의 높이만 다르게 표현되어 있습니다.
- 물은 모두 파란색 계열의 반투명 면으로 채워져 있습니다.
- 각 그릇 위쪽에는 **원형 라벨**이 붙어 있으며, 원 안에 한 글자씩 들어갑니다.
- 공통적으로 **테두리선은 연한 파란색**, 내부에는 **점선/실선**이 섞여 입체감을 표현합니다.
- 선택지처럼 보이는 부분은 `( 가, 나 ) 그릇` 형태로, 괄호 안에 두 라벨이 함께 제시됩니다.

# [세부 내용]
- 왼쪽 그림 `가`
  - 투명한 정육면체/직육면체 모양의 그릇입니다.
  - 물은 그릇 높이의 중간보다 약간 아래까지 차 있는 것처럼 보입니다.
  - 물 표면은 수평선으로 표현되어 있고, 물 색은 옅은 파란색입니다.
  - 내부의 뒷면/아랫면을 나타내는 점선이 보입니다.
- 오른쪽 그림 `나`
  - 왼쪽과 같은 형태의 그릇입니다.
  - 물 높이가 `가`보다 더 높게 표현되어 있습니다.
  - 상단 쪽까지 물이 더 차 있는 모습이며, 물 표면선이 `가`보다 위에 있습니다.
- 두 그림 모두:
  - 윗면 테두리, 앞면 테두리, 옆면 테두리가 입체적으로 그려져 있습니다.
  - 같은 크기와 모양의 그릇임을 시각적으로 비교하도록 배치되었습니다.
- 문제 의도상:
  - 물 높이의 차이를 보고 더 많은 물이 들어간 그릇을 고르게 하는 비교형 문제로 보입니다.
  - 다만 이는 문제를 풀지 않는 범위에서의 **시각적 의도 추정**입니다.

# [Python DSL 작성 힌트]
- 좌측 상단에 문제 번호와 문장형 지문을 한 줄 또는 여러 줄 텍스트로 배치하면 좋습니다.
- 중앙에는 `가`, `나` 두 그림을 **가로로 나란히 정렬**하세요.
  - 각 그림은 동일한 크기의 투명 직육면체 도형으로 구성
  - 라벨 원은 그림의 좌상단 근처에 작게 배치
  - 물 표면 높이는 두 그림에서 다르게 설정
- 오른쪽 아래에는 `( 가, 나 ) 그릇` 형태의 텍스트를 별도 영역으로 두면 좋습니다.
- 하단에는 `(정답)`과 `(해설)` 텍스트 블록을 분리해서 배치하세요.
- 물 높이 비교가 핵심이므로, 두 그릇의:
  - 외곽선 두께
  - 물 채움 영역
  - 내부 점선
  를 구분해서 그리는 것이 중요합니다.

# [주의할 점]
- **정답을 추론해서 채우지 마세요.** 화면에는 이미 `(정답) 나 그릇`이 보이지만, 이 값은 시각 정보로만 기록하고 풀이 과정은 쓰지 않습니다.
- 문제를 수학적으로 해결하는 설명은 하지 마세요.
- `가`, `나`가 각각 어느 쪽에 있는지 바꾸지 말고, 화면 기준 위치를 유지하세요.
- 두 그릇의 모양과 크기는 같고, **물 높이만 다르다**는 점을 시각적으로 분명하게 적어야 합니다.
- 오른쪽의 `( 가, 나 ) 그릇` 표기는 선택지/응답 영역처럼 보이므로, 텍스트 위치를 별도 요소로 분리하는 것이 좋습니다.
- 해설 문장은 화면상 줄바꿈이 있어도 하나의 문장으로 읽힐 수 있으나, 원문 그대로 적기보다 **보이는 대로** 기록하는 편이 안전합니다.

# [불확실한 부분]
- 오른쪽 아래의 `( 가, 나 ) 그릇`이 정확히 선택지인지, 아니면 답안 작성용 표기인지 화면만으로는 완전히 단정하기 어렵습니다.
- 해설 문장의 마지막 부분은 인쇄 상태와 줄바꿈 때문에 일부 글자 경계가 다소 흐려 보입니다.
- 두 그릇 내부의 물 표면 위치를 픽셀 단위로 정확히 측정하기는 어렵고, 현재는 상대적 높이 차이만 명확히 보입니다.
