[문제 개요]
- 문제 번호는 `8.`로 보이며, 지시문은 `〈보기〉에서 물건을 선택하여 문장을 완성해 보세요.`로 읽힌다.
- 중심 구성은 큰 `〈보기〉` 박스 1개와, 그 아래의 문장형 빈칸 문제 1개, 그리고 해설 문장 1개로 보인다.
- 원본 이미지 기준으로, 이 문제는 실생활 물건의 들이(용량)를 단위 `mL`와 연결하는 선택형 문장 완성 문제 형태다.

[원본에 보이는 요소]
- 상단 왼쪽:
  - 작은 네모 표시처럼 보이는 기호
  - 문제 번호 `8.`
  - 지시문 `〈보기〉에서 물건을 선택하여 문장을 완성해 보세요.`
- 중앙:
  - 굵은 검은 테두리의 큰 사각형 박스
  - 박스 상단 안쪽에 `〈보기〉`
  - 박스 내부에 물건 그림 3개가 가로로 배치됨
  - 각 그림 아래에 이름 라벨이 있음
- 보기 박스 안의 라벨:
  - `우유갑`
  - `주스병`
  - `드럼통`
- 보기 박스 아래:
  - `(1) ( 우유갑, 주스병, 드럼통 ) 의 들이는 약 250mL입니다.`
- 해설 문장:
  - `(해설)큰 그릇의 들이는 L를, 작은 그릇의 들이는 mL를 사용하여 나타내는 것이 좋습니다.`
- 화면 하단:
  - `)(1) 우유갑`처럼 보이는 잘린 문자열이 일부 보임
  - 전체 문장은 화면 밖으로 잘려 정확히 확인되지 않음

[수식/관계 구조]
- 등호 `=`가 있는 계산식은 보이지 않는다.
- 별도의 수식 변형 연쇄도 보이지 않는다.
- 문장 내부의 핵심 관계는 다음처럼 읽힌다.
  - `어떤 물건의 들이는 약 250mL이다`
  - 보기 후보 3개 중 하나를 선택하여 문장을 완성하는 구조
- 단위 정보:
  - `250mL`
  - 해설에서 `L`, `mL`가 함께 언급됨

[등호별 변환 의미]
- 해당 없음.
- 원본에는 `=`를 포함한 단계식이나 변형 과정이 보이지 않는다.

[빈칸 정책]
- 빈칸이나 선택 결과는 추론해서 채우지 않는다.
- `( 우유갑, 주스병, 드럼통 )`은 선택지로 보이는 원문 그대로 보존한다.
- 하단의 잘린 `)(1) 우유갑`은 전체 문맥이 불명확하므로 빈칸 채움이나 정답 표시로 단정하지 않는다.
- 원본에 정답이 명시적으로 인쇄되어 보이지 않는 한, 정답을 쓰지 않는다.

[반복 패턴]
- `〈보기〉` 박스 내부에서 다음 구조가 반복적으로 보인다.
  - 그림 1개
  - 그림 아래 라벨 1개
- 세 물건은 모두 용기/통 종류의 그림으로 보이며, 좌우로 균등하게 배치되어 있다.
- 왼쪽부터:
  - 작은 우유팩 형태
  - 병 형태
  - 큰 드럼통 형태
- 각 라벨은 해당 그림의 바로 아래 중앙 부근에 놓여 있다.
- 박스는 사각 테두리로 둘러싸여 있고, 내부는 위아래 여백이 있는 편이다.
- 하단 문장은 `(1)`로 시작하는 문항 형식을 유지하고 있으며, 괄호 안에 3개 선택지가 나열되어 있다.

[DSL 작성 힌트]
- 상단 지시문은 한 줄 텍스트 요소로 배치한다.
- `〈보기〉`는 굵은 사각형 테두리 박스로 표현한다.
- 박스 내부 상단 쪽에 `〈보기〉` 텍스트를 두고, 그 아래 가로 방향으로 3개의 그림을 배치한다.
- 각 그림 아래에는 `우유갑`, `주스병`, `드럼통` 라벨을 각각 중앙 정렬로 둔다.
- 보기 박스와 하단 문장 사이에는 충분한 간격을 둔다.
- 하단 문장은 `(1) ( 우유갑, 주스병, 드럼통 ) 의 들이는 약 250mL입니다.`처럼 보이는 한 줄 문장으로 구성한다.
- 해설 문장은 본문보다 작은 글씨로 아래에 배치한다.
- 화면 하단의 잘린 문자열은 별도 확정 요소로 두지 말고, 보이는 범위까지만 반영한다.
- 레이아웃상 핵심은 `문제 지시문 → 보기 박스 → 문장형 문항 → 해설`의 순서다.

[semantic/solvable 힌트]
- 이 문제는 실생활에서 용량의 크기를 감각적으로 비교하게 하는 선택형 문장 완성 문제로 해석된다.
- `250mL`는 작은 용량을 나타내는 단위로 보이며, 해설은 `작은 그릇의 들이는 mL`를 사용하라는 일반 원리를 안내한다.
- 다만, 어떤 선택지가 정답인지 여기서 판단하지 않는다.
- 문장 의미상 `들이다`는 용량, 부피에 해당하는 맥락으로 읽힌다.
