# [전체적인 구성]
- 화면 상단에는 문제 번호가 붙은 서술형/선다형 문제 문장이 2줄 정도로 배치되어 있습니다.
- 가운데에는 연두색 계열의 **표 1개**가 있고, 표 위에는 가운데 정렬된 제목이 있습니다.
- 표 아래에는 **선택지 4개**가 2열 형태로 배치되어 있으며, 각 선택지 앞에 파란색 원 안 숫자(①~④)가 붙어 있습니다.
- 그 아래 왼쪽에는 **(정답) ③**이 따로 표시되어 있고, 맨 아래에는 **(해설)** 문장이 1줄로 있습니다.
- 전체적으로 “문제문 → 표 → 선택지 → 정답 표시 → 해설”의 순서로 구성되어 있습니다.

# [보이는 텍스트]
- `□ 9.`  
- `운영이는 반 학생들이 운동회에서 하고 싶어 하는 경기를 조사하여 표로 나타내었습니다. 하고 싶어 하는 여학생 수와 남학생 수가 같은 경기는 무엇인가요?`
- 표 제목: `운동회에서 하고 싶어 하는 경기`
- 표의 열 제목:
  - `경기`
  - `달리기`
  - `공굴리기`
  - `줄다리기`
  - `○× 퀴즈`
  - `합계`
- 표의 행 제목:
  - `여학생 수(명)`
  - `남학생 수(명)`
- 표 안 숫자:
  - 여학생 수(명): `1, 2, 3, 6, 12`
  - 남학생 수(명): `4, 8, 3, 2, 17`
- 선택지:
  - `① 달리기`
  - `② 공굴리기`
  - `③ 줄다리기`
  - `④ ○× 퀴즈`
- 아래쪽 표기:
  - `(정답) ③`
  - `(해설) 줄다리기를 하고 싶어 하는 여학생 수와 남학생 수가 3명으로 같습니다.`

# [반복되는 시각 패턴]
- 표는 **가로 방향의 머리글 행 1개 + 데이터 행 2개**로 이루어진 3행 구조입니다.
- 왼쪽 첫 열은 항목명(행 제목)이고, 나머지 열은 경기 종류별 수치를 담는 구조입니다.
- 각 경기 항목은 같은 열 아래에 여학생 수와 남학생 수가 짝을 이루어 배치되어 있어, **각 열을 세로로 비교**하게 만드는 형식입니다.
- 선택지는 2열 배치로 나뉘어 있으며, 왼쪽에 ①, ③ / 오른쪽에 ②, ④가 배치되어 있습니다.
- 선택지 번호는 파란색 원형 마크로 강조되어 있고, 텍스트는 그 오른쪽에 놓여 있습니다.
- 표의 셀 테두리는 연두색 얇은 선으로 일정하게 구획되어 있습니다.
- 제목 행과 왼쪽 행 제목 칸은 연한 연두색 배경으로 강조되어 있습니다.

# [세부 내용]
- 문제는 “여학생 수와 남학생 수가 같은 경기”를 찾는 내용입니다.
- 표에서 각 경기별로 여학생 수와 남학생 수가 한 쌍으로 제시되어 있습니다.
- 보이는 값만 기준으로 보면:
  - `달리기` 열: 여학생 수 `1`, 남학생 수 `4`
  - `공굴리기` 열: 여학생 수 `2`, 남학생 수 `8`
  - `줄다리기` 열: 여학생 수 `3`, 남학생 수 `3`
  - `○× 퀴즈` 열: 여학생 수 `6`, 남학생 수 `2`
  - `합계` 열: 여학생 수 `12`, 남학생 수 `17`
- 여기서 어떤 열이 조건에 맞는지는 화면에 해설과 정답 표시가 있지만, 초안에서는 그 결과를 재진술만 하고 이유를 새로 추론해 확정하지 않는 것이 좋습니다.
- 표의 마지막 열 `합계`는 각 행의 총합을 나타내는 열로 보입니다. 다만 이 문제의 핵심 비교 대상은 각 경기 열입니다.
- 문제 문장에는 “운동회에서 하고 싶어 하는 경기를 조사하여 표로 나타내었습니다”라는 설명이 있어, 표가 설문 결과임을 알 수 있습니다.

# [Python DSL 작성 힌트]
- 표를 그릴 때는 **상단 제목 1개 + 5열짜리 표 + 2행 데이터** 구조로 잡으면 됩니다.
- 표의 첫 행은 열 제목, 첫 열은 행 제목이므로 **좌상단 셀은 ‘경기’**로 넣는 형태가 자연스럽습니다.
- 각 셀의 숫자는 가운데 정렬에 가깝게 배치되어 보입니다.
- 표 아래 선택지는 **2열 2행 배치**로, 왼쪽 열에 ①/③, 오른쪽 열에 ②/④를 두는 구성이 맞습니다.
- 정답 표기와 해설은 본문과 분리된 하단 텍스트로 처리하면 됩니다.
- 파란색 원형 번호는 텍스트 앞의 장식 요소로 취급할 수 있습니다.
- 제목은 표의 바로 위, 가운데 정렬로 배치하는 것이 좋습니다.
- 연두색 배경은 머리글 영역과 왼쪽 행 제목 영역에 적용하는 식으로 구현하면 시각적으로 유사합니다.

# [주의할 점]

- “같은 경기”라는 조건을 만족하는 항목을 분석 도중 추론해 답으로 확정하는 서술은 피하세요.

- `○× 퀴즈`는 기호가 포함된 항목명이므로 문자 그대로 유지해야 합니다.
- `□ 9.`의 작은 네모는 문제 번호 앞 장식으로 보이므로 누락하지 않도록 합니다.
- `(정답) ③`, `(해설)`은 본문 구성 요소이지만, Python DSL에서 자동 생성할 때는 필요 여부를 별도로 판단해야 합니다.
