[문제 개요]
- 객관식 선택 문제이다.
- 주제는 “보고 싶은 문화재” 조사 결과를 보고, 여학생 수와 남학생 수를 알아보는 데 더 편리한 자료가 무엇인지 고르는 내용이다.
- 화면에는 조사 자료 그림, 범례, 문제 지문, 보기, 정답 표기, 해설 문장이 함께 보인다.
- 정답 추론은 하지 않으며, 다만 원본 화면에 `(정답)표`가 인쇄되어 보인다.

[원본에 보이는 요소]
- 상단 안내 문장: `경수네 학교 3학년 학생들이 보고 싶은 문화재를 조사하였습니다. 물음에 답하세요.`
- 큰 제목: `보고 싶은 문화재`
- 2×2 형태의 문화재 조사 그림이 중앙에 있다.
- 네 개의 칸 이름:
  - `첨성대`
  - `다보탑`
  - `숭례문`
  - `종묘`
- 색 점 범례:
  - 분홍색 점 = `여학생`
  - 하늘색 점 = `남학생`
- 문제 번호와 지문:
  - `58. 보고 싶은 문화재별 여학생 수와 남학생 수를 알아보려고 할 때 조사한 자료와 표 중 더 편리한 것을 선택하세요.`
- 보기:
  - `( 조사한 자료 , 표 )`
- 하단 텍스트:
  - `(정답)표`
  - `(해설)표를 보면 보고 싶은 문화재별 여학생 수와 남학생 수를 한눈에 알 수 있습니다.`

[수식/관계 구조]
- 수식, 등호, 계산식은 보이지 않는다.
- 이 문항은 수치 계산 문제가 아니라, 두 표현 방식 중 더 편리한 것을 고르는 선택형 구조이다.
- 보이는 관계는 다음과 같다.
  - 조사한 자료 그림 안의 색 점들 → 여학생/남학생을 나타냄
  - 이 자료를 바탕으로 문화재별 인원 수를 알아보려는 상황
  - 보기의 두 항목: `조사한 자료`, `표`

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 식 변형은 보이지 않는다.

[빈칸 정책]
- 빈칸, 미지수, 누락된 수를 추론해서 채우지 않는다.
- 이 문제의 핵심은 자료 해석과 보기 선택이므로, 점의 개수나 각 문화재별 인원 수를 임의로 세어 확정하지 않는다.
- 원본에 인쇄된 정답 표기 `(정답)표`는 관찰 사실로만 남기고, 이를 근거로 풀이를 전개하지 않는다.

[반복 패턴]
- 2×2 격자의 네 칸이 같은 스타일로 반복된다.
- 각 칸은 비슷한 크기의 사각형 프레임을 가진다.
- 각 칸 좌상단에 둥근 직사각형 라벨이 반복 배치된다.
- 점들은 같은 크기의 원형으로 반복되며, 색만 분홍/하늘색으로 다르다.
- 범례도 `색 점 + 설명 텍스트`의 반복 구조를 가진다.
- 큰 민트색 패널 바깥 테두리와 안쪽 점선 테두리가 함께 보여 중첩된 패널 구조를 이룬다.

[DSL 작성 힌트]
- 상위 컨테이너 안에 다음 요소를 순서대로 배치하는 구성이 적절하다.
  1. 상단 안내 문장
  2. 제목 `보고 싶은 문화재`
  3. 2×2 조사 자료 패널
  4. 범례
  5. 문제 지문
  6. 보기
  7. 정답/해설 텍스트
- 조사 자료 패널은 큰 민트색 배경 사각형 안에 4개의 동일한 소패널을 2행 2열로 배치한 구조로 잡는다.
- 각 소패널의 좌상단에는 문화재 이름 라벨이 들어가야 한다.
- 소패널 내부의 점들은 개별 원형 요소의 반복으로 표현하되, 정확한 개수는 무리하게 추정하지 말고 군집 구조를 우선 반영한다.
- 범례는 아래쪽에 분홍 점과 하늘색 점을 각각 하나씩 두고, 오른쪽에 `여학생`, `남학생` 텍스트를 붙인다.
- 문제 지문은 일반 텍스트 블록으로 처리하고, 보기 `( 조사한 자료 , 표 )`는 선택지 형태로 따로 분리하는 것이 좋다.
- 하단의 `(정답)표`, `(해설)표를 보면 ...`는 별도 설명 블록으로 분리하면 레이아웃 재현에 유리하다.
- 정렬 기준은 가운데 정렬이 중심이며, 문화재 4칸은 행/열 간격이 균등해야 한다.
- 라벨과 점, 범례 사이에는 시각적 여백을 충분히 두어야 한다.

[semantic/solvable 힌트]
- 이 문항은 “조사한 자료”와 “표” 중에서 더 편리한 것을 고르는 이해형 문제이다.
- 관찰되는 해설 문장은 `표를 보면 보고 싶은 문화재별 여학생 수와 남학생 수를 한눈에 알 수 있습니다.`이다.
- 따라서 화면 구성상 표의 장점이 강조되지만, 풀이 자체를 새로 계산할 필요는 없다.
- 문화재별로 여학생과 남학생의 구분은 색 점 범례에 의해 해석된다.
- 문제의 핵심 의미는 “많은 점이 흩어진 그림보다, 항목별로 정리된 표가 비교에 더 편리하다”는 자료 해석이다.

[주의 및 TODO]
- 점의 정확한 개수는 이미지 해상도상 명확히 세지 않아서 TODO로 둔다.
- 각 칸 내부의 점 배치는 매우 조밀하므로, DSL에서는 세부 좌표를 완벽히 복원하기보다 점 군집의 밀도와 색 구분을 재현하는 쪽이 안전하다.
- `조사한 자료`와 `표`의 보기 위치는 괄호 형태로 보이지만, 실제 클릭형 배치인지 단순 텍스트인지 레이아웃상 확인이 필요하다.
- 하단 정답/해설 영역은 본문과 분리된 추가 설명처럼 보이지만, 원본에서의 정확한 경계는 약간 모호하다.
- 제목 `보고 싶은 문화재`는 조사 패널의 내부 제목이므로, 문제 지문과 같은 수준으로 취급하지 않도록 주의한다.
