[문제 개요]
- 문항 번호는 `94.`로 보인다.
- 지시문은 `직각삼각형을 모두 선택하세요.`로 보인다.
- 화면에는 격자 배경 위에 삼각형 6개가 배치되어 있고, 각 도형 안에 한글 라벨 `가, 나, 다, 라, 마, 바`가 들어 있다.
- 중간 왼쪽에 `(정답) 나, 바`가 인쇄되어 보인다.
- 그 아래에 `(해설) 한 각이 직각인 삼각형을 직각삼각형이라고 합니다.`라는 설명 문장이 있다.
- 하단에는 같은 도형 배치를 다시 보여 주는 해설용 그림이 있다.

[원본에 보이는 요소]
- 상단 왼쪽: `94.`와 문제 문장.
- 상단 오른쪽 및 가운데: 연한 파란색 격자 위 삼각형 6개.
- 각 삼각형 내부 라벨:
  - `가`
  - `나`
  - `다`
  - `라`
  - `마`
  - `바`
- 문제 그림 아래 왼쪽: `(정답) 나, 바`
- 그 아래: `(해설)한 각이 직각인 삼각형을 직각삼각형이라고 합니다.`
- 하단: 격자 배경 위에 삼각형 6개를 다시 배치한 해설 그림
- 해설 그림의 `바` 근처에는 작은 네모 모양의 직각 표시가 추가되어 보인다.

[수식/관계 구조]
- 등호나 식 변형은 보이지 않는다.
- 수학적 관계는 “직각이 있는 삼각형 = 직각삼각형”이라는 정의 문장 형태로만 제시된다.
- 문제의 선택 대상은 6개의 삼각형 중 직각삼각형을 고르는 구조로 보인다.

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄, 식 변형, 분수 변환은 보이지 않는다.

[빈칸 정책]
- 빈칸 문제는 아니다.
- 원본에 보이는 정답 표기 `(정답) 나, 바`는 그대로 관찰 사실로만 기록한다.
- 보이지 않는 값이나 추론한 선택 결과를 새로 채워 넣지 않는다.
- 정답 표기가 있더라도, 별도의 계산으로 답을 다시 도출한 것처럼 서술하지 않는다.

[반복 패턴]
- 격자 배경이 상단 문제 그림과 하단 해설 그림에서 반복된다.
- 삼각형 6개가 각각 독립된 도형으로 반복 배치된다.
- 각 도형 내부에 한 글자 라벨이 들어가는 패턴이 반복된다.
- 하단 해설 그림은 상단 문제 그림과 거의 같은 구성을 다시 보여 주되, `바` 도형에 직각 표시가 추가된 형태로 보인다.
- 검은 외곽선 도형 + 연한 파란 격자 배경 + 내부 문자 배치가 반복되는 레이아웃이다.

[DSL 작성 힌트]
- 상단 문제 영역과 하단 해설 영역을 분리된 블록으로 두는 구성이 적합하다.
- 문제 문장, 정답 표기, 해설 문장은 각각 별도 텍스트 요소로 배치하는 것이 좋다.
- 삼각형 6개는 각각 독립된 `polygon` 계열 도형으로 취급하면 된다.
- 각 삼각형 안의 `가, 나, 다, 라, 마, 바`는 도형 중심 근처에 텍스트로 넣는다.
- 격자는 연한 파란색의 규칙적인 배경선으로 반복 생성하는 방식이 자연스럽다.
- 해설 그림에서는 상단 배치와 유사한 도형을 다시 그리되, `바` 쪽에 작은 직각 표시용 네모를 덧붙이는 구성이 어울린다.
- 정답 표기 `(정답) 나, 바`는 본문과 분리하여 작은 텍스트 블록으로 두는 것이 좋다.
- 도형의 정확한 좌표보다 상대적 위치와 방향을 우선 반영하는 편이 좋다.

[semantic/solvable 힌트]
- 이 문제의 의미는 “여러 삼각형 중 직각이 있는 것을 찾기”이다.
- 해설 문장은 직각삼각형의 정의를 알려 주는 역할을 한다.
- 하단 그림에서 `바`에 직각 표시가 보이므로, 해설은 직각 여부를 시각적으로 강조하는 구조로 이해할 수 있다.
- 다만 이 문서는 정답을 새로 추론하는 용도가 아니라, 원본에 이미 보이는 구조를 정리하는 용도이다.
