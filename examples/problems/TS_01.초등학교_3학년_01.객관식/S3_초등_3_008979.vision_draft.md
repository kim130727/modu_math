# [전체적인 구성]
- 화면은 **초등 수학 객관식/선택형 문제 1개**로 보입니다.
- 상단 왼쪽에 **문항 번호 `14.`**가 있고, 그 오른쪽에 문제 지시문이 한 줄로 배치되어 있습니다.
- 가운데에는 **각의 그림**이 크게 배치되어 있으며, 아래쪽에는 **선택 표시(○/×)** 와 **정답/해설 문구**가 있습니다.
- 전체적으로 **문제 문장 → 도형(각) → 선택 표시 → 정답 표시 → 해설** 순으로 내려가는 구조입니다.

# [보이는 텍스트]
- 상단 문제 문장:
  - `14. 직각이면 ○, 직각이 아니면 ×를 선택하세요.`
- 그림 아래 왼쪽에 작은 기호:
  - 파란색으로 보이는 `○` 모양의 선택 표시 1개
- 그림 아래 쪽에 작은 기호:
  - 파란색으로 보이는 `×` 모양의 선택 표시 1개
- 그 아래:
  - `(정답) ×`
- 해설 시작:
  - `(해설) 직각 삼각자의 직각 부분을 대어 보았을 때 꼭 맞게 겹쳐지는 각이 직각입니다.`
- 오른쪽/가운데 그림은 글자가 아니라 **각을 나타내는 선분 그림**입니다.

# [반복되는 시각 패턴]
- **선택형 문제**에서 흔히 보이는 패턴이 있습니다.
  - 문제 문장에 `○, ×`가 함께 제시됨
  - 아래에 선택 가능한 기호가 따로 배치됨
  - 정답이 별도로 표기됨
- 그림은 **두 개의 선분이 한 꼭짓점에서 만나는 각**의 형태로 보입니다.
- 한 선분은 **거의 수평으로 오른쪽**으로 뻗고, 다른 선분은 **왼쪽 아래 쪽 꼭짓점에서 오른쪽 위로 기울어진 형태**입니다.
- 각 그림 아래쪽에 선택 기호가 있어, **그림을 보고 직각 여부를 판단하는 유형**으로 보입니다.

# [수식 단계]
- 해당 없음.

# [등호별 변환 의미]
- 해당 없음.

# [세부 내용]
- 상단 문항 번호 `14.`는 왼쪽 정렬로 보입니다.
- 문제 문장은 문항 번호 뒤에 이어서 배치되어 있고, `○`는 분홍색 원, `×`는 분홍색 또는 진한 분홍색의 X 기호로 보입니다.
- 가운데 도형은 다음처럼 보입니다.
  - 한 점에서 두 선이 만남
  - 아래쪽 선은 수평에 가깝게 오른쪽으로 뻗음
  - 위쪽 선은 그 점에서 우상향 방향으로 뻗음
- 도형 아래쪽에는 파란색 작은 표시가 하나 있으며, 그 아래/근처에 `×` 표시가 있어 **선택 결과 또는 예시 정답 위치**처럼 보입니다.
- 왼쪽 아래쪽에는 별도의 파란색 `○` 표시가 있어 **선택 보기처럼 나열된 요소**로 보입니다.
- 하단에는 `(정답) ×`가 있고, 그 아래 `(해설)` 문장이 이어집니다.
- 해설 문장은 문장형 텍스트로 길게 적혀 있으며, **직각의 판단 기준**을 설명하는 내용입니다.

# [Python DSL 작성 힌트]
- 문제 유형은 **도형 판별형 1문항**으로 처리하는 것이 적절합니다.
- DSL에서는 다음 요소를 분리해 표현하면 좋습니다.
  1. 상단 문제 지시문 텍스트
  2. 가운데 각 도형
  3. 선택 기호 `○`, `×`
  4. 정답/해설 영역
- 도형은:
  - 꼭짓점 1개
  - 꼭짓점에서 뻗는 2개의 선분
  - 하나는 수평에 가까운 오른쪽 선분
  - 하나는 우상향 선분
  로 구성된 것으로 기술하면 좋습니다.
- 선택 표시의 위치는 **도형 아래 왼쪽/가운데**처럼 상대 배치를 함께 기록하면 좋습니다.
- 정답과 해설은 별도의 텍스트 블록으로 분리하는 편이 좋습니다.

# [주의할 점]
- **직각인지 아닌지의 최종 판단을 이 초안에서 단정해 설명하면 안 됩니다.**
- `○`와 `×`는 문제에서 제시한 선택 기호이므로, **해석해서 답으로 확정하지 말고** 화면에 보이는 그대로 적어야 합니다.
- 도형의 정확한 각도는 사진상 약간 기울어져 보일 수 있으므로, **“거의 수평”, “우상향”**처럼 보이는 방향 중심으로 기술하는 것이 안전합니다.
- 해설 문장은 일부가 줄바꿈되어 보일 수 있으므로, **문장 연결을 임의로 바꾸지 않도록** 주의해야 합니다.
- 그림 속 파란색 표시가 선택지인지 정답 표시인지, 혹은 편집 표시인지 **화면만으로 100% 단정하기 어려울 수 있습니다.**

# [불확실한 부분]
- 가운데 도형의 선분 길이와 정확한 각도는 이미지 해상도상 약간 불명확합니다.
- 파란색 `○`, `×` 표시는 실제 선택지인지, 정답 표시인지, 편집용 주석인지 화면만으로는 완전히 확정하기 어렵습니다.
- 해설 문장 중 일부는 줄바꿈 때문에 단어 경계가 시각적으로 조금 끊겨 보일 수 있습니다.
