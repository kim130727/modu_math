# [전체적인 구성]
- 한 페이지형 객관식 문제 화면이다.
- 상단에 문제 번호와 질문 문장이 있고, 그 아래에 **표**가 중심 시각 자료로 배치되어 있다.
- 표 아래쪽에는 보기 ①~④가 가로로 나열되어 있고, 왼쪽 아래에 **(정답)②**와 **(해설)** 문장이 추가로 보인다.
- 전체적으로 “문제 문장 → 표 → 선택지 → 정답 표시 → 해설” 순서의 구성이다.

# [보이는 텍스트]
- `□ 93. 여수네 반 학생들이 학예회에서 하고 싶어 하는 것을 조사하여 표로 나타내었습니다. 여학생이 가장 많이 하고 싶어 하는 종목은 무엇인가요?`
- 표 제목: `여학생과 남학생이 학예회에서 하고 싶어 하는 종목`
- 표의 열 제목:
  - `종목`
  - `연극`
  - `합주`
  - `무용`
  - `합창`
  - `합계`
- 표의 행 제목:
  - `여학생 수(명)`
  - `남학생 수(명)`
- 표 안의 수치:
  - 여학생 수(명): `2, 7, 3, 1, 13`
  - 남학생 수(명): `4, 3, 2, 6, 15`
- 선택지:
  - `① 연극`
  - `② 합주`
  - `③ 무용`
  - `④ 합창`
- 추가 표기:
  - `(정답)②`
  - `(해설)2, 7, 3, 1 중 가장 큰 수는 7이므로 여학생이 가장 많이 하고 싶어 하는 종목은 합주입니다.`

# [반복되는 시각 패턴]
- 표는 **가로 방향 5개 열**로 반복되는 구조를 가진다.
- 첫 번째 열은 항목명(행 제목)이고, 나머지 4열은 종목 이름이 반복 배치되어 있다.
- 숫자 정보는 두 행에 걸쳐 대응되며, 각 종목 아래에 여학생 수와 남학생 수가 각각 한 칸씩 들어간다.
- 선택지는 `①~④` 형식으로 번호가 붙은 텍스트 보기로 반복된다.
- 문제 화면 하단에 정답 표시와 해설이 별도로 배치되어, 문제와 설명이 시각적으로 구분된다.

# [수식 단계]
- 이 문제 화면에는 등호를 사용한 수식 변형 단계가 보이지 않는다.
- 표의 수치 비교와 해설 문장만 있으며, 계산식을 연속으로 정리한 형태는 아니다.

# [등호별 변환 의미]
- 해당 없음.
- 화면에 `=` 형태의 변환식이 나타나지 않는다.

# [세부 내용]
- 문제 번호 `93.`은 왼쪽 상단에 작게 표시되어 있다.
- 질문 문장은 문제 번호 오른쪽에서 시작해 두 줄로 이어진다.
- 가운데의 표는 연한 색 테두리의 표 형식이며, 상단에는 제목이 표 위쪽 중앙에 놓여 있다.
- 표 내부는:
  - 맨 왼쪽 열에 행 이름이 들어가고,
  - 상단 행에는 종목명이 좌에서 우로 배열되며,
  - 마지막 열은 `합계`로 되어 있다.
- 여학생 수는 종목별로 `2, 7, 3, 1`이 보이고, 합계 칸에 `13`이 있다.
- 남학생 수는 종목별로 `4, 3, 2, 6`이 보이고, 합계 칸에 `15`가 있다.
- 표 아래에는 보기 4개가 일정 간격으로 가로 배치되어 있으며, 각 보기 앞의 동그란 숫자만 파란색으로 강조되어 보인다.
- 정답 표시는 `(정답)②`처럼 괄호와 함께 따로 제시되어 있다.
- 해설 문장은 하단 왼쪽에 길게 한 줄로 이어지며, 숫자 비교에 관한 설명이 포함되어 있다.

# [Python DSL 작성 힌트]
- 문제 재현 시 핵심 요소는 다음 순서로 잡으면 좋다.
  1. 상단 문제 문장
  2. 중앙 표
  3. 표 아래 선택지 4개
  4. 하단 정답/해설 텍스트
- 표는 2행 5열 구조로 보이지만, 실제로는 왼쪽 행 제목 열까지 포함해 **총 5개 열**로 읽으면 된다.
- 각 셀의 텍스트는 중앙 정렬에 가깝게 배치되어 보인다.
- 제목은 표 위 중앙에 별도 텍스트로 두고, 표와 약간 떨어진 위치에 배치하는 방식이 적절하다.
- 선택지는 가로로 배치하고, 번호와 항목 사이에 공백을 두면 원문 인상을 살리기 쉽다.
- 정답 및 해설은 문제 본문과 구분되는 별도 텍스트 블록으로 처리하는 것이 좋다.

# [주의할 점]
- 이 자료는 **문제를 푸는 답안 작성용이 아니라 시각 재현용**으로 봐야 한다.
- `여학생이 가장 많이 하고 싶어 하는 종목`이라는 질문의 결과를 직접 계산해 서술하지 말아야 한다.
- 표의 수치를 임의로 수정하거나 빈칸을 채우지 말아야 한다.
- 해설에 적힌 내용이 보이더라도, DSL에서는 이를 답으로 단정해 자동 채우지 않도록 주의해야 한다.
- 표 제목, 행 제목, 선택지 번호 형식 등을 놓치지 말고 그대로 반영하는 것이 중요하다.

# [불확실한 부분]
- 표의 정확한 선 두께와 색상은 이미지 해상도상 단정하기 어렵다.
- 글꼴의 정확한 종류와 크기는 확인되지 않지만, 전반적으로 교과서/문제지 스타일의 기본 글꼴로 보인다.
- 문제 번호 왼쪽의 작은 사각형 기호 `□`는 장식 요소인지 체크박스인지 확실하지 않다.
- 표 내부 정렬이 완전한 중앙인지, 약간 왼쪽 정렬인지 이미지만으로는 세밀하게 단정하기 어렵다.
