[문제 개요]
- 문제 번호는 `27.`로 보인다.
- 상단에는 한 줄짜리 한국어 지시문이 있다.
- 중앙에는 파란 점선 격자 위에 여러 개의 원이 겹쳐진 도형이 있다.
- 도형 주변과 내부에 원형 표식 안의 한글 자음 선택지가 배치되어 있다.
- 하단에는 `(정답)` 표시와 동그라미로 둘러싸인 기호 1개, 그리고 `(해설)` 문장이 보인다.

[원본에 보이는 요소]
- 상단 문제 문장:
  - `세인이가 그림과 같은 모양을 그렸습니다. 이 모양을 그릴 때 컴퍼스의 침을 꽂지 않은 곳을 찾아 기호를 선택하세요.`
- 도형 주변의 선택지 기호:
  - `ㄱ`
  - `ㄴ`
  - `ㄷ`
  - `ㄹ`
  - `ㅁ`
  - `ㅂ`
- 하단 텍스트:
  - `(정답)` 뒤에 동그라미 안의 기호 1개가 인쇄되어 있음
  - `(해설) 점 ㄱ을 원의 중심으로 하여 그린 원은 없습니다.`
- 도형 내부:
  - 파란 점선 격자 배경
  - 얇은 회색/검은색 선의 큰 원 여러 개가 겹쳐 있음
  - 분홍색 점 몇 개가 원의 중심 또는 강조점처럼 표시됨
- 원형 선택지 배지:
  - 흰색 원 테두리 안에 검은 자음이 들어간 형태로 보임

[수식/관계 구조]
- 수식이나 등호로 이어진 계산식은 보이지 않는다.
- 대신 그림 구조가 “원 여러 개의 겹침 + 각 위치가 중심인지 여부 판단” 관계로 구성되어 있다.
- 선택지는 기호별 위치를 가리키며, 문제의 핵심 관계는 다음처럼 이해된다.
  - 어떤 기호 위치가 컴퍼스의 침을 꽂은 중심점인지 여부를 판별하는 형식
  - 중심으로 사용된 점은 원의 윤곽과 대응되는 기준점으로 해석될 수 있음
- 보이는 해설 문장에는 `ㄱ`이 특정 중심점으로 언급되지만, 이는 텍스트 관찰 사실로만 기록한다.

[등호별 변환 의미]
- 해당 없음.
- 등호가 포함된 변환 연쇄나 식 전개는 보이지 않는다.

[빈칸 정책]
- 빈칸에 해당하는 답은 추론해서 채우지 않는다.
- `(정답)` 옆 동그라미 안 기호가 무엇인지 해상도상 보이더라도, 정답을 확정해 서술하지 않는다.
- 보이는 경우에만 “인쇄되어 보인다” 수준으로만 적고, 빈칸을 메우는 방식의 서술은 하지 않는다.

[반복 패턴]
- 파란색 점선 격자가 정사각형 칸 간격으로 반복된다.
- 격자 위에는 동일한 스타일의 원 윤곽이 여러 개 겹쳐 있다.
- 선택지 배지는 모두 동일한 크기와 형태의 원형 기호로 반복된다.
- 분홍색 점도 여러 위치에 반복 배치되어 있어 중심/기준점 표시처럼 보인다.
- 중앙 도형은 겹치는 원들의 반복 구조로 이루어져 있어, 원 중심 간 배치가 패턴처럼 보인다.

[DSL 작성 힌트]
- 화면은 크게 3영역으로 나누어 표현하면 좋다:
  1. 상단 문제 문장
  2. 중앙 도형 영역
  3. 하단 정답/해설 영역
- 중앙 도형 영역은 레이어를 분리하는 것이 좋다:
  - 파란 점선 격자
  - 여러 개의 원 윤곽
  - 분홍색 강조점
  - 선택지 원형 배지와 자음 텍스트
- 격자는 일정 간격의 수평/수직 점선 반복으로 표현한다.
- 원들은 서로 겹치므로, 각 원의 중심과 반지름을 상대 배치로 잡는 방식이 적합하다.
- 선택지 배지는 작은 흰 원 + 테두리 + 자음 텍스트의 조합으로 표현한다.
- 하단 `(정답)`과 `(해설)`은 별도 텍스트 블록으로 두고, 해설 문장은 원문 그대로 배치하는 방식이 자연스럽다.
- 선택지 기호는 좌표상 원 내부/주변에 분산 배치되어 있으므로, 도형과 겹치지 않도록 적절한 간격을 두는 것이 좋다.

[semantic/solvable 힌트]
- 문제는 “컴퍼스의 침을 꽂은 곳”, 즉 원을 그릴 때 중심이 된 위치를 찾는 도형 판별 문제로 해석된다.
- 보이는 해설 문장만 보면 `ㄱ`과 `원의 중심`이 함께 언급된다.
- 다만 해설의 내용이 곧바로 정답 확정 근거가 되더라도, 여기서는 풀이를 확정하지 않고 관찰 사실로만 유지한다.
- 도형에는 중심처럼 보이는 분홍색 점이 여러 개 있으므로, 각 점이 어떤 원의 중심인지 시각적으로 대응시키는 구조로 이해할 수 있다.

[주의 및 TODO]
- 정답 기호를 추론하여 채우지 말 것.
- `(정답)` 옆 동그라미 안의 기호는 작고 흐려 보이므로, 판독이 불확실하면 TODO로 남겨야 한다.
- 원의 정확한 중심 좌표, 반지름, 겹침 관계는 이미지 해상도상 오차가 있을 수 있다.
- 선택지 `ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ`의 정확한 배치는 대략적 위치로만 다루고, 수치 좌표로 단정하지 말 것.
- 해설 문장에 `원의 중심`이 등장하지만, 이것만으로 정답을 확정하는 서술은 금지한다.
- 등호 연쇄 변형은 없으므로, 식 변환 설명을 추가하지 않는다.
