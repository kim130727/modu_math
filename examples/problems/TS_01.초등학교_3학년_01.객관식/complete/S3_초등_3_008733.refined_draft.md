[문제 개요]
- 한 페이지형 초등 수학 활동 문제 화면이다.
- 문제 번호 `88.`이 보이고, 사다리(연결선) 구조를 따라 분수 카드와 분류 상자를 연결해 `○/×`를 고르는 형식이다.
- 상단에는 실제 문제처럼 보이는 영역이 있고, 중간 왼쪽에는 정답/해설 텍스트 블록이 따로 있다.
- 하단에는 같은 구조를 다시 그린 예시/작성용 그림처럼 보이는 영역이 있다.
- 전체 주제는 분수를 `가분수`, `진분수`로 분류하는 활동으로 보인다.

[원본에 보이는 요소]
- 상단 지시문:
  - `88. 사다리를 타고 내려가 도착한 곳이 참이면 ○표, 거짓이면 ×표를 선택하세요.`
- 상단 분수 카드 3개:
  - `3/10`
  - `2/2`
  - `15/7`
- 상단 분류 상자 3개:
  - `가분수`
  - `진분수`
  - `진분수`
- 각 분류 상자 아래 선택지:
  - `(○, ×)`
  - `(○, ×)`
  - `(○, ×)`
- 왼쪽 중간 해설 블록:
  - `(정답) ○, ×, ○`
  - `(해설) 3/10 → 진분수`
  - `2/2, 15/7 → 가분수`
- 하단에도 같은 분수 카드 `3/10`, `2/2`, `15/7`와 같은 분류 상자 `가분수`, `진분수`, `진분수`가 다시 보인다.
- 하단에는 점선 연결이 추가된 사다리 형태가 보인다.

[수식/관계 구조]
- 이 화면은 계산식 변형 문제라기보다, 분수와 분류명 사이의 대응 관계를 연결선으로 표현한 구조이다.
- 각 카드에서 아래의 분류 상자까지 세로/가로 연결선이 이어져 있다.
- 상단 구조에서는:
  - 왼쪽 카드 `3/10`이 왼쪽 상자 `가분수` 쪽과 연결된 형태로 보인다.
  - 가운데 카드 `2/2`가 가운데 상자 `진분수` 쪽과 연결된 형태로 보인다.
  - 오른쪽 카드 `15/7`이 오른쪽 상자 `진분수` 쪽과 연결된 형태로 보인다.
- 왼쪽 해설에는 화살표 `→`가 사용되어 분수와 분류명을 연결한다.
- 보이는 화살표 관계는 수식 계산이 아니라 분류 결과 표시로 해석된다.

[등호별 변환 의미]
- 등호 `=`가 연속으로 나타나는 식은 보이지 않는다.
- 대신 화살표 `→`가 보이며, 이는 다음처럼 대응을 보여 주는 표현으로 보인다.
  - `3/10 → 진분수`
  - `2/2, 15/7 → 가분수`
- 각 화살표는 풀이의 수식 변형이 아니라, 해설에서 분수와 분류명을 연결하는 안내로 보인다.
- 상단과 하단의 연결선도 같은 대응을 도식적으로 보여 주는 요소로 보인다.

[빈칸 정책]
- 화면에 보이는 빈칸이나 선택 영역이 있더라도 추론값을 채워 넣지 않는다.
- `○`, `×`는 선택지로만 취급하고, 실제 선택 결과를 임의로 확정하지 않는다.
- 하단의 점선 사다리 구조가 무엇을 의미하든, 보이는 형태만 기록하고 의미를 단정하지 않는다.
- 정답이 화면에 인쇄되어 보이는 부분은 관찰 사실로만 적고, 추가 계산으로 보강하지 않는다.
- 보이지 않는 값, 가려진 값, 해상도 때문에 애매한 값은 모두 빈칸 정책에 따라 유지한다.

[반복 패턴]
- 파란 테두리의 분수 카드가 3개 반복된다.
- 분수는 세로 배치로 표시된다.
  - 위: 분자
  - 가운데: 가로분수선
  - 아래: 분모
- 분홍 테두리의 분류 상자가 3개 반복된다.
- 각 분류 상자 아래에 `(○, ×)`가 반복된다.
- 카드와 상자 사이에 세로선과 가로선이 이어져 사다리처럼 보인다.
- 하단 영역은 상단 영역과 동일한 요소를 다시 보여 주며, 점선 연결로 강조된 부분이 있다.
- 상단은 비교적 단순한 실선 구조, 하단은 점선 포함 강조 구조로 보인다.

[DSL 작성 힌트]
- 페이지 전체를 하나의 캔버스로 두고, 상단 지시문을 좌측 상단에 배치한다.
- 중앙 상단에 파란 테두리 분수 카드 3개를 가로로 일정 간격으로 배치한다.
- 각 카드 내부는 세로 분수 형태로 렌더링한다.
  - 분자 텍스트
  - 가운데 수평선
  - 분모 텍스트
- 카드 아래에는 분홍 테두리의 분류 상자 3개를 가로 정렬한다.
- 카드와 상자 사이에는 세로선/가로선으로 이어지는 연결선을 넣어 사다리 형태를 만든다.
- 각 분류 상자 아래에 `(○, ×)` 텍스트를 각각 배치한다.
- 좌측 중간에는 작은 해설 블록을 따로 두고 `(정답)`, `(해설)` 문구와 화살표를 함께 배치한다.
- 하단에는 같은 종류의 카드/상자 구조를 한 번 더 배치하되, 점선 연결선을 추가해 예시처럼 보이게 한다.
- 상단과 하단은 서로 다른 수준의 시각적 위계가 느껴지도록 크기, 간격, 선 스타일을 조절한다.
- 분홍/파랑 테두리 색상은 구조 구분에 중요하므로 유지한다.
- 사다리 연결선은 실선과 점선을 구분해 그린다.
- 분수 카드의 내부 텍스트는 중앙 정렬이 적절하다.

[semantic/solvable 힌트]
- 이 문제는 분수를 `가분수`와 `진분수`로 분류하는 활동형 문제로 보인다.
- 화면에는 분수 카드와 분류 상자의 대응이 함께 제시되어 있다.
- 왼쪽 해설 블록은 정답 및 분류 예시를 보여 주는 보조 정보로 보인다.
- 다만 이 정답/해설을 그대로 채택할지 여부는 DSL 작성 목적상 별도 판단이 필요하다.
- 상단과 하단의 관계가 동일한지, 하단이 예시인지 실제 입력용인지 화면만으로는 단정하지 않는다.

[주의 및 TODO]
- `정답` 표기가 실제 정답 표시인지 해설 예시인지 애매하므로, 메타 정보로만 남기고 단정하지 말 것.
- 하단의 점선 사다리 구조가 입력 대상인지, 설명용 강조인지 불확실하므로 TODO로 남길 것.
- 가운데와 오른쪽 연결선의 정확한 교차/접점은 해상도상 다소 애매하므로 과도하게 단정하지 말 것.
- 상단 분류 상자 텍스트는 현재 보이는 글자 기준으로 `가분수`, `진분수`, `진분수`로 기록하되, 오독 가능성이 있으면 TODO로 남길 것.
- 분수 `2/2`, `15/7`의 분류를 여기서 직접 판정하지 말고, 화면에 보이는 해설 텍스트만 관찰 사실로 적을 것.
- 전체 구성은 활동형 문제이므로, 실제 DSL에서는 텍스트, 카드, 상자, 연결선, 선택지를 분리된 컴포넌트로 다루는 것이 안전하다.
