[전체적인 구성]
- 한 페이지형 초등 수학 문제 화면이다.
- 상단에 문제 지시문이 있고, 중앙에는 세 개의 분수 카드와 그 아래 분류 상자, 연결선이 배치되어 있다.
- 중간 왼쪽에는 “정답”, “해설”이 작은 글씨로 표시된 설명 영역이 따로 있다.
- 하단에는 같은 형태의 구조를 한 번 더 보여 주는 예시/작성용 그림이 있으며, 점선 테두리와 점선 연결이 추가되어 있다.
- 전체적으로 “분수 카드 → 분류(가분수/진분수)”를 연결해 맞히는 활동형 구성이다.

[보이는 텍스트]
- 문제 번호와 문장:
  - `88. 사다리를 타고 내려가 도착한 곳이 참이면 ○표, 거짓이면 ×표를 선택하세요.`
- 중앙 위쪽 카드의 분수:
  - `3/10`
  - `2/2`
  - `15/7`
- 분류 상자:
  - `가분수`
  - `진분수`
  - `진분수`
- 각 분류 상자 아래:
  - `(○, ×)`
  - `(○, ×)`
  - `(○, ×)`
- 왼쪽 중간의 안내:
  - `(정답) ○, ×, ○`
  - `(해설) 3/10 → 진분수`
  - `2/2, 15/7 → 가분수`
- 하단에도 위와 같은 분수 카드 `3/10`, `2/2`, `15/7`와 `가분수`, `진분수`, `진분수` 표기가 다시 보인다.

[반복되는 시각 패턴]
- 파란 테두리 사각형 카드 안에 분수가 세로로 배치되어 있다.
- 분수는 위에 분자, 아래에 분모가 있고 가운데 가로분수선이 있다.
- 분류 상자는 분홍색 테두리의 작은 사각형이며, 텍스트가 가운데 정렬되어 있다.
- 각 분류 상자 아래에는 `(○, ×)` 형식이 반복된다.
- 카드와 상자 사이에는 세로선 및 가로 연결선이 있어 “사다리”처럼 보이는 연결 구조를 만든다.
- 하단 예시에서는 같은 구조가 다시 나타나지만, 일부 연결 부분은 파란 점선으로 강조되어 있다.

[수식 단계]
- 해당 문제 화면에는 계산식의 연속 변형이 아니라, 분수의 분류를 연결하는 구조가 보인다.
- 다만 왼쪽 해설 영역에 `3/10 → 진분수`, `2/2, 15/7 → 가분수`처럼 화살표가 있는 연결 표현이 있다.
- 이 화살표는 수식 변형이 아니라 “분수와 분류 결과의 연결”을 나타내는 것으로 보인다.

[등호별 변환 의미]
- 등호가 이어진 식은 보이지 않는다.
- 대신 화살표(`→`)가 사용되어 분수와 분류 이름 사이의 대응을 보여 준다.
- 보이는 의미상:
  - `3/10` 옆에는 `진분수`
  - `2/2`, `15/7` 옆에는 `가분수`
  - 라는 식으로 연결된 안내가 있다.
- 이는 풀이 과정의 변환이라기보다 정답 예시 또는 해설 표시로 보인다.

[세부 내용]
- 상단 문장은 “사다리를 타고 내려가 도착한 곳이 참이면 ○표, 거짓이면 ×표를 선택”하라는 지시문이다.
- 중앙의 세 분수 카드가 각각 별도의 경로처럼 아래쪽 분류 상자와 연결된다.
- 왼쪽 카드 `3/10`은 아래 왼쪽 상자 `가분수`와 연결된 선이 보인다.
- 가운데 카드 `2/2`는 아래 가운데 상자 `진분수`와 연결된 선이 보인다.
- 오른쪽 카드 `15/7`은 아래 오른쪽 상자 `진분수`와 연결된 선이 보인다.
- 연결선은 회색 실선으로 그려져 있고, 하단 예시에서는 같은 위치에 점선 강조가 추가되어 있다.
- 각 카드 아래의 분류 상자 아래에는 `(○, ×)` 선택지가 있어, 도착한 곳의 내용이 참인지 거짓인지 표시하도록 되어 있다.
- 왼쪽 해설에는 정답 순서가 따로 제시되어 있어, 세 항목에 대해 각각 ○ 또는 ×를 고르는 문제 형식임을 알 수 있다.

[Python DSL 작성 힌트]
- 먼저 페이지 전체를 하나의 캔버스로 두고, 상단 지시문 텍스트를 좌측 상단에 배치한다.
- 중앙 영역에는 파란 테두리의 분수 카드 3개를 가로로 일정 간격으로 배치한다.
- 각 카드 내부는 세로 분수 형식으로:
  - 위칸 분자
  - 가운데 가로선
  - 아래칸 분모
  로 구성한다.
- 카드 아래에는 분홍 테두리의 분류 상자 3개를 가로로 배치한다.
- 카드와 상자 사이에는 세로선/가로선으로 이어지는 “사다리” 형태의 연결선을 그린다.
- 각 분류 상자 아래에 `(○, ×)`를 각각 배치한다.
- 왼쪽 중간에 작은 해설 텍스트 블록을 따로 두고, `(정답)`, `(해설)` 표기와 화살표를 포함한 문자열을 배치한다.
- 하단에는 동일한 분수 카드와 분류 상자 구성을 한 번 더 그리되, 일부 경로를 점선으로 강조하면 원본과 유사하다.
- 텍스트 크기와 카드 크기 차이를 두어 상단 문제와 하단 예시의 위계가 드러나게 한다.

[주의할 점]
- 정답을 직접 계산해서 채워 넣지 말고, 화면에 보이는 배치와 텍스트만 옮겨 적는 방식으로 작성해야 한다.
- `○`와 `×`는 선택지로만 보이며, 실제 선택 결과를 추론해 하나로 고정하면 안 된다.
- `2/2`가 진분수인지 가분수인지 같은 판정은 여기서 단정하지 말고, 화면에 보이는 해설 표기만 따로 기록한다.
- 하단 예시와 상단 문제는 같은 요소가 있지만, 완전히 같은 의미인지 단정하지 말고 “같은 구조로 보임” 정도로만 적는다.
- 점선 연결은 강조 표시일 수 있으므로, 실선과 점선을 구분해서 기록한다.
- 분홍/파랑 테두리의 색상 차이는 구조 구분에 중요한 요소이므로 누락하지 않는 것이 좋다.

[불확실한 부분]
- 하단의 점선 사다리 구조가 실제 입력 대상인지, 설명용 예시인지 화면만으로는 완전히 단정하기 어렵다.
- 왼쪽의 `(정답) ○, ×, ○`가 실제 정답 표시인지 해설용 예시인지, 화면 맥락상 해설에 가깝지만 절대적으로 확정하긴 어렵다.
- 가운데와 오른쪽 연결선의 정확한 교차 위치는 해상도 때문에 약간 애매하다.
- 상단과 하단의 분류 상자 중 일부가 서로 다른 문구처럼 보일 가능성이 있으나, 현재 보이는 글자는 모두 `가분수` 또는 `진분수`로 읽힌다.
