[문제 개요]
- 초등 수학 문제 1개가 있고, 핵심은 **분수/대분수의 크기 비교**이다.
- 상단에는 문제 문장이 있고, 중앙에는 **계층형 비교 도식(트리 구조)** 이 배치되어 있다.
- 화면 왼쪽 중간에는 `(정답)`, `(해설)`이 별도 텍스트로 표시되어 있다.
- 하단에는 비교 과정을 문장형 해설로 다시 적어 놓았고, 그 아래에 작은 반복 도식이 한 번 더 나온다.

[원본에 보이는 요소]
- 문제 번호: `26.`
- 문제 문장:
  - `두 분수의 크기를 비교하여 더 큰 분수를 빈칸에 써 넣으려고 합니다. ①에 들어갈 분수로 알맞은 것을 가장 아래 칸의 분수들 중에 선택해 보세요.`
- 중앙 도식의 최상단 상자 내부: `①`
- 하단 후보 분수들(4개):
  - `23/6`
  - `21/6`
  - `3 2/6`
  - `3 4/6`
- 중간 왼쪽 텍스트:
  - `(정답) 23/6`
  - `(해설) 23/6 > 21/6, 3 2/6 < 3 4/6`
- 하단 문장:
  - `23/6 과 3 4/6의 크기를 비교하면 23/6 = 3 5/6 이므로 23/6 > 3 4/6입니다.`
  - `따라서 ①에 알맞은 수는 23/6 입니다.`
- 하단에 작은 동일 구조의 도식이 한 번 더 보이며, 일부 값(`23/6`, `3 4/6`)이 파란색으로 강조되어 있다.

[수식/관계 구조]
- 도식은 위에서 아래로 내려가는 3단 구조로 보인다.
  1. 최상단: `①`
  2. 중간: 빈 상자 2개
  3. 하단: 분수/대분수 카드 4개
- 하단 카드의 배열:
  - 왼쪽 묶음: `23/6`, `21/6`
  - 오른쪽 묶음: `3 2/6`, `3 4/6`
- 관계식으로 보이는 텍스트는 다음과 같다.
  - `23/6 > 21/6`
  - `3 2/6 < 3 4/6`
  - `23/6 = 3 5/6`
  - `23/6 > 3 4/6`
- 의미상, 각 묶음에서 더 큰 수를 골라 상위 상자와 연결하는 형태로 읽힌다.

[등호별 변환 의미]
- `23/6 = 3 5/6`
  - 화면에 보이는 그대로의 변환 결과이다.
  - 가분수 `23/6`를 대분수 `3 5/6`로 바꿔 쓴 표현처럼 보인다.
- `23/6 > 3 4/6`
  - `23/6`이 `3 4/6`보다 크다는 비교 결과를 나타낸다.
- `23/6 > 21/6, 3 2/6 < 3 4/6`
  - 쉼표로 두 개의 비교식을 나란히 적은 형태이다.
  - 각 비교는 서로 독립적인 관계로 보인다.
- 등호 연쇄형 계산은 보이지 않으며, 표시된 변환은 한 단계식으로 읽는 것이 안전하다.

[빈칸 정책]
- `①`은 **빈칸 표시**로 취급한다.
- 원본에 정답이 인쇄되어 보이더라도, DSL 초안에서는 빈칸에 값을 **추론해 채우지 말고 빈칸 상태를 유지**해야 한다.
- 정답/해설 텍스트는 보이는 사실로 기록할 수는 있으나, 빈칸 값 확정 근거로 자동 대입하지 않는다.

[반복 패턴]
- 같은 형태의 사각형 카드가 반복 사용된다.
- 카드 스타일은 연한 보라색 테두리와 옅은 보라색 채움으로 보인다.
- 카드 하단에는 보라색 띠처럼 보이는 강조 영역이 있다.
- 도식 구조는 반복적으로
  - 상단 1개
  - 중간 2개
  - 하단 4개
  의 분기형 트리 패턴을 이룬다.
- 아래쪽 작은 도식에서는 일부 숫자가 파란색으로 표시되어 강조된다.
- 분수 표기는 `23/6` 같은 가분수와 `3 4/6` 같은 대분수가 함께 반복된다.

[DSL 작성 힌트]
- 도식은 **트리형 레이아웃**으로 구성하는 것이 적합하다.
- 상자 크기는 가능한 한 통일하고, 내부 텍스트만 달리 넣는다.
- 연결선은 수직 1개 + 수평 분기선 + 다시 수직으로 내려가는 구조로 표현하면 좋다.
- 상자 배치는 다음과 같이 읽히는 것이 자연스럽다.
  - 최상단 1개는 중앙 정렬
  - 중간 2개는 좌우 대칭
  - 하단 4개는 두 묶음으로 좌우 배치
- 텍스트 영역과 도식 영역은 위아래로 분리하는 편이 적합하다.
- `(정답)`, `(해설)`은 작은 본문 텍스트 블록으로 따로 배치한다.
- 분수는 일반 분수와 대분수 표기를 모두 지원해야 한다.
- 강조 색상은 파란색, 기본 카드 색은 연한 보라색 계열로 구분한다.
- 하단의 작은 도식은 별도 예시 블록처럼 배치하면 원본과 유사하다.

[semantic/solvable 힌트]
- 문제의 의도는 후보 중에서 **가장 큰 분수**를 고르는 것이다.
- 상단 `①`은 비교 결과로 선택된 분수를 넣는 자리로 보인다.
- 하단 해설 문장은 `23/6`과 `3 4/6`의 비교를 보여 주며, 화면에는 `23/6 = 3 5/6`이 적혀 있다.
- 다만 이 문서에서는 정답을 다시 계산해 확정하지 않고, **원본에 인쇄된 텍스트만 관찰 사실로 기록**한다.
