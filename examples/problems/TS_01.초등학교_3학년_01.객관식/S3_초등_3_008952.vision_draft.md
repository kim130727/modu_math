[전체적인 구성]
- 전체 화면은 **초등 수학 문제 1문항**으로 구성되어 있습니다.
- 상단에는 문제 번호와 문장이 왼쪽부터 가로로 배치되어 있고, 그 아래 중앙에는 **‘목장별 우유 생산량’**이라는 제목이 있는 그림 표가 있습니다.
- 표 아래쪽에는 **보기 선택지처럼 보이는 그림 답안 묶음**이 여러 개 배치되어 있으며, 왼쪽 아래에는 **(정답) ④** 표시가 보입니다.
- 맨 아래에는 **해설 문장**이 한 줄로 이어져 있습니다.
- 이 문제는 **우유 팩 그림으로 수량을 나타낸 표**를 보고, 초록 목장의 생산량을 바르게 나타낸 그림을 고르는 형식입니다.

[보이는 텍스트]
- 문제 번호: **22.**
- 문제 문장:
  - **목장별 우유 생산량을 조사하여 그림그래프로 나타내었습니다. 초록 목장의 우유 생산량은 430kg이라고 합니다. 초록 목장 우유 생산량을 바르게 나타낸 것은 어느 것일까요?**
- 표 제목: **목장별 우유 생산량**
- 표의 열 제목:
  - **목장**
  - **우유 생산량**
- 표의 행 이름:
  - **신선**
  - **청정**
  - **초록**
  - **푸른**
- 범례처럼 보이는 표시:
  - 우유 팩 그림 옆에 **100kg**
  - 작은 우유 팩 그림 옆에 **10kg**
- 선택지 번호:
  - **①, ②, ③, ④, ⑤**
- 정답 표시:
  - **(정답) ④**
- 해설 문장:
  - **(해설) 초록 목장의 우유 생산량은 430kg이므로 [우유 팩 그림] 4개, [작은 우유 팩 그림] 3개로 나타냅니다.**
- 해설 문장 중 그림은 텍스트 대신 **우유 팩 아이콘**으로 표시되어 있습니다.

[반복되는 시각 패턴]
- 표 안의 각 목장 생산량은 **우유 팩 아이콘의 개수**로 표현되어 있습니다.
- 같은 크기의 큰 우유 팩 그림은 **100kg 단위**, 작은 우유 팩 그림은 **10kg 단위**로 반복 사용됩니다.
- 선택지 ①~⑤도 각각 **큰 우유 팩 여러 개 + 작은 우유 팩 여러 개**의 조합으로 반복 구성되어 있습니다.
- 표와 선택지 모두 **아이콘 개수 비교**를 유도하는 형태입니다.
- 숫자 선택지 아래에 별도의 텍스트 없이 **그림만 나열된 반복 패턴**이 보입니다.

[수식 단계] (해당될 때)
- 이 문제 화면에는 별도의 등식 변형식이나 계산식을 직접 나열한 부분은 보이지 않습니다.
- 다만 해설에서 **430kg을 우유 팩 그림 4개와 작은 우유 팩 그림 3개로 나타낸다**는 형태의 대응이 제시되어 있습니다.
- 따라서 수식 단계는 없고, **수량-그림 대응 관계**만 시각적으로 나타난 것으로 보입니다.

[등호별 변환 의미] (해당될 때)
- 화면에 **등호(=)** 로 이어지는 변형식은 보이지 않습니다.
- 따라서 단계별 등호 의미를 해석할 대상은 없습니다.

[세부 내용]
- 상단 표는 연한 초록색 계열 테두리와 배경으로 구성되어 있습니다.
- 표 제목 **‘목장별 우유 생산량’** 은 표 위쪽 중앙에 별도 박스처럼 배치되어 있습니다.
- 표의 첫 번째 열은 목장 이름이 들어가는 좁은 칸이고, 두 번째 열은 우유 생산량을 그림으로 나타내는 넓은 칸입니다.
- 각 행에는 목장 이름이 왼쪽에 세로로 정렬되어 있습니다.
- 신선, 청정, 푸른 행에는 우유 팩 그림들이 가로로 나열되어 있고, 초록 행은 표 안에서 **그림이 비어 있는 칸처럼 보입니다.**
- 범례는 표 오른쪽 아래쪽에 따로 배치되어 있으며,
  - 큰 우유 팩 그림 1개 옆에 **100kg**
  - 작은 우유 팩 그림 1개 옆에 **10kg**
  로 표시되어 있습니다.
- 표 아래의 선택지들은 ①~⑤ 숫자 표식이 각각 원형에 들어간 형태로 보이고, 각 번호 옆에 우유 팩 조합 그림이 붙어 있습니다.
- 하단 해설은 왼쪽 아래에서 시작해 가로로 이어지며, 중간에 우유 팩 아이콘이 문장 속에 삽입되어 있습니다.

[Python DSL 작성 힌트]
- 표 구조를 먼저 잡는 것이 중요합니다.
  - 큰 제목 박스 1개
  - 2열짜리 표 1개
  - 행은 목장 이름 4개
- 표의 두 번째 열에는 숫자 대신 **아이콘 조합**을 배치해야 합니다.
- 범례는 표 아래 또는 오른쪽에 별도 영역으로 두고,
  - 큰 우유 팩 = 100kg
  - 작은 우유 팩 = 10kg
  관계를 표시해야 합니다.
- 선택지 ①~⑤는 각각 작은 그룹 단위로 배치하고, 번호와 그림이 한 묶음이 되도록 정렬해야 합니다.
- 해설 문장에는 텍스트 사이에 아이콘이 들어가므로, **문장 + inline icon + 문장** 형태를 지원하는 배치가 필요합니다.
- 그림의 크기 차이가 중요하므로, 큰 우유 팩과 작은 우유 팩의 스케일을 분리해서 정의하는 것이 좋습니다.
- 연한 초록색 테이블 배경, 테두리 색, 셀 분할선도 별도 속성으로 처리하면 좋습니다.

[주의할 점]
- **초록 목장의 실제 정답 수량을 계산해서 채우면 안 됩니다.**
- 선택지의 정답 번호가 화면에 보이더라도, DSL 작성용 분석에서는 **정답을 근거로 내용 자체를 재구성하지 않도록 주의**해야 합니다.
- 표 안의 각 행에 있는 우유 팩 개수는 보이는 그대로 기록해야 하며, 추정으로 보완하지 마세요.
- 해설 문장에 등장하는 우유 팩 아이콘은 텍스트로 치환하지 말고, **아이콘이 들어간 위치**만 설명해야 합니다.
- 초록 행은 비어 있는 칸처럼 보이지만, 그 빈칸에 들어갈 그림을 추론해서 적지 마세요.

[불확실한 부분]
- 표 안 각 행의 우유 팩 개수는 해상도 때문에 일부 아이콘 경계가 겹쳐 보일 수 있습니다.
- 선택지 ①~⑤의 우유 팩 개수는 작게 표시되어 있어 정확한 개수 판독에 약간의 오차 가능성이 있습니다.
- 해설 문장 속 아이콘의 정확한 위치와 개수는 화면 크기상 다소 흐릿하게 보일 수 있습니다.
- 초록 행의 빈칸 내부에 원래 어떤 그림이 들어갈지, 문제 지시만으로는 시각적으로 확정된 정보만 적어야 하므로 단정하지 않습니다.
