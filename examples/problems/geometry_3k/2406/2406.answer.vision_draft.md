# [전체적인 구성]
- 화면 상단 왼쪽에 영어로 된 문제 지시문이 1줄 배치되어 있습니다.
- 중앙에는 파란색 선으로 그린 기하 도형이 크게 배치되어 있고, 여러 점과 각도 표기가 함께 있습니다.
- 도형 오른쪽에는 선택지 형태의 텍스트가 2줄로 배치되어 있습니다.
- 오른쪽 아래에는 빨간색으로 보이는 정답 표시 문구가 겹쳐져 있지만, 이는 시각 자료의 일부로 보이며 문제 본문과는 별도로 존재합니다.

# [보이는 텍스트]
- 상단 문제 문장:
  - `Find the value of x in the diagram.`
- 도형 내부/주변 각도 표기:
  - `79°`
  - `(x−1)°`
  - `(x+10)°`
  - `2x°`
- 선택지:
  - `1. 68 / 2. 78 / 3. 79`
  - `/ 4. 136`
- 오른쪽 아래 빨간 표시:
  - `정답: A (68)`

# [반복되는 시각 패턴]
- 각도 표기가 모두 흑색 텍스트로, 도형의 서로 다른 위치에 붙어 있습니다.
- 파란색 선분/반직선이 여러 개 사용되며, 끝부분에는 화살표 모양의 선 두 개로 방향을 나타냅니다.
- 점 표시는 짙은 파란색 작은 원으로 통일되어 있습니다.
- 선택지는 숫자와 값이 슬래시(`/`)로 구분된 문자열 형태로 보이며, 두 줄에 나뉘어 배치되어 있습니다.

# [수식 단계]
- 별도의 등호 변형식이나 계산식 단계는 보이지 않습니다.
- 도형에 각도 표현만 있으며, 식의 전개 과정은 제시되지 않았습니다.

# [등호별 변환 의미]
- 해당 없음.
- 화면에 `=` 기호를 포함한 단계별 변형이 나타나지 않습니다.

# [세부 내용]
- 도형의 중심점처럼 보이는 점은 `D`로 표시된 위치 `(360, 420)` 부근입니다.
- 이 점에서 여러 방향으로 선이 뻗어 있습니다.
  - 왼쪽 위로 향하는 파란 반직선이 있고, 끝에는 화살표 머리가 있습니다.
  - 오른쪽 위로 향하는 파란 반직선이 있습니다.
  - 오른쪽 아래로 내려가는 파란 선이 있고, 점 `B`(약 `(430, 610)`) 쪽으로 연결됩니다.
  - 왼쪽 아래 방향으로는 짧은 선분처럼 보이는 구간이 있습니다.
- 별도의 점 `C`는 `(620, 300)` 부근에 있고, `D`에서 `C`로 이어지는 파란 선이 있습니다.
- 점 `E`는 `(250, 180)` 부근, 점 `I`는 `(930, 610)` 부근에 표시되어 있습니다.
- 각도 표기의 위치상:
  - `79°`는 중심점 위쪽, 왼쪽 위 방향 각도 영역 근처에 있습니다.
  - `(x−1)°`는 중심점 아래쪽 왼편의 각도 영역 근처에 있습니다.
  - `(x+10)°`는 오른쪽 위쪽 각도 영역 근처에 있습니다.
  - `2x°`는 오른쪽 아래쪽 각도 영역 근처에 있습니다.
- 선택지는 오른쪽 중앙에 있으며, 문제와 별개의 텍스트 블록처럼 배치되어 있습니다.
- 빨간색 `정답: A (68)`는 화면 맨 아래 오른쪽에 작게 오버레이된 형태입니다.

# [Python DSL 작성 힌트]
- 도형 요소를 먼저 구성하는 것이 좋습니다:
  - 파란색 선분/반직선 여러 개
  - 점 `B, C, D, E, I`
  - 각도 라벨 `79°`, `(x−1)°`, `(x+10)°`, `2x°`
- 각도 텍스트는 도형의 해당 영역에 상대적으로 배치해야 합니다.
- 문제 문장은 좌상단에 별도 텍스트로 둡니다.
- 선택지는 우측 상단~중앙에 2줄 텍스트 블록으로 처리하면 됩니다.
- 반직선의 화살표 머리는 선 끝에 작은 선 두 개로 표현된 형태를 참고하면 됩니다.
- 점과 선의 색상은 진한 파란색 계열, 텍스트는 검정 계열로 맞추면 원본과 유사합니다.

# [주의할 점]
- 이 초안은 문제를 풀기 위한 것이 아니라, 시각 구조를 옮기기 위한 것입니다.
- `정답: A (68)`는 시각적으로 보이지만, 답을 확정해 서술하거나 문제 풀이로 사용하지 마세요.
- 각도 값들 사이의 관계를 추론해 계산하지 마세요.
- 선택지는 정답 여부와 무관하게 그대로 보이는 텍스트만 옮기세요.
- `79°`, `(x−1)°`, `(x+10)°`, `2x°`는 위치상 서로 다른 각도 라벨이므로 임의로 결합하지 마세요.

# [불확실한 부분]
- 일부 선은 반직선인지 선분인지 SVG 좌표만으로 완전히 단정하기 어렵습니다. 다만 화살표가 있는 선은 반직선처럼 보입니다.
- 점 `I`는 도형의 일부로 표시되지만, 어떤 역할인지 시각 정보만으로는 명확하지 않습니다.
- 선택지 텍스트의 줄바꿈 방식은 SVG 상에서 두 줄로 분리되어 있으나, 실제 문제 편집기에서는 한 줄 또는 별도 항목으로 재배치할 수 있습니다.
