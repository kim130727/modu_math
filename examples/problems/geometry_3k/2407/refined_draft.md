[문제 개요]
- 기하 도형 위에 각도들이 표시되어 있고, 그 관계를 이용해 `x`의 값을 묻는 객관식 문제로 보입니다.
- 문제 문장은 `x의 값을 구하시오.` 입니다.
- 우측에 선택지가 4개 제시되어 있습니다.
- 좌하/좌측 큰 영역은 도형 작업 공간처럼 보이고, 우측은 보기 영역처럼 분리되어 있습니다.

[원본에 보이는 요소]
- 좌측 상단에 문제 문장: `x의 값을 구하시오.`
- 우측 상단 선택지:
  - `1. 56 / 2. 68 / 3. 74 /`
  - `4. 84`
- 도형 영역은 연한 회색 테두리의 큰 사각형 박스 안에 배치되어 있습니다.
- 박스 안에는 검은 점들이 A~J로 보이는 위치에 분산되어 있고, 파란색 선분들이 이 점들을 연결합니다.
- 일부 선분 끝에는 화살표처럼 보이는 짧은 끝표시가 있습니다.
- 각도 표기 텍스트로 보이는 값:
  - `X°`
  - `(X-6)°`
  - `(X+10)°`
  - `56°`
  - `(X+4)°`
- 우하단에 빨간 오버레이로 `정답: C (74)`가 보입니다.
  - 다만 이것은 화면 오버레이로 보이며, 문제 본문과 분리된 표시입니다.

[수식/관계 구조]
- 화면에는 등호 `=` 를 포함한 계산식 연쇄는 보이지 않습니다.
- 대신 도형 위에 여러 각도 표현이 흩어져 있어, 각도들의 관계를 이용하는 기하 문제로 해석됩니다.
- 보이는 각도 표기는 모두 대문자 `X`를 사용한 형태입니다.
- 보이는 각도 표현들 사이의 실제 수학적 관계는 그림만으로 확정되지 않습니다.
- 내부각/외각, 평행선/교차선/삼각형 관계 등은 명시적으로 읽히지 않으므로 단정하지 않습니다.

[등호별 변환 의미]
- 해당 없음.
- 등호가 있는 변형식이나 계산 단계가 원본에 보이지 않습니다.
- 따라서 단계별 변환 설명을 작성할 수 없습니다.

[빈칸 정책]
- 원본에는 빈칸 입력란이 보이지 않습니다.
- 따라서 별도의 빈칸 채우기 정책은 적용되지 않습니다.
- 만약 DSL에서 빈칸을 표현해야 한다면, 추론값을 넣지 말고 빈칸 상태를 그대로 유지해야 합니다.
- 보이는 정답 오버레이 `정답: C (74)`는 관찰 사실로만 남기고, 빈칸의 정답으로 확정해 쓰지 않습니다.

[반복 패턴]
- 점과 선분이 반복적으로 사용되어 복잡한 연결 도형을 이룹니다.
- 파란색 선분 스타일이 전체적으로 통일되어 있습니다.
- 일부 선분 끝에 화살표 머리처럼 보이는 짧은 끝표시가 반복됩니다.
- 각도 라벨이 점 주변 여러 위치에 반복 배치되어 있습니다.
- 객관식 보기 형식이 `1. ... / 2. ... / 3. ... / 4. ...`처럼 슬래시와 줄바꿈을 함께 사용하는 형태입니다.

[DSL 작성 힌트]
- 도형 영역은 큰 사각형 박스 1개 안에 배치하는 구성이 적합합니다.
- 점 A~J는 먼저 배치한 뒤, 파란 선분들을 순서대로 연결하는 방식이 자연스럽습니다.
- 선분 스타일은 동일한 색과 두께로 통일하는 것이 좋아 보입니다.
- 화살표 끝표시는 일반 선분과 분리해서 별도의 장식 요소로 다루는 편이 안전합니다.
- 각도 텍스트는 선분/점보다 위 레이어에 올려서 읽기 쉽게 배치해야 합니다.
- 문제 문장과 선택지는 도형과 분리된 텍스트 블록으로 두는 것이 좋습니다.
- 선택지는 우측 상단에 두 줄 배치되어 있으므로, 줄간격과 정렬을 맞춰야 합니다.
- 우하단 빨간 `정답: C (74)`는 필요 시 참고용 레이어로만 두고, 기본 렌더링에서는 제외하는 쪽이 안전합니다.
- 도형의 연결 관계가 복잡하므로, 단순한 하나의 다각형으로 축약하지 말고 교차/겹침 가능성을 남겨 두는 것이 좋습니다.

[semantic/solvable 힌트]
- 이 문제는 각도 표시들 사이의 관계를 이용해 `x`를 구하는 기하 문제로 보입니다.
- 보이는 값들은 `X`, `X-6`, `X+10`, `X+4`, `56` 형태입니다.
- 다만 각 값이 어떤 각의 위치에 대응하는지, 어떤 합/보각/직선각 관계인지 원본 텍스트만으로는 확정되지 않습니다.
- 따라서 수학적 해결 구조는 보류하고, 도형 내 각도 라벨 배치만 정확히 옮기는 것이 우선입니다.
- 우하단의 `정답: C (74)`는 보이지만, 이것을 풀이 결과로 확정해서 적지는 않습니다.
