# [전체적인 구성]
- 화면 상단 왼쪽에 영어 문제 문장 `Find the area of the figure.`가 크게 배치되어 있습니다.
- 왼쪽 넓은 영역에는 도형 그림이 있고, 오른쪽에는 선택지 4개가 세로로 나열되어 있습니다.
- 도형 영역은 연한 회색 테두리의 사각형 영역 안에 들어가며, 삼각형 모양의 선분 도형이 표시되어 있습니다.
- 오른쪽 선택지는 각각 `1. 30`, `2. 60`, `3. 120`, `4. 240`처럼 번호와 값이 함께 표시됩니다.
- 우하단에는 빨간 글씨로 `정답: 2 (60)` 오버레이가 보이지만, 이는 시각 요소로만 참고해야 합니다.

# [보이는 텍스트]
- 상단 문제 문장: `Find the area of the figure.`
- 도형의 꼭짓점 라벨: `A`, `B`, `C`
- 선분 길이 표기: `13`, `13`, `10`
- 선택지:
  - `1. 30`
  - `2. 60`
  - `3. 120`
  - `4. 240`
- 우하단 빨간 표시: `정답: 2 (60)`

# [반복되는 시각 패턴]
- 삼각형의 변 위 또는 근처에 길이 숫자가 각각 하나씩 붙어 있습니다.
- 두 변에는 같은 값 `13`이 반복 표기되어 있어, 동일 길이 변이 두 개 있다는 시각적 패턴이 있습니다.
- 아래쪽 변에는 `10`이 따로 표기되어 있어, 세 변 중 하나가 다른 값으로 구분됩니다.
- 각 변의 양 끝에는 치수선 스타일의 표시가 반복되어 있습니다.
  - 끝점에 짧은 캡(cap) 선
  - 양끝 방향 화살표 모양 또는 사선 표시
- 점 라벨 `A`, `B`, `C`는 각각 꼭짓점 근처에 배치되어 있습니다.

# [수식 단계]

- 계산식, 변형식, 단계별 등식은 화면에 제시되어 있지 않습니다.

# [등호별 변환 의미]
- 해당 없음.

# [세부 내용]
- 왼쪽 도형은 세 꼭짓점 `A`, `B`, `C`를 가진 삼각형입니다.
- `A`는 왼쪽 아래, `B`는 위쪽, `C`는 오른쪽 아래에 위치합니다.
- 변의 연결은 다음처럼 보입니다.
  - `A`에서 `B`로 이어지는 왼쪽 사선 변
  - `B`에서 `C`로 이어지는 오른쪽 사선 변
  - `A`에서 `C`로 이어지는 아래쪽 수평 변
- 길이 표시는 다음 위치에 보입니다.
  - 왼쪽 사선 변 근처: `13`
  - 오른쪽 사선 변 근처: `13`
  - 아래 수평 변 근처: `10`
- 도형 내부에는 높이선이나 각도표시, 직각기호는 보이지 않습니다.
- 선택지는 화면 오른쪽에 위에서 아래로 일정 간격을 두고 세로 정렬되어 있습니다.
- 전체적으로 객관식 문제 형식이며, 정답을 선택하는 영역과 도형이 분리되어 있습니다.

# [Python DSL 작성 힌트]
- 도형은 왼쪽 영역에 배치하고, 선택지는 오른쪽 열에 세로 정렬하는 구성이 적절합니다.
- 삼각형은 점 `A`, `B`, `C`를 찍고 선분 3개로 연결하는 방식으로 표현하면 됩니다.
- 각 변의 길이 표시는 해당 선분의 중앙 부근에 텍스트로 배치하는 것이 자연스럽습니다.
- 치수선 느낌을 살리려면 변 양끝에 짧은 캡 또는 작은 화살표형 보조선을 추가하면 좋습니다.
- 문제 문장은 상단 왼쪽에 크게 배치하고, 도형과 선택지 사이에 여백을 두면 원본 구조와 유사합니다.
- 선택지는 단순 텍스트 목록으로, `1.`부터 `4.`까지 줄바꿈하여 배치하면 됩니다.

# [주의할 점]
- `정답: 2 (60)` 표시는 자동 생성된 오버레이일 수 있으므로, 문제 본문 요소와 혼동하지 마세요.
- 길이 숫자 `13`, `13`, `10`은 도형의 표시값일 뿐, 이를 이용해 문제를 풀어 값이나 답을 추론하지 마세요.
- 도형은 삼각형처럼 보이지만, 선의 기울기와 라벨 위치를 정확히 반영해야 합니다.
- 오른쪽 선택지는 문제의 핵심 시각 요소이므로 빠뜨리지 않도록 합니다.
- 문제 문장은 영어로 되어 있으므로, 한국어로 바꿔 쓰지 말고 보이는 텍스트를 그대로 인용하는 편이 좋습니다.

# [불확실한 부분]
- 삼각형의 정확한 타입(이등변삼각형 등)은 도형만으로 단정하기 어렵습니다.
- 선분 길이 표기가 실제로 해당 변 전체 길이를 의미하는지, 또는 보조적인 치수 표시인지 SVG만으로는 100% 확정하기 어렵습니다.
- 내부에 추가 표시가 없는 만큼 높이 또는 기준선이 생략된 것일 수 있으나, 화면상으로는 보이지 않습니다.
