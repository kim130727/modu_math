# [문제 개요]
- 문제 번호는 `93.`으로 보인다.
- 두 대분수 `2 1/3` 과 `1 2/3` 를 그림으로 비교하고, 더 큰 분수를 고르는 문제이다.
- 그림을 보고 비교 부호 `( > , < )` 를 선택하는 형식이다.
- 하단에는 `(정답)`과 `(해설)`이 인쇄되어 있다.

# [원본에 보이는 요소]
## 보이는 사실
- 상단 문장:
  - `2 1/3 과 1 2/3 중에서 어느 분수가 더 큰지 알아보려고 합니다. 그림을 보고 알맞은 것을 선택하세요.`
- 가운데 그림:
  - 위쪽 줄 왼쪽에 `2 1/3`
  - 아래쪽 줄 왼쪽에 `1 2/3`
  - 각 줄 오른쪽에 가로 직사각형 막대 3개가 나란히 배치됨
  - 각 막대는 세로 점선 2개로 3등분된 형태
  - 분홍색 채움이 일부 칸에 들어가 있음
- 설명 문장:
  - `색칠한 부분의 길이가 ( 2 1/3 , 1 2/3 )이/가 더 길므로`
  - `2 1/3 ( > , < ) 1 2/3 입니다.`
- 하단 인쇄:
  - `(정답) 2 1/3 , >`
  - `(해설) 색칠한 부분의 길이가 2 1/3이 더 길므로 2 1/3 > 1 2/3 입니다.`

## 추론되는 수학적 의미
- 그림의 색칠 길이를 비교하여 두 대분수의 크기를 판단하게 하는 문제로 보인다.
- 비교 기준은 “색칠한 부분의 길이”이다.
- 다만 정답 판단은 이미지에 이미 인쇄된 내용만 사실로 언급해야 하며, 별도 추론으로 채우지 않는다.

# [수식/관계 구조]
## 보이는 구조
- 비교 대상: `2 1/3` 와 `1 2/3`
- 선택 구조: `( > , < )`
- 문장 완성형:
  - `색칠한 부분의 길이가 ( 2 1/3 , 1 2/3 )이/가 더 길므로`
  - `2 1/3 ( > , < ) 1 2/3 입니다.`

## 구조 해석
- 등호 연쇄 변형식은 없다.
- 부등호를 고르는 비교 문제이며, 식 변형보다 시각 비교가 핵심이다.
- 비교 대상은 같은 단위의 막대 그림으로 나타난 두 대분수이다.

# [등호별 변환 의미]
- 등호 `=` 를 사용한 변환 사슬은 보이지 않는다.
- 따라서 단계별 등호 변형 설명은 해당 없음.
- 대신 문장 의미의 흐름은 다음과 같다.
  - 그림의 색칠된 길이를 본다.
  - 두 표현 중 더 긴 쪽을 고른다.
  - 그 결과를 `>` 또는 `<` 로 적는다.
- 하단 해설에는 `2 1/3 > 1 2/3` 이 인쇄되어 보인다.

# [빈칸 정책]
- 빈칸이나 선택 칸이 있더라도 추론값을 채우지 않는다.
- `( > , < )` 는 원문 그대로 보존한다.
- 문제 풀이로 정답을 새로 생성하지 않는다.
- 원본에 정답이 인쇄되어 보이는 경우만 관찰 사실로 기록한다.
- 이 문제는 비교 부호 선택형이므로, DSL 작성 시에도 빈칸은 빈칸으로 유지하는 것이 안전하다.

# [반복 패턴]
- 위아래 두 줄에 같은 형태의 가로 막대 3개가 반복된다.
- 각 막대는 직사각형이고, 내부에 세로 점선 2개가 있어 3등분 구조를 만든다.
- 두 줄 모두 왼쪽에 대분수 표기가 있고, 오른쪽에 대응하는 막대 그림이 배치된다.
- 분홍색 채움은 같은 형식의 막대들 사이에서 길이 비교를 눈으로 하게 만드는 반복 장치이다.
- 막대의 크기와 간격은 두 줄이 거의 동일해 보이며, 채움 정도만 다르게 보인다.

# [DSL 작성 힌트]
## 렌더링/레이아웃 힌트
- 상단 문제 문장은 좌측 정렬 텍스트로 배치한다.
- 가운데에는 2개 행의 비교 그림을 세로로 정렬한다.
- 각 행은:
  - 왼쪽의 대분수 텍스트
  - 오른쪽의 가로 막대 3개
  - 각 막대 내부의 세로 점선 2개
  로 구성한다.
- 각 막대는 같은 폭과 높이를 유지하고, 행 사이 간격을 충분히 둔다.
- 분홍색 채움은 막대 내부의 채움 영역으로 표현한다.
- 하단 설명문은 본문보다 아래쪽에 두고, `(정답)`, `(해설)`은 더 아래에 구분해서 배치한다.

## DSL component hints
- 대분수 텍스트 요소
- 3등분 막대 3개를 한 행에 배치하는 그룹
- 막대 내부 점선 분할선
- 채움 색상 영역
- 비교 부호 선택형 텍스트
- 정답/해설 텍스트 블록

# [semantic/solvable 힌트]
- 이 문제는 그림 비교형이다.
- 핵심 해석은 “색칠한 부분의 길이” 비교이다.
- 부등호 선택 칸은 답을 직접 계산해서 채우기보다, 원본에 보이는 텍스트 구조를 보존하는 것이 우선이다.
- 하단 해설에는 `2 1/3 > 1 2/3` 가 인쇄되어 보인다.
- 다만 DSL 작성 단계에서는 정답을 새로 산출하지 말고, 보이는 요소만 구조화한다.
