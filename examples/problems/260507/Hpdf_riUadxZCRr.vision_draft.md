# [전체적인 구성]
- 화면 상단에는 큰 글씨의 문제 문장이 여러 줄로 배치되어 있습니다.
- 하단에는 직사각형 모양의 테이프를 나타내는 도식이 가로로 길게 그려져 있습니다.
- 도식 위아래에 길이 표시가 곡선 점선과 직선 치수선 형태로 붙어 있습니다.
- 전체적으로 **문제 텍스트 + 그림(도형 설명)** 구조입니다.
- 계산식이나 등호가 이어지는 풀이 단계는 보이지 않습니다.

# [보이는 텍스트]
문제 본문으로 보이는 텍스트:
- `긴 변이 12 cm, 짧은 변이 3 cm인 직사각형`
- `모양의 종이테이프 3장을 2 cm씩 겹치게 이`
- `어 붙여서 직사각형을 만들었습니다. 만든 직`
- `사각형의 네 변의 길이의 합은 몇 cm입니까?`

그림 속 표기:
- 위쪽 곡선 점선 근처: `12 cm`
- 왼쪽 세로 쪽: `3 cm`
- 아래쪽 겹치는 부분 근처: `2 cm`

# [반복되는 시각 패턴]
- 직사각형 테이프가 **가로로 길게 3개 이어진 형태**로 보입니다.
- 각 테이프는 연한 회색/흰색 계열의 채움색과 검은 외곽선으로 표현되어 있습니다.
- 테이프가 이어지는 경계 부분에는 **겹침 구간을 나타내는 더 진한 회색 세로 띠**가 반복됩니다.
- 전체 길이를 나타내는 듯한 표시는 **위쪽의 점선 곡선 호**로, 특정 구간의 길이를 강조합니다.
- 높이 표시로는 **왼쪽에 세로 방향 치수**가 붙어 있습니다.

# [수식 단계]
- 해당 없음.
- 화면에는 등호(`=`)를 사용한 변형식이나 계산 과정이 보이지 않습니다.

# [등호별 변환 의미]
- 해당 없음.
- 풀이식이 제시되어 있지 않아 단계별 변환 의미를 관찰할 수 없습니다.

# [세부 내용]
- 문제는 **직사각형 모양의 종이테이프 3장**을 **각각 2 cm씩 겹치게** 붙여서 더 큰 직사각형을 만든 상황을 설명합니다.
- 그림에서 처음 테이프의 세로 길이로 보이는 `3 cm`가 왼쪽에 표시되어 있고, 가로 길이로 보이는 `12 cm`가 위쪽에 표시되어 있습니다.
- 아래쪽 `2 cm` 표시는 테이프끼리 **겹치는 부분의 길이**를 뜻하는 것으로 보입니다.
- 3장의 테이프가 일렬로 연결되어 있으며, 중간중간 겹치는 부분은 회색으로 더 진하게 표시되어 있습니다.
- 완성된 도형은 하나의 긴 직사각형처럼 보이지만, 내부에 겹침 경계가 남아 있어 원래 테이프들의 연결 구조를 알 수 있게 되어 있습니다.

# [Python DSL 작성 힌트]
- 상단 문제 문장은 4줄로 나뉘어 배치된 것으로 보이므로, 줄바꿈을 반영해 텍스트 객체를 배치하면 좋습니다.
- 하단 도식은 다음 요소들로 분해해서 그리면 좋습니다.
  - 왼쪽에 세로 `3 cm` 치수 텍스트
  - 가로로 긴 직사각형 외곽선 3개
  - 각 테이프의 겹침 구간을 나타내는 진한 회색 세로 띠 2개
  - 위쪽의 `12 cm` 점선 곡선 호
  - 아래쪽의 `2 cm` 치수 텍스트와 짧은 치수선
- 테이프의 본체는 연한 회색 채움, 겹치는 구간은 더 진한 회색 채움으로 구분하는 방식이 적절합니다.
- 치수 표시는 실제 도형과 분리된 보조선으로 그려야 합니다.
- 문제의 핵심 수치 `12 cm`, `3 cm`, `2 cm`는 도형 주변에 각각 독립적으로 배치해야 합니다.

# [주의할 점]
- 답을 구하거나 최종 값으로 보이는 수치를 추론해 넣지 마세요.
- `12 cm`가 전체 완성 도형의 길이인지, 한 장 테이프의 길이인지 같은 의미 해석은 조심해서 다뤄야 합니다. 그림과 문장 전체 맥락상 그렇게 읽히지만, DSL 작성 시에는 **보이는 위치 관계**를 우선 반영하는 것이 안전합니다.
- 겹침 구간은 실제로 두께가 다른 회색 띠로 보이므로, 단순한 분할선이 아니라 **겹친 부분 강조 영역**으로 처리하는 편이 좋습니다.
- 상단 문장 중 `이`와 `어 붙여서`가 줄바꿈으로 분리되어 있어, 원문을 그대로 옮길 때는 줄 경계를 고려해야 합니다.
- 그림의 곡선 점선 호는 정확한 호의 끝점이 다소 애매해 보일 수 있습니다.

# [불확실한 부분]
- 위쪽 `12 cm` 표기가 **어느 구간을 정확히 의미하는지** 그림만으로는 약간 애매합니다. 곡선 점선이 가리키는 범위는 보이지만, 이를 수치적으로 단정해 해석하지는 않는 것이 좋습니다.
- 아래의 `2 cm` 표시는 겹치는 길이로 보이지만, 정확히 어느 두 테이프의 어느 경계를 기준으로 잰 것인지는 도식상 완전히 명시적이지 않습니다.
- 완성된 직사각형의 내부 경계선이 실제 테이프의 경계인지, 시각적 구분을 위한 선인지 약간 혼재되어 보일 수 있습니다.
