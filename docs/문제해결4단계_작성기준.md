[폴리아 1단계: 문제 이해]

사용 필드:
- inputs
- given
- target

작성 목표:
- 무엇이 주어졌는가?
- 무엇을 구해야 하는가?
- 어떤 조건이 있는가?
- 답의 단위는 무엇인가?
- 객관식인지, 빈칸형인지, 직접 답하기인지 구분한다.

[폴리아 2단계: 계획 세우기]

사용 필드:
- method
- plan

작성 목표:
- 어떤 대표 방법을 사용하는가?
- 어떤 순서로 풀 것인가?
- 계산 전에 해결 경로를 제시한다.
- plan에는 최종 계산 결과를 미리 쓰지 않는다.

[폴리아 3단계: 계획 실행]

사용 필드:
- steps

작성 목표:
- 실제 계산과 추론을 순서대로 수행한다.
- 한 단계에 하나의 핵심 결과만 둔다.
- expr은 수행한 계산이나 판단을 기록한다.
- value는 그 단계에서 얻은 실제 결과를 기록한다.

[폴리아 4단계: 되돌아보기]

사용 필드:
- checks
- answer

작성 목표:
- 역연산, 대입, 조건 확인 등으로 답을 검증한다.
- 객관식이면 계산값이 보기에 있는지 확인한다.
- answer.value에 최종 수학적 답을 기록한다.
- 보기 번호와 정답값을 구분한다.
